　　第661章
<br/>　　这件事令程大雷有些头疼，他手托下巴，心情郁郁不快。
<br/>　　可对和珅来讲，这根本不应该算做一件事，更不值得程大雷考虑。大男人当去取天下，人心间的尔虞我诈，诸侯间的勾心斗角方才是程大雷要做的。至于那些许的女儿心思，完全不值一提。
<br/>　　古往今来，评价一个男人，多是看他的事业，他的胸襟，他的学识才华，可从未看过一个男人在儿女情长上的表现。如程大雷这般身份，便是三妻四妾，府中酒池肉林，旁人也说不出什么不是来，而委实程大雷在这方面确实也满检点。
<br/>　　“大当家该不至于如此为难吧？”和珅问。
<br/>　　“呃，有么，我只是刚才在想别的事，一时有些走神。”程大雷嘴硬不肯承认。
<br/>　　“如果大当家真的钟意，便将她纳了房，这倒也不算一件大事。”
<br/>　　程大雷现在大概也明白身处是一个怎样的环境，不过江山易改，本性难移，让他忽然适应这个世界，还是有几分难为人。心中仔细思索，委实不是没有贪念，若柳芷真随了别人，他心中也极其别扭。可若将她收房做小做妾，程大雷也不是对这件事太积极，毕竟苏樱和樊梨花已经令他很难办。
<br/>　　可事情总该解决的，僵局迟早要打破，如何破局，程大雷是一点法子都没有。
<br/>　　这种男女间的私事，程大雷也不好和珅说得太多。自己毕竟是他的上司，多多少少还是要面子的。
<br/>　　“哈，这等小事不提它，不提它。与我说说，城中现在如何。”
<br/>　　和珅也觉得这是小事，本该讨论的就是安邦定居的大事。他顿时来了精神，道：“从昨天开始，陆陆续续便有人来，目前到城中的人有五千余，看这等形势，一日两日停不下来。问题是，这些人都是走投无路的亡命徒，要让他们安分下来，并不是一件容易事。”
<br/>　　程大雷点点头，心中也明白这一点。以后会有更多人来到朔方城，成千上万人每个都有不同的经历，不同经历造成不同的想法，不同需求。而和珅要做的事，是砍掉他们身上的枝枝叉叉，让万人心成一人心，这当然不是一件容易的事情。
<br/>　　能做的也无非是用重典管理他们，施以恩，施以罚，让他们多朔方城产生归属感。
<br/>　　程大雷曾经做过一次这样的事，但那时候他身边的人是逐步增加的，但这次朔方城的人口是突然暴增。
<br/>　　五千人才是一个开始，朔方城的人口会暴增到怎样程度，现在还说不好。
<br/>　　“粮食怎样？”程大雷开口问道，人口一旦增加，粮食便是必须的消耗品。
<br/>　　“粮食倒不是太大的问题，黑风寨劫了一批，城外农田的粮食也要成熟，收获之后应该能撑一段时间。”和珅答道，说罢顿了顿：“说也奇怪，这批庄稼的成熟时间要早些。”
<br/>　　程大雷不动声色，没有接和珅的话茬。粮食总是一件要紧的事，不可有一丝一毫的大意，如果粮食供应不上，这几万人将从好事变成坏事。
<br/>　　“田里的粮食还得看紧些，莫要别人过来捣乱，粮食毁了，朔方城便也毁了。”
<br/>　　“明白，我已经派人日夜看守，在收获之前，不会让任何人接近农田。”
<br/>　　和珅做事果然让人放心，程大雷想到的他都想到了，程大雷没有想到的，他也能想到。农田连在一起，正是天干物燥的时节，只要一个火星，就能让一年的忙碌报销。
<br/>　　“嗯，这件事你做得对。”程大雷想起捣乱这两个字，就下意识就想起宋伯康：“凉州城哪里什么动向？”
<br/>　　“喜字队的情报一直传过来，他们的注意力还在京州，那里闹得一团乱麻，估计不会分心对付我们。”
<br/>　　“如此最好，但还是不可大意，等收获之后，便可松口气了。”
<br/>　　城中诸多事要忙碌，还有源源不断的流民迁入。和珅坐了一阵便走了，程大雷坐在远处，依旧郁郁发愣。
<br/>　　朔方城虽然很忙，但交给和珅还是令人放心的。程大雷心中头疼的还是后院的事……他和苏樱的关系江河日下，长此以往也不是个办法。关键是又添了个柳芷，当真是一波未平一波又起。
<br/>　　程大雷委实没有处理的法子，于是他决定不去处理，先在朔方城躲一段时间，总是能想到法子的。
<br/>　　……
<br/>　　凉州城，城头之上。
<br/>　　宋伯康一袭儒衫，看着远处的黑点，此刻心情十分不愉快。
<br/>　　本打算用朔方城分化程大雷的力量，自己先灭百里胜，再灭程大雷。可现实出人预料，自己没平得了百里胜，程大雷也未被自己分化。
<br/>　　竟有源源不断的人奔向蛤蟆城，而粮食危机，也被程大雷解决了。
<br/>　　如此，只能一日一日看着程大雷做大，假若今日不解决他，那明日凉州是谁家天下，也就是说不好的事情了。
<br/>　　程大雷是一个麻烦，还有一个麻烦是京州。尉迟离死后，相府开始扩充势力，这并不是一件容易做到的事情。尉迟离还当真有些死忠，殊死反抗，也是颇为令人忌惮。
<br/>　　只是相府的麻烦，同样也是宋伯康的麻烦，而除此之外，宋伯康还有宋伯康的心思。
<br/>　　他现在坐拥凉州，铁骑十万，是有争天下的资格。如此，自己是不是还要对相府言听计从便成为一个问题。
<br/>　　然而，宋伯康也不能完全抛弃相府，比如身边的戴白，陆洛都是顺从相府的人，如果自己与相府撕破脸皮，他们怕未必会听自己的。
<br/>　　乱世之中，每个人都在谋划着自己的利益，程大雷如此，宋伯康也是一样。有人因为共同的利益走在一起，但同时也有各自的小心思。
<br/>　　“委实有些麻烦呐。”
<br/>　　宋伯康长叹一声，身边的陆洛道：“王上无忧，我们的人已经混进去了，今夜就要行动，一把火烧了他们城外的农田，也就是王上除掉程大雷的时候。”
<br/>　　陆洛不知宋伯康的心思，他怎知道宋伯康的心里早已不局限一个程大雷。
<br/>(本章完)
