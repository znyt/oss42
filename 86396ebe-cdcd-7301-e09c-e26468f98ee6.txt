　　叶少阳真有点无语了。恨不得把他百年后的情况告诉他，但是一想，因为没有对证，最多也就是让他觉得自己编的比较好而已，不由挠起后脑勺，突然想到什么，把自己的佩剑七星龙泉剑接下来，递给道渊真人。
<br/>　　“茅山至宝，七星龙泉剑！你看看。”
<br/>　　道渊真人一听，讶然失色，接过去观察起来，脸上现出了震惊过度而呆滞的神情。毛小方也在一旁看着，感受到七星龙泉剑凌厉逼人的气息，脱口赞道：“好剑！”
<br/>　　“九段光的神兵。”道渊真人喃喃说道，“甚至比一般九段光的法器要好的多……”
<br/>　　“是啊，所以这就是七星龙泉剑。”
<br/>　　道渊真人把剑合上，还给他，盯着他的脸庞看了一会，说道：“你不怕我抢夺你的剑？”
<br/>　　“你不是那种人，不然我就不会来找你了。”
<br/>　　道渊真人听了这话，目光一动，说道：“剑是好剑，但我没见过七星龙泉剑，我不知道是不是。”
<br/>　　叶少阳当场有点无语了，把自己随身挟带的几件茅山的法器都拿出来给他们看，不过也只是更让人怀疑他的出身。
<br/>　　“你有这么多上好法器，你的来历绝不寻常，为什么要冒充茅山弟子呢？”道渊真人十分纳闷。
<br/>　　叶少阳没法回答，目光一扫，发现了他挂在腰间的佩剑，看到剑鞘的一瞬间，叶少阳惊呆了，猛然间想起什么，对道渊真人说道：“松纹古定剑？”
<br/>　　道渊真人愣了一下，点点头。
<br/>　　叶少阳道：“这把剑，是什么时候在你身上的？”
<br/>　　“我是龙虎山张天师后人，这把剑是我家传，我十三岁的时候，晋入真人牌位，这把剑就交给我保存，一直到现在，寸步不离。”
<br/>　　叶少阳一拍大腿，说道：“好啊，你十三岁……就算按照你我现在的年龄，那时候我最多也就两三岁吧，也就是说，我不可能有机会见到这把剑，我也是第一次见到你。对吧？”
<br/>　　道渊真人狐疑地看着他，不明白他的意思，不过还是老实回答：“那是当然。”
<br/>　　“那你平时给别人看过这把剑没有？”
<br/>　　“自然不会。这是我佩剑，又不是给人欣赏的，怎会随便给人看。你究竟想说什么？”
<br/>　　“我想说，你这把剑的上一些特征，是不是很少有人知道？”叶少阳道，“如果我说，我对这把剑了如指掌呢？”
<br/>　　“不可能！”道渊真人一句话就否定了，“这把剑是我张家传家之宝，不传外姓，就算是在龙虎山，也没有弟子看过这把剑的剑身。”
<br/>　　叶少阳道：“我看过，我知道剑身的特点！”
<br/>　　道渊真人刚要否定，叶少阳抢先说道：“这把剑的前端，从剑刃数起，大概一寸三的地方，镶嵌着一块吸灵石，每次斩杀邪物，能吸收一部分修为，滋养剑身，提升灵力，另外健身上刻着一道龙形符号，大概是最初它的主人入云龙公孙胜镌刻的……”
<br/>　　叶少阳继续回忆着松纹古定剑上的一些特点，陆续说出来，说完之后，一脸平静地望着道渊真人。
<br/>　　看道渊真人傻傻地表情，叶少阳就知道，自己没有说错，当然也不可能说错。
<br/>　　“就算你说的是真的，你又是怎么知道我这松纹古定剑中的秘密？”
<br/>　　叶少阳犹豫了一下，觉得如果什么都不说的话，这一关是没法过了，说道：“在我的那个时代里，你把松纹古定剑送给了我。”
<br/>　　“送给了你？怎么可能！”道渊真人差一点跳起来。
<br/>　　“是真的。不然你说我是怎么知道这把剑的秘密？”
<br/>　　道渊真人说道：“我为什么要把它送给你？”
<br/>　　叶少阳道：“因为我帮你办成了一件事，你把他送给了我，当然还是希望我用它和七星龙泉剑双剑合璧，对抗天劫。”
<br/>　　“天劫，什么天劫？”
<br/>　　叶少阳愣了一下，想了一下说道：“将来发生的事情，你确定想提前知道么？”
<br/>　　道渊真人犹豫了一下，说道：“那我不问了，不过，我对你的身份还有所怀疑，你还有没有别的能证明的？”
<br/>　　叶少阳摇摇头，“有些事情涉及太多将来发生的事，我不想告诉你。”
<br/>　　“你说松纹古定剑在你手上，现在在哪里？”
<br/>　　“不在我身上，因为我有七星龙泉剑，所以把它送给我媳妇了。我之前说了，我媳妇被尸族抓起来了，我的目的就是回到自己的时代，然后去救她。”
<br/>　　道渊真人道：“就算你说的是真的，你来找我有什么用？我没有办法让你回去。”
<br/>　　“我不知道找你能做什么，但你是我在这个世界唯一认识的人了，哦，阴司还有一些，但阴司不让我去了。”想到上次走阴的经历，想起萧逸云，无奈地叹了口气。
<br/>　　不过他这番话，却是让道渊真人想起了什么，说道：“我有办法验证你是不是在撒谎！”
<br/>　　“怎么办？”
<br/>　　“我带你去阴司！毛道长，有劳你也跟着一起，帮我做个见证！”
<br/>　　毛小方讷讷地点了点头，其实就算道渊真人不要求，他也想要跟着一起去见证一下，叶少阳说的事情，对他来说也是一次对世界观的巨大的冲击。
<br/>　　毕竟，他们这个时代，没法像百年之后叶少阳所在的那个时代一样，对于穿越时空这种事，有着一定的了解和接受度，这个时代的人，因为科学的闭塞，根本就没有这方面的哪怕是任何想象。
<br/>　　叶少阳背包里就有走阴的东西，拿出来布置走阴的法坛，结果道渊真人让他不必了，起身走到岩石前面去，一只手按在岩石上。
<br/>　　叶少阳这才看到岩石上刻着五行元素组成的符号，像是一个圆圈，道渊真人的手掌贴上去之后，五道符文被激活，现出黑色的雾气，在圆圈的中间蔓延开来，形成一团虚无的漩涡。
<br/>　　叶少阳一眼就看出，这是空间裂缝。人间有很多空间裂缝，有人造的，也有空间运转的节点，通过这些空间裂缝，任何人都可以进入别的空间。
