　　第129章
<br/>　　幽州、幽州城。
<br/>　　幽州城原先不叫这名字，这座城池自有名字，但自从幽州王杨龙停坐镇此地后，此地便代表了整个幽州。渐渐，人们提起它，就想到幽州城三个字，提起幽州城，便会想到坐镇此地的杨龙停。
<br/>　　杨龙停外貌看上去不像一员武将，倒像一名儒士。他常穿一身素白公子儒衫，戴扇子，佩剑，坐轿而不骑马。他身材也并高大，上了年纪就更显瘦弱。
<br/>　　在城主府一间厢房内，光线昏暗，台上供着一排灵位，借外面微弱的光线，能扫到上面的名字……
<br/>　　周遇鱼、秦雷、穆人义、常风、李山，庞虎……最后一个名字好像是：韩虎踞……
<br/>　　杨龙停盘腿坐在灵前，腿上搁着一柄长剑，身侧摆着一坛酒，几只碗。
<br/>　　“老韩，你也走了，咱们一个营里出来的弟兄，就只剩下我一个了。”
<br/>　　“常风是第一个走的，那时候咱们还都是小卒，后来官越坐越大，带的兵越来越多，身边的弟兄却越来越少。”
<br/>　　“老韩，你他妈真不是个东西，我真羡慕你，你走了还有我送你，等我哪天走了，就没人送我了。”
<br/>　　“虎哥，等我，拿了仇人的头在你灵前祭奠。”
<br/>　　杨龙停举杯，将碗中酒一饮而尽，然后起身，握剑，大步踏出房外。
<br/>　　房外阳光不错，一行人垂手而立。
<br/>　　“主公！”
<br/>　　“外面风大，别着了寒气，我们去暖房说话。”
<br/>　　屋子并不大，里面烧着炕，有仆人暖着炭炉进来，不多时暖意便泛上来。
<br/>　　“主公，遵您的命令，已经撒出消息去，通知幽州城的匪类在风雨亭一会，名册在这里，请主公过目。”幽州府军师李善言道。
<br/>　　杨龙停盘腿坐在炕上，挥挥手道：“坐，都坐，不用都站着。把名册先给大家看看。”
<br/>　　“主公，这些匪类都是一群乌合之众，平日里作恶多端，咱们真要将他们收归帐下么，岂不是毁了主公的名声。”
<br/>　　杨龙停手指敲着炕桌：“好有好的用法，坏有坏的用法，垃圾也未必没有用处，关键要看怎么用。幽州境内匪患猖獗，却也不是一天两天了。这群人永远不能成为我们自己人，用他们磨磨刀锋也好。况且，青牛山……”
<br/>　　“主公，青牛山我去过，现在是一个姓程的做主，他态度很好，很有报效主公的意思。”
<br/>　　“呵，倒是一颗识事务的墙头草，好，风雨亭上军师见见他，如果他真的能用，就赏他个差事，如果烂泥糊不上墙，也就算了。这次风雨亭我就不去了，军师去会会这帮草莽。”
<br/>　　“是！”
<br/>　　“义父！”门在此时突然撞开，一人高马大的汉子走进来：“一帮匪类，灭了就是，何必这么麻烦。义父你给我五百兵马，一月之内我将他们全部铲除干净，落个省事。”
<br/>　　此人名叫罗铁，是杨龙停的义子，为人最是自傲狂妄，谁也不看在眼里。敢闯杨龙停门的，也就他一个了。
<br/>　　杨龙停撇撇嘴，无奈道：“叉出去，叉出去，这几天看好了，别让他去风雨亭。”
<br/>　　……
<br/>　　幽州千里地，一到冬天便去了大半人气，小户人家愁吃愁穿，大户人家在宅子里窝着，这个天气也不愿出门。
<br/>　　大户人家的规矩是多的，闺房小姐轻易不许出门，每年也就元宵节、七巧节能出门转转。幽州又偏僻些，没那么多娱乐活动，于是日子更加无聊。
<br/>　　幽州城，某座不知名宅邸，小姐绣楼内。
<br/>　　地上丫鬟的尸体倒在地上，血流了一地，绣床上，这家的小姐一丝不挂，用绣帕勒着嘴，身上却是一丝不挂。
<br/>　　“好妹子，你别叫，把你父母招来，再被爷杀了，你也伤心不是。”
<br/>　　床上，一脸色苍白的男人靠在床头，表情是满足后的疲惫且得意，手中把玩着一柄精致的匕首。说着，匕首在手里漂亮的转个圈，斩下小姐额头秀发一缕。
<br/>　　赛燕子单飞。
<br/>　　幽州道上有名的飞贼，仗着一身好轻功，穿家入户如入无人之境，仗着轻功好，也不和其他人合伙，自己就把买卖干了，于是不把任何人看在眼里。但绿林道很少有人知道，单飞除了是飞贼，还是采花贼。
<br/>　　出道这几年，也不知祸害过多少千金小姐。只不过这种事，即便发生了也属于家丑不可外扬，牙掉了往肚里咽，胳膊折了往袖子里藏，真把人逼死自尽的，家里也会编出个坠井之类的理由掩人耳目。
<br/>　　那小姐倒在床上，吓得脸白如纸，却是一句话都说不出来，只是眼泪一滴滴往下掉。
<br/>　　“听说了么，姓杨的要招安道上的弟兄，等爷去了混个将军什么的，再正大光明的来你家提亲。到时候爷也娶他三十几房的小妾，你也别上火，不管什么时候……”
<br/>　　单飞的手突然拍在女人肩膀上，赤果的身躯颤栗，眼中是说不尽的恐惧。
<br/>　　“你是大房！”
<br/>　　说罢，单飞哈哈怪笑一声，跳窗窜到屋顶，几个起落身形已经在夜幕中消失。
<br/>　　……
<br/>　　野人渡，一座茅草屋，一艘渔船停泊在冰冻的水面上。
<br/>　　“小二的冻伤还没好，再这样下去腿怕是要废了。现在河面冻着，咱们已经大半个月没做过生意了。”茅屋内，一个中年人抽着烟袋，烟锅在桌上敲了敲：“半月前做那桩买卖，把那女人和孩子宰了，却让那男人跑了，不然还能多撑几天。”
<br/>　　他又点了一袋烟，深吸一口气道：“这次姓杨的想招安我们，不知他有什么坏心，但总之是个机会，至少能吃口饱饭。风雨亭咱爷们都去碰碰运气。”
<br/>　　茅屋内没有点灯，只有烟袋的火光一闪一闪的明灭着。男人一边抽烟，一边向几个儿子传授人生经验。
<br/>　　极远出，一座荒废的坟墓中有人探出只手……
<br/>　　更远处，伏虎山上有一猛人在睡梦中睁开眼，翻个身揉揉眼继续睡觉，床头放着两柄大斧头。
<br/>　　幽州王一纸令下，沉睡中的幽州道被惊动了，多少魔王在冬眠中睁开眼睛，想的都是同一件事。
<br/>　　十日后，风雨亭。
