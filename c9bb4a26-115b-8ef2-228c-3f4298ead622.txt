        &lt;h3&gt;第三卷第二百四十一章  接风宴&lt;/h3&gt;
<br/>
<br/>        宁家为宁雅举行了迎接晚宴，宁少天带着宁玥、凌辰和董亚子到达酒店的时候，宁雅还没有到，他们四个人坐着聊了会天，宁雅掐着时间点到。
<br/>
<br/>        “雅儿，你回来了？”宁少天看见宁雅愣了一下，但是想着宁雅在陈子昂公司上班，也就释然了。
<br/>
<br/>        “姐姐，今天请客吃饭，爸爸又没有叫你，你怎么来了？”宁玥看见宁雅感觉很奇怪，出言询问。
<br/>
<br/>        “雅儿，你同事呢？”宁少天见宁雅一个人有些不切断的问。
<br/>
<br/>        “哪有什么同事？你们今天不是请客吗？”宁雅微微一笑，真是此一时彼一时。
<br/>
<br/>        父亲和妹妹的态度真的是一点也不敢恭维。
<br/>
<br/>        宁少天脸色一变，但是很快反应过来了，陈子昂安排宁雅回来和他们谈论合作。
<br/>
<br/>        “是啊，今天请客，但是不是请的你啊，你回来了不是应该回家吗？跑过来干嘛？”宁玥口无遮拦，再加上凌辰在旁边，也让凌辰看看宁雅是多么的不懂事。
<br/>
<br/>        宁雅看了一眼宁玥，她坐在凌辰的旁边，凌辰也在看着她，眼里疼惜让她感觉微微一暖，瞬间又冷了下来。
<br/>
<br/>        还有董亚子，他们四个人可能来了一会，桌上的茶似乎已经更换过，凌辰大概是送宁玥回来，俩个人的感情看来不错。
<br/>
<br/>        宁雅深吸了一口气，稳了稳神，原本她是要坐下的，听见宁玥的话她站住了：“妹妹这是不想让我来，那我回去回复陈总，就说我们宁家看不上她那点小生意，我，这就告辞。”
<br/>
<br/>        “雅儿。”宁少天看着宁雅想走，喊住了宁雅：“你妹妹的性子你又不是不知道，和她一般见识干嘛？”
<br/>
<br/>        “来、来、既然雅儿已经来了，凌少，董总，请坐。”原本是请陈子昂公司的宁雅，立马变成了请凌辰和董亚子。
<br/>
<br/>        凌辰自然而然的就和宁少天坐了主坐，宁玥坐在了凌辰的旁边，董亚子有些为难，她不知道她到底该怎么坐。
<br/>
<br/>        看样子，今天他们请客的人应该是陈子昂公司的人，可是陈子昂却安排宁雅来和宁家商谈合作，而宁家之前似乎不知道情况，这才出现了尴尬的一幕。
<br/>
<br/>        她看了一眼宁雅：“宁总，请！”
<br/>
<br/>        今天的宁雅是代表陈子昂来的，她对陈子昂本身就很尊敬，而且对宁雅在宁家的情况看得一清二楚，心里就倾向于宁雅了。
<br/>
<br/>        宁雅微微一笑，在座的四个人，怕是只有董亚子还能看见她的存在。
<br/>
<br/>        “谢谢，董总，请！”董亚子无论在何种状况都是客人，礼宾是中国人的传统美德。
<br/>
<br/>        宁雅亲切的拉着董亚子的手，将董亚子拉到了宁少天的身旁坐下，她陪坐在董亚子的下首。
<br/>
<br/>        凌辰一直冷眼旁观，一言不发，从宁雅进门之前他就盯着她看，可是宁雅一个眼神都没有给他，连招呼都不带打的，不过这种状况，宁雅也来不及和他打招呼，他有些羡慕董亚子能坐在宁少天和宁雅之间了。
<br/>
<br/>        落座之后，宁少天微笑着说：“今天就我们自己人，感谢凌少和董总对宁玥的照顾，刚好小雅也回来了，原本我想让宁梓来陪你们的，但是宁梓今天有应酬，雅儿和玥儿今天就好好陪陪凌少和董总。你们是年轻人，多交流交流。”
<br/>
<br/>        无酒不成宴，喝酒是少不了的，董亚子家里就是酒厂，凌辰是总代，宁雅号称一斤不倒，而宁少天也是生意人，哪能不喝酒？
<br/>
<br/>        只是宁玥比较年轻，喝酒能少一些，但是有着那样的父亲和姐姐，酒量也差不到哪里去。
<br/>
<br/>        几个人各怀心思，坐在一起，酒，既是拉近人们之间的距离，也是各种矛盾的催化剂。
<br/>
<br/>        宁少天心里藏着事情，他是没有预料到陈子昂会安排宁雅回来和他谈判，陈子昂年纪轻轻，心思却是极其深沉。
<br/>
<br/>        宁雅被他下了管理的位置，跑到陈子昂的公司打工，如今以合作者的身份出现，这不是在打他宁少天的脸吗？
<br/>
<br/>        这个陈子昂和宁雅的关系难道胜过他们之间合作的关系？或者说宁雅的能力远远比他看到的强？他也知道陈子昂是出了名的爱才，进入陈子昂公司的人是很少能被别人挖走的。
<br/>
<br/>        不过商谈合作的人既然是宁雅那就好办了，无论如何宁雅都是他的女儿，也有可能陈子昂是向他们示好，故意安排宁雅回来的。
<br/>
<br/>        宁玥此时也是知道了宁雅是代表陈子昂公司来的，但是再怎么说，宁雅在宁家就是一个不被待见的人，就算是代表陈子昂公司来的又怎么样？这更是说明了宁雅只是一个外人而已，她宁玥才是宁家的掌上明珠。
<br/>
<br/>        凌辰心里有一千只小马飞腾而过，明明请的是陈子昂公司的人，知道是宁雅态度三百六十度大转弯，这种转变真的是让他接受不了，无论是对公还是对私，宁雅都不应该受到如此待遇，宁雅可是宁家的嫡长女，宁玥算老几？她又什么资格坐在他的身边？想归想，但是对于未来老丈人的决定他还是不敢发一言，现在暂且忍耐，等到将雅儿追到手再说。
<br/>
<br/>        宁少天举起了酒杯，要是知道今天是宁雅回来，他就不会来这里了，他一个长辈竟然来陪几个小辈，而且还是他的女儿，这个憋屈！
<br/>
<br/>        明天无论如何也要讨回这个公道。
<br/>
<br/>        “来，凌少，感谢你对宁玥的照顾，我们一起拿起来。”
<br/>
<br/>        “谢谢伯伯，我记得这顿饭应该不是感谢我的，而是为陈子昂公司来的人宁雅宁总接风的，这杯当请宁总先喝。”你不愿意正视现实，我就提醒你正视现实。
<br/>
<br/>        “凌辰哥哥，小雅是我姐姐，哪有父亲敬女儿酒的道理？”宁玥开口了，算是给她父亲一个台阶。
<br/>
<br/>        “伯伯是不可以啊，但是你是可以的，第一，你是妹妹，第二，好像你明天代表你们公司和宁总进行商谈。再说了，当为合作者，无关乎年龄，大小，身份地位。”凌辰一口一个宁总，直接将宁雅的身份拉到了合作的位置上。
<br/>
<br/>        宁雅看了一眼凌辰，心里微微一动，他到底还是站在自己的这一边。
<br/>
<br/>        “凌总，这杯酒应该我来敬大家，第一，敬宁总裁，第二，敬我爸爸，第三，敬凌总和董总，感谢二位对我妹妹的照顾，第四，敬我的妹妹宁玥，感谢这么多年我们姐妹相亲相爱。”宁雅将“相亲相爱”几个字咬得特别重。
<br/>
<br/>        董亚子：“宁雅姐姐，你真的好明事理，来，我们干了这杯酒，你以后就是我的姐姐了。”无论如何，宁雅都是宁家的女儿，不管宁家对宁雅如何，这个是无法更改的事实，只是宁雅能不能扳回她自己的局面，还是要看她自己的能力。
<br/>
<br/>        陈子昂已经给了宁雅的机会，就看宁雅自己如何行动了。
<br/>
<br/>        董亚子是个局外人，看得比他们都清楚。
