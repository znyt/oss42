        章海昌和方浩博立即对马智杰挤眉弄眼，示意他不要多事。
<br/>
<br/>        马智杰瞥了章海昌和方浩博一眼，完全不顾他们的提醒，将手机又还给诺筱颖后，接着说道：“筱颍，你跟我来！”
<br/>
<br/>        他说完，便只身走出了寝室门。
<br/>
<br/>        诺筱颖连忙交代了一下让章海昌和方浩博照看一下她的行李箱，她提起自己的手提包后，立即跟随马智杰而去。
<br/>
<br/>        留下章海昌和方浩博两个人面面相觑。
<br/>
<br/>        诺筱颖跟上了马智杰的步伐，见马智杰心事重重的样子，不由地抿了抿唇，关心地问道：“马智杰，你这是怎么了？”
<br/>
<br/>        “待会见了何明旭后，答应我，不要哭。”马智杰一边往前走，一边深沉地说道。
<br/>
<br/>        其实，这是何明旭和诺筱颖两个人的事，他何必插手？
<br/>
<br/>        但是……
<br/>
<br/>        马智杰想到这里，眉头紧锁。
<br/>
<br/>        诺筱颖以为何明旭出了什么大事，急得嗓子都发出了嘶哑的声音：“是不是阿旭出什么事了？他要不要紧？”
<br/>
<br/>        “他没事，而且，活得很好。”马智杰冷淡地回答。
<br/>
<br/>        随后的这段路程，不管诺筱颖怎么问，马智杰总是用“到了之后，你就知道了”来搪塞诺筱颖。
<br/>
<br/>        诺筱颖不得不识趣地闭上了嘴。
<br/>
<br/>        两人去了学校的后门，后门外面是一条宽敞的马路，马路对面则是一幢又一幢的五六层的居民房和小吃街。
<br/>
<br/>        马智杰带着诺筱颖进了一条胡同，然后拐弯不到十米的地方，有一张不锈钢栏珊门。
<br/>
<br/>        这里进进出出的，都是成双入对的青年男女。
<br/>
<br/>        诺筱颖隐约间明白了什么，但是仍旧不敢去相信自己心里的那个想法。
<br/>
<br/>        马智杰拉开了不锈钢栏珊门走进了去，诺筱颖微微低着头，随后默默地跟了进去。
<br/>
<br/>        他们上了三楼，在一张墨绿色的防盗门前停下了步伐。
<br/>
<br/>        马智杰从裤兜里掏出一把钥匙，开了门，只身走了进去。
<br/>
<br/>        诺筱颖也跟着进去后，才知道，这房子里有四居室，带客厅、阳台、餐厅和厨房，是她和苏漫雪一起租的那个小出租屋的三倍大。
<br/>
<br/>        马智杰径直走到最左边的那张房门前，轻轻地敲了敲后。
<br/>
<br/>        房门里传来了何明旭的声音：“谁啊？”
<br/>
<br/>        “是我！”马智杰应了声。
<br/>
<br/>        何明旭又问道：“你不是说今晚回寝室去住吗？怎么又回来了？”
<br/>
<br/>        “你出来，我找你有事。”马智杰接着说道。
<br/>
<br/>        “那你等会儿，我穿上衣服后再说。”何明旭也应了声。
<br/>
<br/>        紧接着，房门内还传来一个女人的声音：“这个马智杰，真是的，来得一点都不是时候。”
<br/>
<br/>        “亲爱的，别急。等我问了他是什么事情，回来再跟你继续。”何明旭温柔地哄道。
<br/>
<br/>        从房门内传出来的声音并不大，虽不足以听清楚里面的人在说什么，但是却能分辨出里面的人，除了何明旭以外，还有一个女人。
<br/>
<br/>        诺筱颖就站在马智杰的身旁，听的一清二楚。
<br/>
<br/>        房间里除了何明旭，还有一个女人。
<br/>
<br/>        现在还没到睡觉的时间，何明旭为什么要穿好了衣服才能出来？
<br/>
<br/>        诺筱颖鼻子一酸，眼泪瞬间在眼眶里打转。
<br/>
<br/>        当何明旭打开房门后，看到站在马智杰身旁的诺筱颖时，瞬间惊怔了。
<br/>
<br/>        “明旭，你发什么愣啊？”那个女人的声音，又响了起来。
<br/>
<br/>        下一秒，声音的女主人便出现在了诺筱颖的眼前。
<br/>
<br/>        女人长发披肩，唇红齿白，脸上化了淡妆，身上还穿着何明旭的白衬衫，下露两条雪白的大长腿。
<br/>
<br/>        “哟，马智杰。你这书呆子，终于找女朋友了啊！”女人将诺筱颖打量了一番后，红唇微扬地嘲讽。
<br/>
<br/>        马智杰不以为然地斜睨了林若琴一眼，嘴巴刚一张，准备对何明旭说什么的时候，何明旭抢先了一步，打断了马智杰欲要说的话。
<br/>
<br/>        “她是我表妹！乡下来的！”何明旭微笑着说。
<br/>
<br/>        这一刻，诺筱颖心痛到就连眼泪都无法流出来了，只有唇瓣在微微颤抖着。
<br/>
<br/>        “对，你表妹突然过来找你。所以，我就把她带这儿来了！”马智杰冷冷地附和着。
<br/>
<br/>        何明旭微微侧身，温柔地对身旁的女友林若琴说道：“若琴，我先带我表妹去吃晚饭，安顿好她后，就回来陪你。她从乡下坐长途大巴过来，一定还没吃晚饭。”
<br/>
<br/>        “嗯，好。正好，我也不想出去了。就不陪你的表妹了！”林若琴撒娇地说道，顿了顿后，转眼看向诺筱颖，微微一笑，“小表妹，初次见面，你好呀！我是你表哥的女朋友，林若琴。”
<br/>
<br/>        “你们……在一起……多久了？”诺筱颖的声音有些哽咽。
<br/>
<br/>        林若琴腼腆地笑了笑，然后亲密地挽着何明旭的臂弯，回答道：“快一年了吧！旭，对不对？”
<br/>
<br/>        “先不说这些了，我带我表妹吃饭去了。”何明旭连忙岔开话题。
<br/>
<br/>        林若琴微微点头，放开了何明旭的手。
<br/>
<br/>        何明旭从房间里出来，并带关了房门，脸色突然一变，愤恨地瞪了马智杰一眼。
<br/>
<br/>        诺筱颖什么话也没再说了，而是转身就走，甚至大脑不听使唤地跑了起来。
<br/>
<br/>        何明旭连忙追了出去。
