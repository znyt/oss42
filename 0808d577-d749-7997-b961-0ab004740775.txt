厉嫣然悲伤失落的找到了夜滢，她抱着夜滢大哭了一场。

<br/>

<br/>        什么？
<br/>
<br/>        夜滢面色一变，陆寒霆和厉嫣然订婚还没有多久，当时陆厉联姻轰动了整个帝都城，现在陆寒霆竟然要退婚，这事如果传出去大家都会说嫣然是被陆家退掉的，是被抛弃的，到时嫣然会成为整个帝都的笑话，会被别人指指点点一辈子。

<br/>        绝对不可以！

<br/>
<br/>        夜滢沉吟了片刻，今晚她失手了，让夏夕绾给逃脱了，她已经不能再出手了，因为打草惊蛇了。

<br/>        不过，夏夕绾真的要从这个世界上消失！

<br/>        夜无忧回来了！
<br/>
<br/>        毒王夜无忧！

<br/>        厉嫣然迅速忘记了哭泣，震惊的长大了嘴巴，无忧哥哥出关了？

<br/>
<br/>        夜老对这个孙子倾注了所有的心血，夜无忧在几年前就闭关不出了，听说他的毒已经到了出神入化的地步了。
<br/>
<br/>        “妈咪，无忧哥哥真的回来了吗，那太好了，赶紧让无忧哥哥来对付这个夏夕绾，夏夕绾绝对不是无忧哥哥的敌手！”厉嫣然迅速开心的笑道。

<br/>        夜滢也点了一下头，只要夜无忧出手，夏夕绾只有死路一条！

<br/>        “可是妈咪，外公一直不喜欢无忧哥哥参与进女人的这些事里，上一次明珠还被外公给臭骂了一顿，想让外公同意，这个太难了。”
<br/>
<br/>        夜滢目光闪了闪，“这件事你不用管了，我自然有办法说服你外公的，嫣然，你等着吧，很快陆寒霆就会娶你的，我要陆寒霆三媒六聘，满城红妆的来迎娶你，给你一场盛世婚礼，让你成为他的陆太太！”
