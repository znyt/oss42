        &lt;h3&gt;第三卷第一百四十二章  新项目启动&lt;/h3&gt;
<br/>
<br/>        寒伯安兄弟三人的新事业已经启动了！
<br/>
<br/>        他们在旅游发达的城市同时投放几百量档次不同的车辆，在前期招聘的工作人员都经过统一培训，培训期间有从哲担任负责人。
<br/>
<br/>        每一个经过培训出来的工作人员工作态度都是焕然一新。
<br/>
<br/>        在上岗之前郝景文给大家上了一堂课，将服务的重要性讲给大家，也给大家讲了工作的价值性和世界观。
<br/>
<br/>        郝景文是这样给大家安利和画饼的：
<br/>
<br/>        “我亲爱的同事们，我们是一个服务行业，但是我们区别与传统的服务行业，我们以人性和服务为目的，我们面对的是我们的衣食父母，我们服务的对象不分贫富贵贱，只要坐我们车的都是我们的家人，对，我们就要像为我们的家人开车一样，我们要充满爱，让我们的用户体验到温暖、温馨、关怀，要让我们的服务对象觉得坐上我们的车是安全、舒心、放心的。试想一下，如果你坐上一辆车，感受到的是亲切的、温暖的、彬彬有礼的关怀，你会不会幸福感爆棚？”
<br/>
<br/>        大家的掌声响起来，郝景文等掌声停下，他继续说：“我们的服务不但是服务别人，其实是在服务我们自己，我们以爱待人，别人也会以爱待我们，我们给了别人关怀，别人也会给予我们关怀。我们要让这个世界充满爱，充满欢乐！我们要从骨子里去尊重我们的每一个用户，爱别人亦是爱自己！”
<br/>
<br/>        “请记住，我们是用爱在行动！”
<br/>
<br/>        当某一件事情上升到到社会责任感的时候，人们都会认为自己是伟大的，更愿意为了这份事业而去奋斗，因为他们觉得他们不仅仅是为了生活，他们是为了更大的梦想。
<br/>
<br/>        他们会用爱去温暖这个世界！
<br/>
<br/>        在前几批的服务人员里，他们是最成功的人员，他们为了这个公司付出了很多，也为这个公司创出了品牌。
<br/>
<br/>        在他们的这次会议中，他们瞬间觉得自己高大了起来，不但是工作人员，就连身旁站着的寒伯安，江俞轩、陈子昂都觉得自己有了更大的责任感。
<br/>
<br/>        陈子昂的软件还在开发之中，因为他们原来的信息化项目使用单位后期发展的需要，有很多接口多了起来，赵绾绾带领的科研人员已经忙得顾不上软件的开发，所以陈子昂听从了赵绾绾的建议，寻找有能力的合作单位，进行合作。
<br/>
<br/>        他们目前只是一个简单的可以起调度作用的app，对于目前的运营是没有问题的，但是对她设想的那个一系列的套餐服务后期的使用完全没有开发出来，按照目前的状况恐怕没有两三年是出不来的，但是时间是不等人的。
<br/>
<br/>        趁着大家都在，陈子昂将她的困难和顾虑摆出来了，耗材已经开始，后期也会有不停的更新的型号，所以皇甫双双和苟江凌带领的耗材研发会一直继续下去。
<br/>
<br/>        “寒总，郝总、俞轩，今天大家都在，软件目前开发效果不好，原来的项目很多都要重新对接新的接口，目前人员调度不开，再招聘人员一时也没有办法上手，我打算和人合作。”陈子昂对大家说。
<br/>
<br/>        “软件已经交给你了，怎么做是你的事情，我们只是需要尽快的拿到使用权。”寒伯安看着陈子昂。
<br/>
<br/>        “陈总，我们可以说是一家人了，目前软件开发已经交给公司来做，前期的这个app只能使用调度，如果后期的服务要求增加，这个根本不行，所以该怎么做，无论是自己开发还是和人合作，你都得加快步伐。”郝景文如是说。
<br/>
<br/>        江俞轩看了一眼陈子昂，对陈子昂说：“你想要怎么做都行，我们不会阻拦你的，只要有一个好的结果就行。”
<br/>
<br/>        陈子昂听大家这样一说，心里那块石头就落地了。
<br/>
<br/>        “耗材的情况怎么样了？”寒伯安问陈子昂，耗材可是陈子昂一直看好的，也是费了很多心血的。
<br/>
<br/>        “一切都很好，按照预期在进行，其实目前已经在大量的推广了，有些地区已经收获颇丰。而且也可以现在直接推向市场，但是年前事情比较多，只好放年后了。另外，我年后要去甸城发展，郝总到时候帮着打理一下公司。”陈子昂微笑着说，现在在座的都是大股东，反而她成了小股东了。
<br/>
<br/>        陈子昂将她的股份都分出去了。虽然大家拿到的份额都不是很多，但是陈子昂真的是收获了一份人心，在公司遇到困难的时候，很多的人都找到凤盛夏，将自己的资金都拿出来，对当时的公司来说是杯水车薪，但是对他们来说，他们都拿出来自己的全部家当帮公司度过难关。
<br/>
<br/>        这就是凝聚力，这就是同甘共苦！
<br/>
<br/>        “你这是撂挑子啊。”郝景文苦笑着说，他已经很忙了好不好？
<br/>
<br/>        “不是，如果甸城的项目能够拿下来，那个份额会是西城总项目的总和，而我们可以说拿下这个项目可以少奋斗几年。”陈子昂和祁峥通过电话，谷强带着祁峥已经搞定了项目，申请的项目资金基本都下来了。
<br/>
<br/>        “项目是很大，怕也是有风险存在吧？”江俞轩问陈子昂。
<br/>
<br/>        “每个事情都有风险，如果怕风险什么都干不了。”陈子昂第一次抵触江俞轩这样说话。
<br/>
<br/>        “陈总可以放心大胆的去看看，后方有我们支撑，我们是一个生意人，有市场肯定要去发掘。”寒伯安饶有兴趣的看着陈子昂。
<br/>
<br/>        看来陈子昂与郝景文之前分析的一样，只要她认定的事情是不会改变的，为了股东之间的和谐，还是不要太干涉她为好，毕竟她的努力其实都是为了公司，而江俞轩的担心应该是担心陈子昂的个人安危。
<br/>
<br/>        不过陈子昂已经在社会上摸爬滚打这么多年，又怎么可能任人欺负？
<br/>
<br/>        陈子昂听寒伯安这样说，冲江俞轩挤了一下眼睛，嘟了一下嘴“哼”了一声，然后展颜一笑。
<br/>
<br/>        三个男人都被陈子昂那调皮的样子逗笑了，江俞轩偶尔会看到调皮的陈子昂，寒伯安和郝景文每次见到的陈子昂都是意气风发的女强人，哪里见过这么少女心的陈子昂。
<br/>
<br/>        “嘿，子昂，我还是第一次见你这么调皮，我就说嘟嘟的精灵古怪不知道是继承谁的呢。”郝景文莫名的就觉得和陈子昂之间的距离近了一些，这样的陈子昂才接地气嘛。
<br/>
<br/>        陈子昂不好意思的笑着说；“我也是小女孩嘛，不过是老了的小女孩。”
<br/>
<br/>        大家噗嗤的笑了，一场严肃的工作讨论就轻松的结束了。
<br/>
<br/>        陈子昂因为有了寒伯安的那句话心中大安，只等耗材正式推向市场之后，再去甸城，去甸城的计划便再也不会更改了。
<br/>
<br/>        三姐回到西城，她要回去西城看看三姐，便和江俞轩又一起回到了西城。
