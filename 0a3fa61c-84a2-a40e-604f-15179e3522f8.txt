　　阳光落在黑色宝马轿车车顶上，令得这辆黑色车顶周围形成了一圈光晕。
<br/>　　轿车四周的车窗上贴着深色窗膜，从外面看不清楚里面的情况，黑色宝马在油气厂大门口停了下来。
<br/>　　保安室中，警卫看到了准备进入油气厂的黑色宝马，立刻拿着一把霰弹枪走了出来。
<br/>　　在枪械管制并不严格的K国，合资油气厂的警卫配发一把使用电击弹的霰弹枪实在太正常了。
<br/>　　警卫走到驾驶室旁边，敲了敲车窗，示意让里面的智天使出示证件，接受检查；但是警卫的要求并没有得到回应，警卫又敲了敲车窗，重复了一遍自己刚才的要求。
<br/>　　这次智天使降下了车窗，但是从里面递出来的，并不是进入油气厂厂区的证件，而是一把安装了消音器的手枪。
<br/>　　看到指着自己的黑洞洞的枪口，警卫顿时慌了神，惊恐的向后退，右手伸向腰间子弹盒，从子弹盒中拿出一颗独头弹，打开弹仓，试图把里面的电击弹替换出来，但是独头弹从他的手里滑落在地上。
<br/>　　警卫慌乱着想要再从子弹盒中拿出一颗子弹，可是智天使并不会给他这个机会。
<br/>　　“biu！”，“biu！”
<br/>　　“噗！”，“噗！”
<br/>　　智天使扣动扳机，警卫胸口血花飞溅而起，涌出的鲜血很快就将警卫身上的制服给浸透，胸前一片嫣红，在阳光下显得格外刺眼。智天使面无表情的收回消音手枪，升起车窗，驾驶着汽车进入油漆厂厂区。
<br/>　　杀掉一个警卫，对于智天使来说根本不会有什么触动。因为他杀掉过太多太多的人，多到他已经完全记不得自己杀了多少人。
<br/>　　智天使使用消音手枪枪杀警卫，期间没有枪声，可是却被安装在油气厂大门口的监控摄像头给拍了下来；很快，油气厂中就发出凄厉的警报声，油气厂中的警卫拿着换上了鹿弹的霰弹枪从警卫室冲出来。
<br/>　　嗤啦——！
<br/>　　智天使猛打一把方向，黑色宝马车在厂区空地上做出一个漂移，两道黑色的橡胶印痕分外明显。
<br/>　　吴云推门下车，打开后备箱，拿出里面的HK自动步枪和塞满弹匣的战术背心，将其中一件战术背心穿在自己身上，将剩下那件战术背心扔给了智天使，哗啦一声，拉开自动步枪枪栓，把子弹上膛，小幅度转身，抬手瞄准，开枪射击。
<br/>　　哒哒！哒哒！哒哒！
<br/>　　三个点射，三名拿着霰弹枪的警卫中弹倒地，鲜血“嗞嗞”的往外冒着，很快就没有了生机。
<br/>　　这座合资油气厂临海而造，智天使所制定的撤退计划的最后一步，就是坐船经由油气厂背后的海路逃离。
<br/>　　哒哒！哒哒！哒哒！哒哒！哒哒哒哒！
<br/>　　密集的枪声回荡在这座油气厂的上空，每一个点射声响起就意味着有一个警卫中弹倒地失去生命，这些警卫经过专业的训练，会简单的擒拿格斗和轻武器使用，对付当地一些混混，一个打几个都不是问题，但是面对智天使和吴云这种人，瞬间就变成了“战五渣”一样的存在。
<br/>　　十几个警卫只有一人扣动了霰弹枪的扳机，但还是在五十米外的距离，这个距离下用装填了00号鹿弹的霰弹枪射击，当铅弹打在吴云身上的时候，基本已经对他造成不了杀伤。
<br/>　　而那名警卫则是被吴云甩出的军刀刺穿心脏，到底抽搐了几下后便气绝身亡。
<br/>　　接到从合资油气厂传来的警报后，李然砸了一下拳头，想明白了智天使的撤离意图，带着几名战备一队队员搭乘K国快速反应部队的直升机前往合资油气厂。
<br/>　　数分钟后，李然和几名战备一队队员抵达合资油气厂上空，直升机缓缓下降高度，被螺旋桨卷起的尘土吹在地面上那些警卫尸体上，被尸体上黏稠的血液黏住，眨眼间，那些警卫尸体上便是灰蒙蒙的一层。
<br/>　　直升机还没完全落地，李然和几名战备一队队员便是持枪跳下直升机朝着油气厂背后的悬崖冲去，悬崖背后就是大海，如果智天使打算经由海路逃跑的话，只能是从那里离开。
<br/>　　李然带着四名战备一队队员赶到油气厂背后悬崖的时候，看到吴云正在将连接着悬崖上滚轮支架的绳索跟自己腰带上的D型锁扣相连。
<br/>　　“吴云！智天使！”
<br/>　　李然手中端着沙色HK416自动步枪，对着跟自己只隔了一道铁丝网的智天使和吴云大吼一声，全息瞄准镜中的红点停留在智天使身上，手指扣在扳机上，只要智天使稍有异动，李然绝对会毫不犹豫的扣下扳机。
<br/>　　智天使扭头朝着自己身后碧蓝色的大海看去，海面上一艘快艇正在以极快的速度朝着油气厂的方向靠近，毫无疑问，这艘快艇就是智天使和吴云用来撤退的工具。
<br/>　　“你们比我预计抵达时间要早那么一点点。”智天使比着手指说道：“但是这并不能改变什么。”
<br/>　　“少他妈给老子废话！”
<br/>　　李然举着枪，对着智天使大声说道：“立刻解开绳索，给我从那边过来！”
<br/>　　说话间，汗水顺着李然两侧脸颊流淌下来，汇聚到他下巴处，然后再沿着他的脖子流入他的衣领。
<br/>　　海面上快艇距离油气厂悬崖已经很近，站在李然的位置已经能够看到站在快艇上的人影。
<br/>　　“我再说一遍，立刻解开绳索给我滚过来！”李然对着智天使说道：“除非你认为你的索降速度能快过我的子弹，否则你的下场就会跟油气厂中那些警卫一样。”
<br/>　　“不要把我跟那些警卫比，这根本就不是一个层面比较。”智天使解开了自己身上的绳索仍在地上，但是在解开绳索的时候，他把一枚V40手雷扣在了手心中，看着李然说道：“妖瞳你认为你这次抓住我，但是我却并不认同你的看法。智天使做事总是会把一切可能发生的情况考虑在内，并且制定出相应对的方案，比如像这样从眼皮子地下逃走……”
<br/>　　说罢，智天使和吴云后仰，从悬崖边上落下……
