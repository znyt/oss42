　　照片上的男子正是这次进入王权酒店，把佟千雪从酒店中带走的阿扎普。
<br/>　　“他叫阿扎普，以前是M市的一个出租司机，他的老婆提供‘租妻’服务过程中怀孕，在去诊所进行流产手术的时候，发生了意外，他现在跟着一个叫普提查的人干。”
<br/>　　葛铭又从遮阳板上抽出一张照片递给罗昊，说道：“这就是普提查，毒贩，手下有近200人的势力，被T国警方通缉了几年，可是依旧活得好好的。”
<br/>　　“面部轮廓识别？”
<br/>　　罗昊拿着照片，颇有些不解的对葛铭问道：“你们还有这样的技术？”
<br/>　　“我们当然没有。”葛铭踩下离合器，踩刹车，放慢车速保持着跟前车之间的车距，说道：“但是T国警方有啊。”
<br/>　　“你的意思说，T国警察中也有我们的人？”
<br/>　　罗昊这次是真的感觉到意外了，他知道龙隐部队驻境外的特勤小队队员从事各行各业，建立起情报系统，收集有价值的情报。其中有从商的，有做老师的，有开餐厅、开咖啡馆的，也有像葛铭这样做司机的，但是进入他们政府系统中工作，罗昊是第一次遇到。
<br/>　　“T国警察系统没有你想象中那么复杂，要进去也没有那么难。”葛铭从口袋里摸出一支烟，塞到嘴里，但是没有立刻点燃，说道：“这些代价还是值得的。”
<br/>　　说着，葛铭去摸口袋里的打火机，可是还没等他把打火机摸出来，他嘴里的烟就被罗昊拿走，折断，仍在了地上，取出一块口香糖递给葛铭，说道：“忍不住了就嚼点口香糖，这东西也没什么营养，但总比抽烟好。”
<br/>　　葛铭看着罗昊递给自己的口香糖，苦笑，他们经常会熬夜，熬夜的时候，咖啡、浓茶都尝试过，但逐渐的这些东西就不再有作用，最后还得靠抽烟来解决。
<br/>　　烟越抽越多，牙齿越来越黄，烟瘾也越来越大……
<br/>　　葛铭最后还是没有拒绝罗昊的好意，剥开包装，把口香糖塞进嘴里，克制一下烟瘾。
<br/>　　旅游大巴车转进一条小路后，往前行驶了没多远就停了下来。
<br/>　　司徒空走下大巴车，看着街道两边的门面和广告牌，感觉自己来到了一条“红灯街”。在T国这个se情行业完全合法国家，这些招揽生意的广告牌也是制作的毫无掩饰，你不需要看懂上面跟蚯蚓干一样的文字表达的是什么意思，因为你只需要看广告牌上的图片，就能很清楚的了解到广告牌要表达的意思是什么。
<br/>　　葛铭走到一间夹在两个“SPA中心”的二层楼房前，从门口的花盆下摸出一把钥匙，把门打开，示意战术中队的队员进屋。
<br/>　　“这里不会就是我们的安全屋吧？”
<br/>　　司徒空看着自己眼前普通到没有任何特色的屋子，不住撇嘴。
<br/>　　“这是我们在T国的安全屋之一。”葛铭介绍道：“平时我们都有各自的工作，所以也不会住在这里，只有在有需要的时候才会来这里。其他人晚点会到，你们可以在这里先休息一下，这里有方便面，有需要的可以自己动手。”
<br/>　　“这还是算了，这里方便面那股酸甜辣，再加上海鲜的混合口味，我无法享受，还好在飞机上吃了东西，还不觉得饿。”司徒空撇嘴道。
<br/>　　葛铭闻言，笑了笑，打开冰箱从离开拿出一罐啤酒，打开后往嘴里灌了一口。葛铭很理解司徒空此时对T国食物风味的感觉，他当初刚来这里的时候，也是同样的感觉，那股子酸甜辣还混合着海鲜味的口味，让葛铭发自内心的拒绝，那个时候，葛铭是无比怀念国内某品牌的红烧牛肉面，那个时候就算是能喝口面汤，对于葛铭来说都会是一种享受。
<br/>　　不过时间长了，逐渐适应了这里的饮食风味，葛铭也不再排斥T国的“重口味”饮食风味。当然在葛铭自己住的地方，他准备了一些香辣口味的方便面，偶尔改善一下自己的口味。
<br/>　　葛铭从一个箱子里拿出好几个桶面，扔给坐在沙发上的战术中队队员，说道：“国内货，不是这里的酸甜辣海鲜味方便面。”
<br/>　　罗昊刚把面汤喝完，门外就传来了门铃声，葛铭从门口鞋柜中摸出一把手枪，轻轻拉开套筒，把子弹上膛，走到门口，透过门上的猫眼往外看了眼，把手枪塞到腰后，打开了门。
<br/>　　一个穿着T国警察制服，手臂下夹着一个警帽的男人走了进来，这个男人应该就是之前葛铭所说的，混入T国警察系统中的自己人。
<br/>　　从男人身上的警服判断，他的级别应该还不低，绝对不是那种普通的基层警察。
<br/>　　男人走进屋子，啪的一声站直身体，对着客厅中的战术中队众人敬了一个礼，说道：“驻T国特勤小队副队长何帆！”
<br/>　　“战术中队队长，罗昊！”罗昊站起身向何帆回了一个礼，向何帆介绍道：“副队长齐凡、副队长梅庆伟……”
<br/>　　“我听说过你们太多的事情。”何帆显得有些激动，说道：“你们所参与的那些行动，只是听着就让人血脉喷张。”
<br/>　　“大家只是分工不同，我们的行动可是离不开你们的情报。”罗昊说道：“没有你们的情报支援，我们就是睁眼瞎，什么事情都做不了。”
<br/>　　“葛队长、何队长。”梅庆伟把普提查和阿扎普的照片放在茶几上，说道：“这次绑架行动很显然是有预谋的行动，而且龙头分析认为，这次的绑架可能并不像我们所看到的那么简单。”
<br/>　　“这个普提查被T国警方通缉了几年。”
<br/>　　何帆手指在普提查的照片上点了点，说道：“T国警方也对他采取过几次行动，但是都失败了，比起国内，这里警察的差得不是一星半点。行动失败，第一是T国警方本身能力问题；第二，是因为在这个普提查背后还有人在支持着他的活动，他们的武器装备比起T国警方要更加先进，他们甚至还拥有AT4反坦克导弹和‘毒刺’防空导弹！”
