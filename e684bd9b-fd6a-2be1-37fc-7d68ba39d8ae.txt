        &lt;h3&gt;第三卷第一百二十二章  相亲相爱&lt;/h3&gt;
<br/>
<br/>        潘青松的工作越来越紧张，也越来越忙，之前的那种可以喝着茶看着窗外风景的日子没有了，是没日没夜的调查、分析、寻找证据。
<br/>
<br/>        他是一个认真而努力的人，自从张函来了一趟西城之后他发现对体制内的要求也是越来越严，整个风气是越来越好。
<br/>
<br/>        他对手下的工作人员的要求是：360度杜绝一切不良作风，整风肃纪，先从内部纪律抓起。目前对他们来说都是一些小鱼小虾，可是小鱼小吓也得撒网和收网。
<br/>
<br/>        他忙得不亦说乎，他的老婆也忙得不亦说乎，俩个人差不多一两个月都见不到面，这天，他疲惫的回到家，发现老婆陈子澜的旅行箱放在客厅。
<br/>
<br/>        “老婆，老婆你回来啦？”潘青松开心的喊着。
<br/>
<br/>        “嘘！”陈子澜穿得整整齐齐的从房间走出来，她在打电话。
<br/>
<br/>        他静静的看着陈子澜，他们认识很多年了，可是她的眉眼还是那么好看，身上的制服穿在身上还是那么英姿飒爽。
<br/>
<br/>        看着她这个样子，他知道她要去出差。能让老婆出差的在泉城来说都是大事情。
<br/>
<br/>        他悄悄的放下手中的包，走到洗手间，打开水龙头，关上门，让外面听不到水流声。
<br/>
<br/>        他们都明白彼此的工作性质，从来都不会去过问彼此手中的事情，不是说他们彼此不信任，而是他们对他们身上的制服负责，他们都要守口如瓶，哪怕是自己的爱人。
<br/>
<br/>        陈子澜打完电话，潘青松也从洗手间出来了。
<br/>
<br/>        “青松，我要出差。”陈子澜满眼满心都是潘青松。
<br/>
<br/>        “老婆。”潘青松走过去拥着陈子澜。
<br/>
<br/>        陈子澜静静的靠着潘青松，他们也有很多天没有见面了，潘青松的，怀里很温暖。
<br/>
<br/>        “我要出差了。”陈子澜轻轻的说。
<br/>
<br/>        “我知道的，老婆，出门一定要小心，我们的年龄都大了。”潘青松舍不得松开陈子澜。
<br/>
<br/>        “知道了，你放心。”每次只要是陈子澜出远门，潘青松都放不下。
<br/>
<br/>        “我这次是去甸城。”陈子澜说。
<br/>
<br/>        “甸城？”潘青松送开陈子澜，紧紧的盯着陈子澜。
<br/>
<br/>        “是的，不过你放心，我们过去就是调查一下，那边已经联系好了，会配合我们的。”陈子澜看着潘青松那忧虑的眼睛说。
<br/>
<br/>        “你一定要注意安全。”潘青松再一次叮咛。
<br/>
<br/>        “我们这一次过去四个人，就是配合调查一下，那边有人配合，放心吧。”陈子澜伸出手替潘青松整理了一下衣服。
<br/>
<br/>        “我该走了，他们在机场等我，我就是想回来看看你。”陈子澜转身拉起行李箱。
<br/>
<br/>        “我送你。”潘青松拿过陈子澜手中的旅行箱送陈子澜出门。
<br/>
<br/>        陈子澜也没有推辞，她也想和潘青松多呆一会。
<br/>
<br/>        “最近见潘毓了吗？”陈子澜问潘青松。
<br/>
<br/>        “没有，我好久都没有看到他了。就是前几天给我打了个电话，报了平安，你呢？”
<br/>
<br/>        “一样，就是打打电话，算了，孩子大了就由着人家发展。”陈子澜微笑着说。
<br/>
<br/>        他们的儿子是他们的骄傲。
<br/>
<br/>        俩个人家长里短的聊着天，谁都没有问各自工作上的事情。
<br/>
<br/>        “二哥，还是不回西城？”
<br/>
<br/>        “二嫂已经将离婚的协议发给二哥了，二哥签字发给二嫂了，他们等于是离婚了，二哥算是净身出户了。”陈子澜叹了口气。
<br/>
<br/>        “他们还真的要离婚？”潘青松惊问。
<br/>
<br/>        “二嫂的个性强硬，这一次也不知道为什么二哥也不示弱，俩个人就越来越僵。二嫂说要离婚二哥也不去挽回。”陈子澜还是觉得很可惜，好好的俩个人，说散就散了。
<br/>
<br/>        “那二哥现在怎么办？”潘青松也是很替陈子建惋惜，但是想到陈子建找到他，让他陪着去相关部门退那些东西的时候，心里竟然莫名的轻松了一下。
<br/>
<br/>        汽车行驶在高速路上，外面的夜黑漆漆的，窗外有雪花在飞舞。
<br/>
<br/>        “二哥说他最近这一段时间刚好休息休息，他最近在整理他的工作经验，准备出书，对后面的人来说也是一种借鉴。况且爸爸妈妈年龄大了，刚好他可以陪陪他们。”陈子澜看着外面飞舞的雪花，皱了一下眉头。
<br/>
<br/>        “哈哈，他现在倒是轻松了。”潘青松笑了起来。
<br/>
<br/>        “子霖弄的那些事情二哥刚好也可以帮帮忙，这一段时间二哥回去了，爸妈他们都开心了很多。”陈子澜也笑着说。
<br/>
<br/>        “没有想到，二哥说放下就放下了，真的很佩服。”潘青松感慨的说，在一个位置上那么久，说放下就放下，这份xong襟不是一般人能做到的。
<br/>
<br/>        “我估计是二嫂伤了二哥的心，按照二哥以前对二嫂的态度，要是小事情二哥是会原谅二嫂的。”
<br/>
<br/>        潘青松：“有一次，我们去二哥家聚会，二哥好像动手打了二嫂，好像还是当着江俞轩的面打的，也不知道二嫂是不是说了子昂啥。就是那一次引起的俩个人的矛盾，二嫂回娘家了，二哥发现二嫂之前背着二哥收了很多东西，二哥找到我，让我陪着他上缴了。”
<br/>
<br/>        “如果是这样的，我到觉得他们分开也没有什么可惜的，人的贪欲只要一激发便会无止境，只怕二嫂在这条道上会越走越远。”陈子澜心里还是有些惋惜。
<br/>
<br/>        “可不是，你看看二嫂，一副趾高气扬的样子，将孩子也养歪了，好像天下就他最大，见了我们这些长辈都是鼻孔朝上。”潘青松说，不过话刚出口，他感觉不对，于是对陈子澜说：“老婆，我是不是说错话了？”
<br/>
<br/>        “你没有说错话，这也怪二哥，要不是二哥无底限的宠着二嫂，什么事情都由二嫂，也不至于是现在这个样子，当初，爸将家里值钱的东西都给了他们，你看看他们帮过家里一分钱没有？就算是离婚了，二哥的工资卡还在二嫂的手中。要不是老家有一套院子，爸爸妈妈还在，二哥怕是要睡露天地了。”陈子澜深深的叹了口气，夫妻之间不是无底限的忍让，应该是平等的、互补的。
<br/>
<br/>        “老婆，我很庆幸遇见了你，是你一直督促着我，是你给了我动力，你，是我的榜样。让我一直都是直的。”潘青松总是不时的拍一下老婆的马屁。
<br/>
<br/>        “你本来就是直的，我也不用费力将你扳直，再说如果从根子就开始养歪了，想扳回来也是很难的。”陈子澜微微笑着，她也很庆幸自己嫁给了潘青松。
<br/>
<br/>        “嗯，老婆真好，只有你从始至终知道我是直的。”潘青松哈哈大笑。
<br/>
<br/>        “好好开车。”陈子澜脸上一热。
<br/>
<br/>        “嗯，我会为老婆好好服务的，一定将车开好。”潘青松严肃的看着前方，一本正经。
<br/>
<br/>        看着潘青松严肃的样子，陈子澜真想拉着潘青松给他松松筋骨。
<br/>
<br/>        到达机场，同事们已经等着陈子澜了。
<br/>
<br/>        潘青松还是不忘秀一下恩爱；“老婆，香香。”
<br/>
<br/>        “这么多人。”陈子澜无奈至极。
<br/>
<br/>        “怕什么，我们是老夫老妻！”潘青松表示很委屈。
<br/>
<br/>        “乖了，回去等着我回来。”陈子澜接过潘青松手中的行李箱朝安检走过去。
<br/>
<br/>        看着妻子愈走愈远，潘青松的心也提得越来越高。
