　　“我没有听错吧？他居然敢和屠苏家主这么说话？”

<br/>　　众人既感到惊叹，同时又百思不得其解。
<br/>　　而那屠苏玄，早已经气得肺都要炸开了，

<br/>　　区区一个小辈，竟然敢说饶了自己，自己这个屠苏家的家主，什么时候都变得这么不值钱了？
<br/>　　“小子！你找死！！”

<br/>　　“化神之下，皆为蝼蚁，那小子似乎只有金丹境，竟然敢和屠苏家主叫板，我看他待会死都不知道怎么死的！”

<br/>　　众人议论纷纷，几乎所有人，望向叶尘的目光中，都是一副幸灾乐祸的神情。

<br/>　　无论是屠苏玄，还是叶尘，显然没有想到，冷蝉衣会在这个时候站出来。
<br/>　　屠苏玄见状，不由得眉头一皱，
<br/>　　冷蝉衣本身不值一提，可她是云丹宗宗主最得意的弟子，也是如今云丹宗的第一天才，若是伤了她或是杀了他，即便是他屠苏玄，也不好交代。

<br/>　　轰！

<br/>　　蹬蹬！
<br/>　　冷蝉衣不由得倒退了两步，俏脸有些发白，不过去并没有退缩，反而应向屠苏玄的目光，
<br/>　　“屠苏家主，我奉师尊之命，要请此人去云丹宗一趟，所以您现在，还不能杀了他！”
<br/>　　冷蝉衣这话倒是没有瞎说，她的确是奉了云丹宗宗主的命令，特地来请宝丹的主人，前去云丹宗做客，只不过她怎么也没有想到，这个人居然会是叶尘。

<br/>　　屠苏玄冷哼一声，犹如雷鸣般轰轰作响，
<br/>　　“小丫头！你少拿宗主来压我！此人先是残害我儿元庆，方才又当众对我不敬，本家主若是不加以惩戒，岂不是要被天下人耻笑？”

<br/>　　“让开！”

<br/>　　轰隆！
<br/>　　一道硕大的巨掌虚影，朝冷蝉衣横扫而来。
<br/>　　冷蝉衣顿时脸色大变，显然没有想到，屠苏玄，竟然真的会对自己出手！

<br/>　　冷蝉衣不由得再次惊呼了一声。

<br/>　　叶尘竟然既不抵挡，也不躲闪，硬生生替她接下了那一掌。

<br/>　　冷蝉衣忍不住低声问了一句。
<br/>　　叶尘随手拍了拍衣衫上的灰尘，淡淡一笑，道：

<br/>　　叶尘这话一出口，全场顿时一片死寂。

<br/>　　屠苏玄更是气得一佛出世，二佛升天，感觉整个胸膛都要炸裂开了，双眸也变得赤红无比，
