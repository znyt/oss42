老夫人陷入了昏睡，虽然气息虚弱，但是已经脱离了危险。
<br/>
<br/>        夏夕绾整个人都瘫在了地上，这么温暖的气温里她竟然出了一身的冷汗，刚才的情况太惊险了。
<br/>
<br/>        夏夕绾拿出湿巾给老夫人擦干净了脸上的血，然后起身，她要去叫人把老夫人抬到病房里进行观察。
<br/>
<br/>        夏夕绾先走开了。
<br/>
<br/>        ……
<br/>
<br/>        这边夏夕绾刚走，有一个人就过来了，是夏妍妍。
<br/>
<br/>        夏妍妍刚到研究院，这时就看到有一个老夫人躺在地上，她迅速上前，想查看什么情况。
<br/>
<br/>        这时陆续有人走了过来，因为上班时间到了，大家都来研究院了，“妍妍，发生什么事了，这个老夫人怎么了？”
<br/>
<br/>        “这个老夫人看起来像是刚突发了疾病，妍妍，你是不是给这个老夫人做了急救？”
<br/>
<br/>        “妍妍，你真厉害啊，刚上班就救了一个老夫人。”
<br/>
<br/>        夏妍妍被众星拱月的围住，大家都误以为是她救了这个老夫人，对她赞不绝口，她一开始还想解释的，但是她已经很久没有尝过这种被围绕的滋味了，她就像是被一个天上掉下来的惊喜给砸中了，完全不想解释了。
<br/>
<br/>        这时周萍周主任带着人手匆匆的赶了过来，“徐老夫人！徐老夫人！徐老夫人是我们研究院里收的一个重症患者，刚才她突然疾病走了出来，刚才是谁救了她？”
<br/>
<br/>        周萍很紧张这个徐老夫人，看来徐老夫人的身份不简单，夏妍妍双眼一亮，迫不及待的站了出来，“周主任，是我救了这位老夫人。”
<br/>
<br/>        周萍看向夏妍妍，“你怎么救徐老夫人的？”
<br/>
<br/>        “我用的针。”
<br/>
<br/>        夏妍妍也是有医术的，刚才她已经看出这个老夫人身上有针灸的痕迹了，关于针灸，她是很有自信的。
<br/>
<br/>        周萍查看了一下徐老夫人，很快就看到了徐老夫人头部被针灸过的地方，她面色大变，无比震惊的看着夏妍妍，“妍妍，你竟然会金针封穴？”
<br/>
<br/>        金针封穴，这话一出，大家都抽吸了一声。
<br/>
<br/>        现在的中医针灸大都都是用的银针，像这种纯黄金打造的金针十分的柔软，如果不是医学大师，很难驾驭这种金针。
<br/>
<br/>        现在能驾驭金针的估计用手指数都能数的过来，而且都在帝都，夏妍妍绝对是海城第一个能驾驭金针的人。
<br/>
<br/>        而且金针封穴，这可是上古针法，精湛绝伦。
<br/>
<br/>        周萍和大家都震惊的望着夏妍妍。
<br/>
<br/>        听到金针封穴，夏妍妍迅速挺直了自己的腰背，因为她也会金针封穴。
<br/>
<br/>        她一直在苦练这个金针封穴，虽然还没有机会实践，但她觉得自己已经能完美驾驭了。
<br/>
<br/>        夏妍妍不知道刚才救了徐老夫人的人是谁，现在也没有知道的必要了，救人的就是她，她要抓住这个机会好好的表现一番。
<br/>
<br/>        “周主任，没错，我会金针封穴，刚才我就是用金针封穴救了这位徐老夫人！”夏妍妍笑道。
