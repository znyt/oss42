　　甚至是因为星域传送阵的存在，殿王城中，就连其他星域的武者都可以看见。
<br/>　　无数异族，行走在街上。
<br/>　　楚风眠一眼看去，都看到了像是一些古老的种族，海族，蛮族，灵族等等，甚至许多大罗天星域都没有的种族，这里都可以见到。
<br/>　　这也算是让楚风眠开了眼界。
<br/>　　“有着星域传送阵，纵然是在数十个星域的武者，都可以通过星域传送阵到来，进行贸易，这殿王城便是因此繁华的。”
<br/>　　海公主看来并非是第一次来到这殿王城，对楚风眠开口解释道。
<br/>　　“这殿王城，是由阎罗星域四大家族，一共管理的，在其中不允许任何的争斗，不然就会被四大家族，一同镇压。”
<br/>　　“恩。”
<br/>　　楚风眠随意点了点头道。
<br/>　　“走，我们先逛逛这殿王城。”
<br/>　　楚风眠也是第一次来这殿王城，自然是要好好逛一逛，楚风眠要去沧海星域，也不差这么一会时间。
<br/>　　而且楚风眠的心中，也有着他自己的想法。
<br/>　　“如果可以找到一件蕴含巨大空间的须弥空间，我也就可以带走这太古蜈蚣，不然就只能够让他回到剑道门之中去了。”
<br/>　　楚风眠步入到了殿王城之中，也在考虑着太古蜈蚣的问题。
<br/>　　偶然降伏了这太古蜈蚣，对楚风眠来说，也算是意外之喜，超出了楚风眠的计划。
<br/>　　本来在楚风眠的计划之中，他跟海公主应该是乘坐星域传送阵，直接前往接近沧海星域的。
<br/>　　可星域传送阵，可带不走这太古蜈蚣，毕竟太古蜈蚣的身躯太过巨大了，足有千丈之长，可没有如此巨大的传送阵。
<br/>　　除此之外，还有一个办法，便是得到须弥空间，将这太古蜈蚣放入到须弥空间之中，然后随着楚风眠前往沧海星域。
<br/>　　须弥空间，跟空戒极为相似，但是炼制手段，其中的种种玄妙，要比起空戒玄妙万倍。
<br/>　　须弥空间唯一的一点，便是可以承载活物。
<br/>　　像是空戒，只能够是盛放死物，当然灵药不在此列，但是武者，妖神，乃至妖兽，只要是进入到空戒之中，便是唯有死路一条。
<br/>　　曾经楚风眠倒是也得到过一件黑盒子状的须弥空间。
<br/>　　不过那须弥空间其中的空间太小了，仅仅只能够是放入几人，想要将这太古蜈蚣放入其中的不可能的。
<br/>　　以这殿王城的繁华，楚风眠还真想要找找，看看能不能找到能够将这太古蜈蚣放入的须弥空间，能够找到，那这太古蜈蚣的问题，也足以迎刃而解了。
<br/>　　当然不止是因为太古蜈蚣，楚风眠自己也想要见识见识这殿王城的繁华。
<br/>　　海公主看着楚风眠的态度，也是陪着楚风眠逛起了殿王城。
<br/>　　前往沧海星域，也不着急这一时。
<br/>　　由她带着楚风眠，走在这殿王城的街道之中。
<br/>　　殿王城繁华，规矩却不算森严，路边有着不少的武者，坐在那里，在叫卖着面前的东西。
<br/>　　许许多多的商会，店铺，在这殿王城之中，也是拥有无数，单单是商会，殿王城中，就有上千个。
<br/>　　不过对现在的楚风眠来说，大部分的诸天奇物，都对他没有多少用处了，至于一些灵药，宝物，楚风眠更是不放在眼中。
<br/>　　想要在这街道上淘到好东西可不容易，毕竟大部分的宝物，都被有眼界的人，早就买走了，剩下的这些，也都是一些平平之物。
<br/>　　楚风眠也只是随意逛着，一直是走到了一座高大的木楼前。
<br/>　　这木楼，足有九层，牌匾之上，只有一个大大的陈字。
<br/>　　从这木楼之中，来回进出的人，络绎不绝，算是这殿王城之中，最为繁华的商会之一了。
<br/>　　“这是陈家的商会，陈家是阎罗星域的四大家族之一，这陈家商会是殿王城中最大的商会之一了，公子要不要去看看？”
<br/>　　海公主在一旁，看着楚风眠看向了这木楼，开口问道。
<br/>　　她现在俨然一副侍女的样子，跟随在了楚风眠的身边，不过现在楚风眠降伏了沧海都，答应放过了她一系的弟子，又是答应她去沧海星域杀人，算是对她大恩了。
<br/>　　海公主心中也感谢楚风眠，才会如此恭敬。
<br/>　　“陈家的商会？进去看看。”
<br/>　　楚风眠看了一眼这木楼，便是踏入了其中。
<br/>　　在外看来，这木楼就有九层，极为宏大了，可一旦是踏入其中，便是可以发现，这木楼之中，还另有乾坤。
<br/>　　也是九层，但是看起来，却是一座九层宝塔，比起那外表看起来的木楼，气势宏伟的多。
<br/>　　一座座古老的院落位于其中，摆满了无数的宝物，无数武者妖神都在其中做着交易。
<br/>　　“二位前辈不知要些什么？我们陈家商会，一切应有尽有。”
<br/>　　一名陈家商会的弟子，看到楚风眠跟海公主踏入其中，急忙的迎了上来说道。
<br/>　　他见人无数，一眼就看得出来楚风眠跟海公主身上的气势不凡。
<br/>　　纵然是现在楚风眠跟海公主已经是收敛起了身上的气息，可他一眼也看得出，楚风眠跟海公主的来历不凡，不敢怠慢。
<br/>　　“带我们去第九层。”
<br/>　　比起楚风眠，海公主对着陈家商会熟悉不少，直接开口道。
<br/>　　这陈家商会，一共九层，层数越高，便是代表卖的东西价值越高。
<br/>　　以楚风眠跟海公主的身份，自然是要去第九层。
<br/>　　不过这陈家商会的第九层，可没那么好去的，几乎都是九劫古帝巅峰之上的人物，甚至是半步天人境的无敌强者，才有资格前去第九层。
<br/>　　许多九劫古帝，都只能够是前去第八层，没有资格进入第九层，能够进入第九层的九劫古帝，都是踏入巅峰，有资格冲击半步天人境的存在。
<br/>　　这样的人物，绝对都是名震天下的强者。
<br/>　　这陈家弟子，听到海公主的话，也是不由的打量了楚风眠跟海公主一眼。
<br/>　　他虽然是看出楚风眠跟海公主绝对的来历不凡，可也没有绝对，楚风眠跟海公主是这样的人物。
<br/>　　他也见过一些有些身份的人物，不知天高地厚，妄想前去第九层的，现在听到海公主的话，心中不由的有些轻视说道。
<br/>　　“不知二位的身份是？我们陈家商会的第九层，必须要问清身份，才可以进入其中。”
<br/>　　“陈家商会，没有这一道规矩吧？”
<br/>　　楚风眠淡然的说道，同时散发出一股气息，轰然笼罩在了这陈家弟子的身上。
<br/>　　这陈家弟子，也是一名一劫真帝，在陈家弟子之中，已经算是境界不低的了，不是一个陈家商会的小厮那么简单，
<br/>　　可是当他感受到楚风眠的气息笼罩而来的一刻，都感觉到是有着一种喘不过气来的感觉，浑身不由的战栗了起来，瑟瑟发抖。
