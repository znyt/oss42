        &lt;h3&gt;第三卷第一百五十三章  别样的表白&lt;/h3&gt;
<br/>
<br/>        郝景文到达他和陈思宇约好的地点时，陈思宇已经到了，毕竟郝景文是老板，她不能让郝景文等她。
<br/>
<br/>        在路上她就很忐忑，她将最近的工作做了一个总结，如果俩个人就工作方面讨论的话，应该能有四十多分钟的时间，讨论完之后她还能早点回去吃晚饭。
<br/>
<br/>        但是到了地点之后，她发现，晚餐估计得和郝景文一起吃了。
<br/>
<br/>        她是一个女生，也没有谈过恋爱，在学校她一直是乖乖女，来公司上班之后，大家都是拼命的工作，忙得陀螺一样，好像都没有时间谈恋爱。
<br/>
<br/>        再加上上班之后，圈子似乎缩小了，都是单位上的人，倒是有几个小男生给她送过花，可是她从来都没有感觉，一点都不会动心。
<br/>
<br/>        那天见了郝景文，心怦怦跳，之后每次见了郝景文心跳都会加速，元旦的时候坐在郝景文的身边智商都拉低了，要不然不会说那么多话。
<br/>
<br/>        都怪郝景文长得太帅，原来她陈思宇就是一个颜控，陈思宇鄙视了一下自己。
<br/>
<br/>        她将要和郝景文讨论的问题在心里过了一遍又一遍，怕自己心慌又漏掉了什么，毕竟郝景文是自己的顶头上司，虽然她不怕被炒鱿鱼，但是她还是很珍惜自己的这份工作的。
<br/>
<br/>        她提前半个小时打卡下班，到了郝景文指定地点，还好，今天比郝景文早到。
<br/>
<br/>        正摊开资料准备在看一遍，她爸爸的电话来了：
<br/>
<br/>        “思宇，周末回来不？”
<br/>
<br/>        “我在上班呢，等放假了回去好吗？”
<br/>
<br/>        “你这孩子，放出去心就野了，周末你回来，有人给你介绍了一个对象，回来看看合不合适？”
<br/>
<br/>        “爸，你能不能别天天催啊催的？好像你女儿嫁不出去一样，别人家是妈妈催，我们家是你催，我妈都不担心，你担心什么嘛？”
<br/>
<br/>        “这么说，你有男朋友了？那周末带回来我看看。”
<br/>
<br/>        “有个......毛线，算了，爸，我不给你说了，啊，来了一个大帅哥，特别帅，哎呀朝我走过来了，挂了挂了。”
<br/>
<br/>        陈思宇连再见都来不及给爸爸说，便挂了电话，郝景文从外面走进来。
<br/>
<br/>        陈思宇用手将头发别到脑后，心，还是心怦怦乱跳。
<br/>
<br/>        “思宇，你这么早？”郝景文走到陈思宇的身边，居高临下的看着陈思宇。
<br/>
<br/>        陈思宇感觉郝景文的言下之意就是：等不及见我了？
<br/>
<br/>        “怕路上堵车，还怕郝总炒了我的鱿鱼，提前半个小时下班。”
<br/>
<br/>        “咦，不怕陈子昂炒你的鱿鱼？倒是怕我炒你的鱿鱼？”郝景文笑眯眯的看着陈思宇。
<br/>
<br/>        眼前的人儿满脸绯红，粉嘟嘟的苹果肌上飞起两朵火烧云。
<br/>
<br/>        “陈总让我配合您以后的工作，郝总，我将最近的工作报告做出来了，您看看。”陈思宇平时那火爆的性子完全在郝景文的面前无用武之地，连“您”都给憋出来了。
<br/>
<br/>        郝景文感觉很好笑，但是一提到工作，他又立马认真起来。
<br/>
<br/>        似乎他真的是为工作而来，陈思宇心里有些忐忑，还有一丝丝失落，感情人家就是来谈工作的。
<br/>
<br/>        既然要谈工作，郝景文索性就和陈思宇坐一起了。
<br/>
<br/>        他挨着陈思宇，拿起陈思宇的工作报告看了看，工作报告做得很细致，一看就知道陈思宇工作很认真。
<br/>
<br/>        “财务管理出身的工作都这么细致吗？”陈思宇的工作报告一目了然，包括每一个项目，起始日和过程都很清楚。
<br/>
<br/>        郝景文看着在心里暗暗点赞！
<br/>
<br/>        “那是当然，一分一毫都不能有差错，包括标点符号。如果财务报表错一个点就是不得了的事情呢。”
<br/>
<br/>        “那你们的记忆力是不是也特别好？”
<br/>
<br/>        “我还可以吧，别人我就不知道了。”只要讨论起工作来，陈思宇就恢复了自信。
<br/>
<br/>        “那你能记得和我见了多少次面吗？”
<br/>
<br/>        “啊？为什么要记这个？和工作又没有关系。”
<br/>
<br/>        “怎么能和工作没有关系呢？难道你每次见我不是因为工作的原因而是另外其他的原因？”郝景文侧脸看着陈思宇，陈思宇清晰的绒毛可见，忽闪着的睫毛俏皮而可爱。
<br/>
<br/>        陈思宇连耳垂都能滴出血来，那大大的眼睛圆睁着，似乎马上就要变脸了，但是看着郝景文帅气的在眼前放大的俊脸，突然就泄气了。
<br/>
<br/>        “我见你当然是工作的原因，但是我没有必要都记得清清楚楚的啊。”
<br/>
<br/>        “原来还是这么没有良心的人，我可都是记得清清楚楚的，第一次是在我办公室，你扎着马尾踏着阳光进来。第二次是在我们的会议室，你带着一阵旋风进来，还有......”
<br/>
<br/>        “没有了，你要不要听我汇报工作？不听了我就回家，晚饭还没有吃。”陈思宇突然就炸毛了。
<br/>
<br/>        这算是什么事情？谈情说爱？谈情说爱还要加上工作？哼，搞暖昧？她陈思宇可是铁铮铮的女汉子一枚，别看你郝景文帅得一塌糊涂，搞暖昧的那一套我可不吃！
<br/>
<br/>        “嗯，饿了？菜马上就来了，耐心的等一会。”郝景文看着突然炸毛的陈思宇，竟然觉得无比可爱，柔声的哄着。
<br/>
<br/>        “不是，郝总，我觉得吧，工作就是工作，我不想占用我下班的时间来和你谈工作，我也是一个花季美少女，我还要谈男朋友，我还有闺蜜。”陈思宇放飞了自我。
<br/>
<br/>        “你有男朋友了？”郝景文抓住了他关注的重点。
<br/>
<br/>        “没有男朋友才要去谈男朋友啊，有男朋友我就和男朋友约会去了。”言下之意我有男朋友的话我还在这里和你谈工作？
<br/>
<br/>        “你可以将我当成你的男朋友，我们也可以谈一谈的。”郝景文心花怒放。
<br/>
<br/>        “你？我？”陈思宇指着郝景文又指了指自己。
<br/>
<br/>        “难道我不够帅？”郝景文盯着陈思宇吃惊的样子。
<br/>
<br/>        “你，你，你帅！”陈思宇竟然咽了一口唾沫，秀色可餐有没有？
<br/>
<br/>        此时，服务生来上菜了！
<br/>
<br/>        “你来得真是时候。”陈思宇看着服务生脱口而出，迟不来，早不来，偏偏这个时候来！
<br/>
<br/>        “没事，我们有的是时间，不怕被别人干扰。”郝景文细心的替陈思宇铺开前面的餐布，意有所指的说。
<br/>
<br/>        陈思宇有那么一愣神，郝景文已经将她爱吃的那个菜布到了她的餐具上：“这个是你爱吃的菜。”
<br/>
<br/>        “你怎么知道？”
<br/>
<br/>        “元旦的时候，看见你多吃了几口。”郝景文又给陈思宇布菜。
<br/>
<br/>        “我自己来。”
<br/>
<br/>        “男朋友不是要宠着自己的女朋友吗？”
<br/>
<br/>        “我们，我......”
<br/>
<br/>        “嗯，我以后会好好宠着你的，你想要什么我都会给你，包括我自己！”郝景文认真的看着陈思宇。
<br/>
<br/>        “就这样被表白了？”陈思宇心里这样想着，嘴里也这样说出来了，心口实在是太统一了。
<br/>
<br/>        “嗯，如果你还想要更正式的话，得等一段时间！”郝景文心满意足的吃了一口菜，今天的饭菜真可口！
