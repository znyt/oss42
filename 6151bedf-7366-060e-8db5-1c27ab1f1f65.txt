　　时间飞快，眨眼间便过去了一个月，而在地球上，已经有人通过卫星发现了九龙拉棺的存在。
<br/>　　同时，在这一天，冥冥中自有注定，主角叶梵与他的同学们来到泰山，只不过，此次却多了一个名叫夜邪的。
<br/>　　泰山，巍峨雄浑，气势磅礴，尊为五岳第一，号称天下第一山！
<br/>　　山莫大于之，史莫古与之！
<br/>　　泰山雄伟壮阔，具有厚重的历史沉淀，可追溯到上古三皇五帝时期，是冀近神灵之地。
<br/>　　上古时期，诸多圣皇与古帝无一例外，皆选择在此地封禅，令泰山笼罩重重迷雾，透发出无尽神秘气息。
<br/>　　一群同学们，于今日来到这泰山之上，登上这泰山。
<br/>　　“会当凌绝顶，一览众山小！”诗圣在此留下千古绝唱。
<br/>　　此时，夜邪身穿一声黑衣长袍，满头黑发飘散，模样俊美，双目冷淡无情，一直沉默不语。他今日随着叶梵登上泰山，也见证了泰山的浩大巍峨。以前的地球，他也去过泰山，但他所在世界的泰山并没有这个世界的泰山这般雄伟壮阔。
<br/>　　“咦！叶梵，这位帅哥是——”有女名为林佳，看到夜邪，向旁边的叶梵问道。
<br/>　　“这位是我朋友，夜邪！”叶梵一脸莫名笑意地看着林佳，向她介绍道。
<br/>　　“夜邪，呃？爷爷？！”女子也瞪大眼睛，满脸不可思议。
<br/>　　“不错，就是夜邪。”
<br/>　　一旁叶梵强忍着笑意，他当初得知这个名字之时，一样一脸不可思议。起这样的名字，你是要占多少便宜，拉多少仇恨啊？你父亲小时候一定是被欺负惨了，所以才会起这个名字，来占别人的便宜。
<br/>　　“呃，爷——帅哥，你真叫那个名字？”女子有点喊不出口。
<br/>　　夜邪抬起头，淡漠地打量了一下这个女子，双眼无情，好似坚冰，看得女子只打了个冷颤，他才缓缓点了点头，也不说话。
<br/>　　“呃？”
<br/>　　女子顿时就有种被无视的感觉，她以前走到哪里，哪里不是有人跟她套近乎，今日这人竟然无视我，难道是我魅力下降了，还是他不行？不过，出于礼貌，她还是没有说出来。
<br/>　　“不好意思，我这个朋友就是这个性格。”叶梵解释道，不过，他脸上哪有一点不好意思了？
<br/>　　“没事，那我先走了。”林佳说了一声，也就离开了。
<br/>　　之后，叶梵带着夜邪见过了他的众位朋友，遇到还算可以的，夜邪也会微微点一点头，稍微示意一下，若是遇到不顺眼的，就直接被他无视了。
<br/>　　一直到傍晚，夕阳西垂。突然，天际间出现几个圆点，而后逐渐变大，竟传来阵阵风雷之响。
<br/>　　九条庞然大物从天而降，竟是九条黑色的长河垂落下来，在这一刻，泰山上的所有人都被惊得表情凝固，愕然相望。
<br/>　　那竟然是九具庞大的龙尸，拉着一口青铜古棺，向泰山峰巅压落下来。
<br/>　　龙，传说中的存在，与神并列，凌驾于自然规律之上的存在。
<br/>　　今日我们竟然能够看到传说中的龙？是这个世界疯了，还是我们眼花了？
<br/>　　游客们经过短暂的震惊后，开始沸腾了，所有人都惊慌奔跑，向着四面八方冲去，躲避那临近的庞大龙尸。
<br/>　　恐惧的惊叫，无助的哭喊，众人纷纷逃避。生命在此刻，是如此的脆弱。
<br/>　　轰！
<br/>　　九条庞然大物沉落下来，将玉皇顶崩裂出一道道裂缝，土石飞溅，尘沙弥漫。
<br/>　　之后的一切，就如预料之中那般，众人经过短暂的震惊、慌乱、恐惧，渐渐恢复到平静、好奇。
<br/>　　好奇心害死猫，总有人会因为各种用原因留了下来。在好奇心的驱使下，众人发现了青铜棺下的五色祭坛，之后在一个女同学不小心摔下去后，触动了机关。
<br/>　　轰！
<br/>　　八卦流转，太极显现，阴阳相合，星门大显。
<br/>　　青铜古棺被打开，在众人的无助哭喊声中，此刻，众人想走也来不及了。之后，众人都无法控制的被吸了进去，古棺关闭。
<br/>　　轰！
<br/>　　九条庞大的龙尸震动，龙尸腾空而起，再次开启了前往星空之路，消散于虚空之中。同学们都被困在了青铜古棺之内。
<br/>　　古棺之内，一片寂静漆黑，弥漫着一股冰森森的寒意，有女同学近乎崩溃，内心充满无助与惶恐，却不敢大声哭喊。
<br/>　　“怎么办，谁能就我们？”
<br/>　　“我们现在真的在铜棺内部吗？”
<br/>　　“为什么会这样，有什么办法可以出去？”
<br/>　　不断有人用手机拨号，但是却根本打不通，无法与外界联系，这更让人恐惧。
<br/>　　“不逃慌，我们虽然暂时无法出去，但泰山上发生如此事情，相信救援人员一定会第一时间赶到。”有名为周易的男子，显得镇定，安慰大家。
<br/>　　“周易说得对，大家不要慌，我们一定要镇定，相信大家一定会没事的。”有叶梵的同学王子文出来稳定大家。
<br/>　　“不错，大家不用慌，事情已经发生了，再恐惧也无用。”女子林佳此刻表现得极为从容，没有惊慌。
<br/>　　叶梵没有出声，静静坐在那里，头脑之中无数讯息转过。
<br/>　　夜邪坐在他身旁，一脸淡漠无情，眼中不见丝毫惊慌，他似乎早知如此。
<br/>　　不少人脸色苍白，瑟瑟发抖，也有人面色镇定，或呜咽抽泣。
<br/>　　在这场突如其来的变故之下，每一个人的内心，都显现了出来。
<br/>　　他们的心性暴露无遗，哪一个优秀，哪一个懦弱，哪一个中庸，都落在夜邪眼中。
<br/>　　“嗡……”
<br/>　　正在此时，一阵声响，顿时，铜棺之内有混沌色光芒绽放，紧接着，无尽光芒，驱散了黑暗，棺内空间变得灰蒙蒙一片，虽然还是无法看得明白，但至少比起之前的无尽黑暗要好得多。
<br/>　　“混沌神殿”四个大字浮现于虚空之中，带着无尽道光，给人无尽威严。
<br/>　　“啊？我看得见了。”有人激动的叫喊道。
<br/>　　“欢迎来到混沌神殿！”
<br/>　　就在此时，一道虚无缥缈的神秘声音突然在棺内响起，于无声处听惊雷。
<br/>　　“谁？谁在说话？”听到这声音，有人惊慌道，纷纷向四周看去，想要找到它。
<br/>　　“谁在装神弄鬼？”
<br/>　　“谁？到底是谁？”
<br/>　　……
<br/>　　“新人们，欢迎来到混沌神殿。首先，恭喜你们，能够有幸被神殿选中。从今以后，你们便是混沌神殿的弟子了。作为第一批被选中的幸运儿，你们有权问十个问题。”
<br/>　　就在此时，又是一道神秘的声音与虚空之中传来，声音淡漠无情。
<br/>　　“谁？你到底谁？”有人惊恐叫道。
<br/>　　“第一个问题，吾乃混沌主神。”那神秘声音再次响起。
<br/>　　“此处是什么地方？”有人大声叫道。
<br/>　　“此乃混沌神殿。”
<br/>　　“混沌神殿？混沌神殿在那里？”有人下意识问道。
<br/>　　“遮天世界，无尽虚空，无穷星域，无数星球，吾，无处不在。”
<br/>　　“嘶！”众人倒吸一口气，只感觉好高大上啊。
<br/>　　“混沌神殿是干什么的？”有周易比较冷静，问出这样一个问题。
<br/>　　……
