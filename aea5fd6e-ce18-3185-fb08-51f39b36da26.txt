        “行吧行吧！”明司寒若有所思地瞥了连星儿一眼，反正都一样，于是改口问道，“那我和洛羽落水了，你先救谁？”
<br/>
<br/>        “我谁也不救。”连星儿笑嘻嘻地说道。
<br/>
<br/>        明司寒听着这话心里更加不爽了，忍不住碎碎念了一句：“你的心好狠！”
<br/>
<br/>        “不是我的心狠不狠的问题，而是我不会游泳啊！”连星儿一本正经地说。
<br/>
<br/>        明司寒顿时难以置信地看着连星儿：“你会防身术，竟然不会游泳？”
<br/>
<br/>        “这有什么好稀奇的啊！我是个早产儿，医生说我天生肺就不好。就算学会了游泳，也折腾不了几下，反而会压迫肺部。所以，我爹地妈咪就没让我学了。”连星儿解释道。
<br/>
<br/>        明司寒听着，不得不替自己圆话：“还好我会！到时候我跟洛羽真掉水里了，我可以自救的。”
<br/>
<br/>        “那你还问我这种问题。”连星儿不禁给了明司寒一个白眼。
<br/>
<br/>        明司寒却慵懒地回了一句：“无聊，随便问问。”
<br/>
<br/>        就在两人话题不知道该如何继续下去的时候，门外响起了一个好听的甜甜的呼唤声：“司寒哥哥，真的是你呀！”
<br/>
<br/>        连星儿好奇地抬起头来，循声望去，只见那声音的女主人从门外先是探头，后是像只小兔子一样蹦了进来。
<br/>
<br/>        女孩年龄看起来与她相仿，十七八岁的样子，扎着斜马尾，发梢卷翘而显得有几分活泼。
<br/>
<br/>        她身上穿着的白色T恤和淡蓝色牛仔裙，倒是跟明司寒今天的这身同样是白色T恤和淡蓝色牛仔长裤的着装，像极其了情侣装。
<br/>
<br/>        “你是谁啊？怎么跟我的司寒哥哥坐得这么近！”女孩走过来后，二话不说，便抓住连星儿的手腕，强行将连星儿从明司寒身旁给拉开。
<br/>
<br/>        连星儿不禁打了个趔趄，差点摔倒，明司寒连忙起身，眼疾手快地将她给搂了回来。
<br/>
<br/>        “你怎么在这？”看到女孩对连星儿一点礼貌也没有，明司寒紧紧地搂着连星儿的腰肢，一脸不爽地问道。
<br/>
<br/>        女孩撅起嘴，瞥了一眼明司寒搭在连星儿腰上的手，大步上前，将他们俩给挤开后，站在中间，完全无视掉连星儿，抬头仰望着明司寒，微笑道：“我爷爷住院了呀！我来看我爷爷的！这不，我刚看完我爷爷，准备回去了，在走廊外听到你的声音，我就好奇进来了。”
<br/>
<br/>        “那你看完你爷爷了，还不赶紧回家。”明司寒催促道。
<br/>
<br/>        女孩趁机挽住明司寒的臂弯，甜甜地笑了笑：“司寒哥哥，你送我回家，好不好？”
<br/>
<br/>        “我……”
<br/>
<br/>        “我就不打扰你们了哈！先回去了！”
<br/>
<br/>        连星儿和明司寒几乎是异口同声，但是明司寒的话还未说完，连星儿便将自己的话一口气给说完了。
<br/>
<br/>        明司寒连忙甩开女孩的手，一脸微笑着看着连星儿，温和地说道：“我送你回酒店。”
<br/>
<br/>        “不用了！”连星儿当然是识趣，这女孩对她充满了敌意，她还是离他们远点比较好。
<br/>
<br/>        “我觉得我还是……”
<br/>
<br/>        “司寒哥哥……”女孩再次挽住明司寒的臂弯，扭着身子，摇晃起明司寒的胳膊，还打断了他的话。
<br/>
<br/>        连星儿挥了挥手，二话不说地转身离去。
<br/>
<br/>        当明司寒在连星儿的眼里看不到一丝醋意的时候，心里莫名其妙地凉了一大截。
<br/>
<br/>        连星儿离开后，女孩又嗲声嗲气地问道：“司寒哥哥，她是谁呀？”
<br/>
<br/>        “我的未婚妻。”明司寒再次将女孩的手给甩开。
<br/>
<br/>        女孩不依不饶地黏上去，愤愤不满地质问：“你的未婚妻可是我诶！什么时候变成她呢？”
<br/>
<br/>        “贺思思！”明司寒不耐烦地怒吼了一声，“麻烦你离我远点！”
<br/>
<br/>        “我……”贺思思顿时一脸无辜地噘着嘴。
<br/>
<br/>        明司寒不再理会贺思思，独自疾步走出了病房。
<br/>
<br/>        另一边，连星儿离开医院后，站在出租车等候区的路边时，才发现自己身上什么也没带。
<br/>
<br/>        就在这个时候，身后忽然有人喊了她一声：“连星儿。”
<br/>
<br/>        连星儿本能地回头，循声望去时，只见一个熟悉的身影向她这边走了过来。
<br/>
<br/>        第一眼，她没认出是谁，待这人走近后，她才猛然间反应过来。
<br/>
<br/>        “班长？！”连星儿疑惑地唤了声。
<br/>
<br/>        男生微微一笑：“嗯，是我！怎么，我摘了眼镜，你就不认识我了吗？”
<br/>
<br/>        连星儿不禁笑吟吟地夸赞道：“对啊！你摘了眼镜后变得更帅了，所以我没认出来。”
<br/>
<br/>        这男生名叫严文斌，是她读高三时的同班同学，也是他们那个班的班长。
