　　其余人躲过之后，立刻去攻击她的身后，但已经不敢再使出全力，只围着她，一边尝试进攻，寻找破绽。
<br/>　　星月奴冷哼一声，面对任何攻击，都是随手一抓，撕开虚空裂缝，试图将人拉进去。
<br/>　　一点近身的机会也没有。
<br/>　　但大伙不敢放松攻势，就这么围着她，让她也腾不出手去干别的。
<br/>　　慈心师太为首的那一帮法师，都退到了山门外面，目瞪口呆地观望着这场生死战争。
<br/>　　他们一个都没动手，他们不知道自己该帮谁。
<br/>　　他们刚归顺了轩辕山，自然不可能再反水
<br/>　　“清长风，你的确是有些手段的！”
<br/>　　隔壁战场，叶少阳还被困在七星阵中，被动之极，却突然开口说道。
<br/>　　清长风正在跟四宝斗法，阻止他去营救叶少阳，突然听到这句话，心头一紧，果然，下一秒钟，叶少阳突然大喝一声，口中念咒，左手翻转，一道古铜色的光斑从他手中弹了起来。
<br/>　　雕母大钱！他身后最后一枚雕母大钱。
<br/>　　一直被他夹在手指中，此刻吸收了充足的罡气，弹起来的瞬间，灵气爆发，将七颗铁离子中的一个打落。
<br/>　　七颗铁离子，凑在一起便是北斗七星，运行之中攻守兼备，自成体系，被他这么敲掉一个角，阵法威力立刻减弱的数倍，叶少阳右手一翻，七星龙泉剑抓在手中。
<br/>　　“七星长鸣，龙泉杀敌！诛邪！”
<br/>　　一声龙吟，紫气冲天而起，瞬间击碎了七星阵，也在这时，原本一味防守的四宝陡然加强了攻势，观想出的韦驮天手持禅杖，朝清长风脑袋上砸过去，距离太近，躲避不开，清长风只能格挡。
<br/>　　以他的实力，格挡这样一次进攻不成问题，但叶少阳早就等着这一瞬间，长剑一挺，刺向他的喉咙，
<br/>　　他已经躲不过这一剑了。
<br/>　　咔嚓！
<br/>　　清长风右边空空的袖子里，突然伸出来一只手，在千钧一发之际，抓住了七星龙泉剑的剑锋。黑气从指缝中不断溢出，与宝剑的灵力互相消耗起来。
<br/>　　“我知你阴险狡诈，早就防着你这一招，扮猪吃老虎！你是没想到，我还有一只手吧！”
<br/>　　从一开始叶少阳被他的七星阵困住，他就看穿了叶少阳在演戏，以退为进，算计自己，他当场没揭穿，将计就计，故意配合，等着叶少阳交底。
<br/>　　他本就不是人类，断臂短世间内不能再生——幻术变出来只是好看，不能过气，斗法的时候完全没用。之前在星月奴说服大家的时候，他就拆了肉身上一根灵骨，重塑了手臂，只是藏在宽大的袖子里，露出来，为的就是麻痹敌人。
<br/>　　叶少阳什么都算到，就是没算到他还有一只手。
<br/>　　在抓住七星龙泉剑的一瞬间，清长风猛然法力，趁着叶少阳呆滞的那一秒钟，抢先逼退了宝剑上叶少阳贯通的灵力，并且顺着手臂灌入他体内，生成出寂灭二气，在他体内横冲直撞起来。
<br/>　　叶少阳身体剧烈地抖了一下。
<br/>　　四宝那边一怔，抢攻而上，清长风用一只手去结印抵挡，不在话下，冲叶少阳笑道：“寂灭二气，是我用了几百年时间修成的，入你之身，便如跗骨之蛆，一生都无法驱散！你还不磕头求我破解法门！”
<br/>　　当下大喝一声，口中念咒不断，寂灭二气在叶少阳体内更加疯狂地窜起来。
<br/>　　这种感觉有点像是醉酒时，能感觉到酒精在血管里作用，一涨一涨的那种疼。
<br/>　　叶少阳赶紧封锁住了心脉，盯着清长风，突然吐出一句话：“你还有没有第三只手？”
<br/>　　清长风一惊，突然觉得脑后生风，猛然回头，一把霜一般的剑朝自己飞来。
<br/>　　没什么法术招式，也没太强的手段，就是一个字：快！
<br/>　　快到在不远处被围攻着的星月奴发现的瞬间，挥手甩出一道金光来救，都来不及了。
<br/>　　一剑贯穿了清长风的左眼，爆出一股子绿血。
<br/>　　吴嘉伟回头看了一眼，又朝远处飞奔，去帮小马他们去了。
<br/>　　这才是叶少阳真正的战术：所有人只守不攻，全都拖住对手，然后叶少阳这边，先集中力量干掉几人中实力最弱的清长风。
<br/>　　虽然很想好好跟他打一架，验证下自己而今的实力，但这次是生死之战，只能追求速战速决。
<br/>　　于是叶少阳故技重施，先故意中招，把自己困住，这种情况下，清长风肯定会想方设法弄死自己，一旦着重攻击，必然忽略防守，然后叶少阳再出其不意地堪破七星阵，配合四宝的进攻，急促反攻。
<br/>　　一般人到这，必中这一剑。
<br/>　　但清长风虽然自大，却跟叶少阳打过几次交道，知道这货喜欢扮猪吃老虎，在他被困之初，已然做好了防备，叶少阳也没料到他还有第三只手，不过没关系，这本来就是一个计中计。
<br/>　　真正的后手，不是方才自己那一击，而是神出鬼没的吴嘉伟。
<br/>　　四宝当肉盾，卡位置，一边骚扰，分散他注意力，然后配合攻击，进一步吸引清长风的注意，吴嘉伟则早就在远处游荡（这一点十分重要，在之前二打一的时候，一定要让清长风觉得身边不会有人来增援，否则他哪怕下意识也会提防），等到最佳时机，一剑封喉。
<br/>　　不对，一剑刺瞎了眼睛。
<br/>　　星月奴释放的那一道神光来到，在他们头顶一分为十，如剑一般刺下来。
<br/>　　叶少阳也早就看到，当下从腰带里摸出天风雷火旗，念咒朝天上扔过去，挡住了神光，左手抬起，将一张焚天灭地符贴在他额上。
<br/>　　“太上三清，四方大帝，神威通天，焚天灭地，急急如律令！”
<br/>　　轰的一声，火焰腾起，将清长风的脑袋烧成了一个大火球。
<br/>　　“去死！”
<br/>　　叶少阳右手反过来就是一剑，将清长风脑袋斩了下来。
<br/>　　“师兄！”
<br/>　　星月奴一声大叫，一时间气息大盛，将周围几人都撞开，冲了过来。
<br/>　　四宝赶紧祭出罗汉金身，上前挡住。
<br/>　　叶少阳遁走，一口气退到了道观外面。
<br/>　　“走开！”
