　　叶尘冷冷一笑，
<br/>　　“他不过就是我从外面找来带路的，是死是活与我何干？堂堂药王宗的宗主，说这话未免也太幼稚了些！”
<br/>　　纵然叶尘对祝小白很是喜爱，但也不可能为了他把自己搭进去。
<br/>　　更何况，他越是紧张祝小白的性命，云天青就越会以此来要挟他。
<br/>　　反之，他假装不关心祝小白的死活，云天青看到他没有利用价值，反而会救了他。
<br/>　　果不其然，
<br/>　　云天青皱着眉头，思索了片刻后，似乎也相信了叶尘从话，于是直接将祝小白，随手丢给后方的药王宗弟子，冷然道：
<br/>　　“将这小孽种关押起来，放消息出去，看看能不能把火神宗的其他余孽给引出来！”
<br/>　　“是！”
<br/>　　那名药王宗弟子，当即应了一声，带着祝小白离开了。
<br/>　　叶尘见状，顿时暗暗松了一口长气。
<br/>　　看样子，祝小白暂时应该不会有生命危险，而他少了这么一个包袱，带着苏蔓一人，先暂时逃离药王宗，应该也就没有那么困难了。
<br/>　　‘小白，等我先安顿好蔓姐，再回来救你！’
<br/>　　心中打定主意之后，叶尘抱着苏蔓，脚下猛然一顿，瞬间拔升到数百米的高空！
<br/>　　“不好！他想逃跑！快拦住他！”
<br/>　　云天青大喝一声，五人当即纷纷操控着妖尸，想要拦住叶尘的去路。
<br/>　　“十方无敌！”
<br/>　　唰唰唰！
<br/>　　转眼之间，
<br/>　　成千上百道紫色剑芒，在空中划过。
<br/>　　砰砰砰！
<br/>　　那五具妖尸，顿时便被叶尘这一招，打的阵型大乱。
<br/>　　“走！”
<br/>　　叶尘瞅准机会，当即便要带着苏蔓逃离药王谷。
<br/>　　然而就在这时，忽然一道犹如雷音般的声音陡然响起，
<br/>　　“小子！你当我药王宗，是你想来就来，想走就走的吗？”
<br/>　　声音落下，天空之上，不知道何时，出现了一名须发尽白，一身白衣，飘然若仙的老者。
<br/>　　只见那老者，抬手猛然一拍
<br/>　　轰隆！
<br/>　　一道长达数百米的青色巨掌，便朝下方的叶尘狠狠拍落下来，一下子便将叶尘的去路，全部彻底封死！
<br/>　　“是太上长老！”
<br/>　　“太上长老出手，这下看这小子还不死？”
<br/>　　“太上长老的修为，似乎又有精进了，莫非已经成就化神天君了？”
<br/>　　“太强大了！”
<br/>　　……
<br/>　　伴随着药王宗众人的议论声，那道巨掌也狠狠落下。
<br/>　　叶尘只感觉头顶上方，仿佛出现了一座巨山！
<br/>　　“没想到药王宗，竟然还有如此高手！难道此人就是药王宗的太上长老云千秋？”
<br/>　　看到这一幕，叶尘也不由得脸色大变，
<br/>　　根据初步判断，对方恐怕至少也是元婴境巅峰之境，甚至应该已经触摸到了化神境的门槛，只不过还没有完全突破。
<br/>　　但饶是如此，也让叶尘感觉压力倍增，
<br/>　　‘药王宗擅长用毒，我的近战优势，根本发挥不出来，他们只需要将我拖住，耗也能把我耗死！’
<br/>　　‘而且蔓姐体内的毒气，似乎也越来越强了，同样耽搁不得，必须尽快为她驱毒……’
<br/>　　叶尘在思绪飞转的同时，抬起手中的紫琼剑，朝天空上的那道巨掌迎了上去。
<br/>　　轰隆！
<br/>　　两股强悍的力量，狠狠相撞在一处，连周围空间都跟着一阵乱晃。
<br/>　　一些离得近的药王宗弟子，顿时被站立不稳，连连倒退。
<br/>　　就连云天青五人，也纷纷退开一旁，不过两眼却死死盯着天空上方的战况。
<br/>　　砰！
<br/>　　两股力量相撞之下，叶尘似乎不敌，竟然如同流星般，朝下方急速坠落而去。
<br/>　　周围的药王宗弟子，看到这一幕，顿时纷纷欢呼起来，
<br/>　　“太上长老威武！”
<br/>　　“这小子这下死定了！”
<br/>　　“胆敢在我药王宗撒野，这就是下场！”
<br/>　　……
<br/>　　众人欢呼了一阵，可是很快看到接下来的一幕，顿时个个目瞪口呆。
<br/>　　只见，叶尘被云千秋一掌拍落之后，竟然一个掉头，直接冲进了下方的那个洞穴！
<br/>　　“不好！这小子竟然闯入了我们药王宗的禁地！”
<br/>　　“这万毒窟禁地，只有历代宗主才有资格进入，这下该如何是好？”
<br/>　　“自然是要把他从里面揪出来！”
<br/>　　……
<br/>　　在众人的议论声中，天空之上的白衣老者，缓缓降落下来。
<br/>　　云天青和四大长老，当即上前行礼。
<br/>　　其中一名长老道：
<br/>　　“太上长老，这小子竟然敢闯入我们药王宗的禁地，要不要进去把他抓出来？”
<br/>　　太上长老云千秋，顿时眉头一皱，
<br/>　　“胡闹！你又不是不知道，万毒窟禁地，对于我们药王宗来说，意味着什么！”
<br/>　　“倘若我们就这么冲进去，这家伙狗急跳墙，把里面给毁了，这个责任谁来承担？”
<br/>　　那名长老顿时脸色微微一边，但又有不甘心，道：
<br/>　　“可是，难道我们就这么任由他不管了吗？”
<br/>　　云千秋冷冷一笑，
<br/>　　“这禁地之中，遍地都是毒虫、毒草、毒兽，就算是老夫，也最多只能在里面待上一天一夜，我就不信他能撑得住？”
<br/>　　“更何况这里面，可还有一头千年毒兽，便是化神天君，也不能等闲视之，我就不信这小子还能活着出来！”
<br/>　　云天青五人，听到千年毒兽之时，眼眸中纷纷露出一抹畏惧之色，继而又面露喜色。
<br/>　　“太上长老言之有理！”
<br/>　　……
<br/>　　却说叶尘带着苏蔓，冲入那洞穴之中后，发现后面没有人追来，也顿时松了一口气。
<br/>　　不过很快，他便发现，这里的情况，似乎并没有他想的那么好。
<br/>　　这洞穴之中的毒气，是外界的十倍都不止，而且到处都是各种毒虫、毒草，叶尘必须运起护体真元，将他和苏蔓包裹起来，方才能抵挡这些毒气的入侵。
<br/>　　“难怪药王宗的那些人没有追进来，他们肯定算准了，我不可能在这里面待的太久，所以干脆守株待兔，等我体内的真元，被毒气消耗的差不多，自然也就不得不出去了……”
<br/>　　想明白了其中缘由之后，叶尘也不由得眉头紧皱，
<br/>　　“如今的当务之急，是要先帮蔓姐驱毒，可是这里毒气如此浓郁，实在不适合运功驱毒！”
<br/>　　正不知该如何是好，脑海之中，忽然想起了吞天鼎的声音，
<br/>　　“主人，你们可以先进入我的体内空间，只要你分一部分功力给我，我就能帮你抵挡周围的毒气！”
<br/>　　叶尘顿时一拍脑袋，
<br/>　　“是我急糊涂了，竟然把这个方法给忘了！”
