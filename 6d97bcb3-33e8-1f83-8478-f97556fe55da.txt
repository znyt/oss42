　　叶少阳当场无语，苦笑道：“你都说我法力低微了，你觉得我有办法从一个天师身边偷走或抢走天师牌吗？”
<br/>　　功曹道：“可你的法力，不可能是天师。”
<br/>　　叶少阳叹了口气，这才说道：“我不是天师，我是灵仙。”
<br/>　　“灵仙？”
<br/>　　两个功曹对视了一眼，哈哈大笑起来。
<br/>　　“实话告诉你，小子，人间灵仙，在我天子殿都是挂了号了，总共就那么寥寥两三个，你……”功曹说到这里，笑的自己都说不下去了，之前他还有些疑惑叶少阳这块天师牌的来历，听“他”说出灵仙两个字之后，立刻就释然了，认定这块天师牌一定不是叶少阳的东西，而且他绝对不是道门高级弟子，否则不会敢吹牛说自己是灵仙。
<br/>　　叶少阳还想开口辩解，突然脑后挨了一下，被打得几乎魂飞魄散。
<br/>　　“老实交代，你到底是什么人，从哪里来，这块天师牌是哪里弄来的！”
<br/>　　打他的是一直站在他身后按着他肩膀的那个功曹。
<br/>　　“我说大哥，用得着动手么，我又没反抗。”
<br/>　　叶少阳缓了半天，转过头去，看到那个功曹，突然愣了一下。
<br/>　　“谁是你大哥！”那功曹举手还要打，叶少阳突然叫了一声：“老王！”
<br/>　　功曹一下怔住，抬起的手停留在半空中，皱眉望着叶少阳，道：“你认识我？”
<br/>　　叶少阳还真认识他。其实说起来，因为橙子和萧逸云的缘故，天子殿没人不认识他，但是真的跟他打过交道的却不多，对面那个叶少阳只是有点面熟，不知道叫什么，这个老王叶少阳是认识的，之前几次来找萧逸云和橙子，跟他都有过接触。
<br/>　　之前每一次来，这老王对他都毕恭毕敬，叶少阳也是没想到他会有在自己面前耀武扬威的一天，还出手打了自己。
<br/>　　不过，他也知道，自己面前这个老王，并不是自己所在世界的那一个，现在是一百年前，这时候的老王不可认识自己。
<br/>　　叶少阳冲他点点头，说道：“认识，也算是熟人了。”
<br/>　　老王盯着他看了好久，说道：“你别套近乎，我好像不认识你。”
<br/>　　“你不认识我，是正常的，我不光认识你，我还认识天子殿的很多人，你们萧逸云押司是我哥们。”
<br/>　　老王和对面那功曹都怔住了，相互看了看，老王喝道：“莫要胡说，萧郎君怎么可能认得你！”
<br/>　　叶少阳道：“我是不是胡说，你把萧郎君叫出来见见不就知道了，我这次来，还真就是来找他的。”
<br/>　　老王一听，有些迟疑地对另外那个功曹看去，那功曹说道：“老王你在天子殿当差就百年了，有人认得你并不意外，至于萧郎君更不用说，我天子殿第一大红人，外面谁不认得他。他这般套近乎，必然更证明了是有备而来，先擒住他，回去慢慢问话便是，莫耽误正事。”
<br/>　　当下安排几个鬼差，继续将旁边那些吃瓜看热闹的鬼魂往天子殿里引。
<br/>　　叶少阳不服抗辩，老王也不听，用勾魂索捆住了叶少阳的脖子，就要往里面拽，叶少阳挣扎无效，眼看着要被拖走，突然从里面传来一个严厉的嗓音：“吵什么，出什么事了！”
<br/>　　一个人影翩翩走了出来，是个玉面公子，手里拿着把玉如意，劈头对那功曹说道：“怎么回事，里面半天没有生魂进去，你们却在这里纠缠什么！”
<br/>　　“小白脸！！”
<br/>　　叶少阳激动地叫了起来，面前不是别人，正是他在天子殿的好兄弟萧逸云！
<br/>　　这是叶少阳在这个世界第一个真正见到的熟人，甚至是亲人，他怎么能不激动。
<br/>　　萧逸云怔了一下，一张白如温玉的脸立刻就黑了，沉声喝道：“你叫我什么！”
<br/>　　额……叶少阳这才意识到自己刚才一时激动，叫错了，凝视着萧逸云的脸，喃喃道：“萧郎君，我是叶少阳。”
<br/>　　“叶少阳……”萧逸云咀嚼着三个字，脸上陷入沉思的表情，随即喃喃道：“我怎么好像在哪里听过这名字似的。”
<br/>　　本来不抱什么希望的叶少阳，听他这么说，顿时激动起来，然而，萧逸云之后的一句话却是让他失望了，萧逸云盯着他凝视片刻，说道：“我好像并不认识你。”
<br/>　　叶少阳深吸了一口气，苦笑起来，说道：“现在的你，当然不认识我，但一百年后，你是我兄弟，还是……”
<br/>　　他本来想说“妹夫”两个字，猛然间警觉地想到，这件事有点泄露天机的嫌疑，毕竟透露的事情太过具体，而且实际上现在并未发生。其次如果让他知道太多，对自己百年之后的命运有所察觉，也未必是一件好事，这才忍住了没往下说。
<br/>　　“一百年后？”萧逸云看着他，冷笑道：“你知道一百年后的事？”
<br/>　　叶少阳耸了耸肩道：“稍微知道一点吧，反正你一百年后是什么样子，我是知道的。”
<br/>　　萧逸云笑了笑，明显不相信，双手背在身后，走了过来，让老王和那功曹先去忙正事。老王解开了叶少阳身上的勾魂索，跟那功曹到一边去了。
<br/>　　萧逸云走到叶少阳面前，脸上带着轻蔑的笑意，说道：“那你倒是说说看呢，一百年后的我，是什么样子？”
<br/>　　叶少阳道：“是我我就不问，自己去体验多好，提前知道人生会怎么样，还有什么乐趣？”
<br/>　　萧逸云噗嗤一笑，“你理由还挺多，那你想要跟我说什么。”
<br/>　　老王正在检验进入天子殿的生魂，听到他们的对话，回过头来说道：“他说他是来找你的，他说他是灵仙。哦对了，他还有个天师牌，在老陈手上。”
<br/>　　之前那个跟叶少阳对话的功曹就是老陈，听见老王的话，把叶少阳的天师牌递给了萧逸云。
<br/>　　萧逸云检察了一番，结果当然跟老陈的一样，确定这东西是真的，心中略感诧异，也问叶少阳这东西是哪来的。
<br/>　　叶少阳实在不想在这个问题上纠缠下去，冲萧逸云说道：“我不跟你扯这些没有的，未来的事……我不想说，就算说了也没什么能证明的，这样吧，我跟你说点过去的事，我知道你不少秘密。”
