        这样的痛，带着淡淡的苦，恍若从胃部蔓延到诺筱颖的心口。
<br/>
<br/>        难道是晕车了吗？
<br/>
<br/>        诺筱颖并未多想，眼见着快要到达目的地了，她索性提前了一站下车，或许，走了一段路程，人会舒服点。
<br/>
<br/>        为了能设计出最好的办公室装修图出来，这两天，诺筱颖除了在网上寻找一些资料以外，还利用了中午休息的时间跑去了市图书馆，翻阅了一些国外室内设计大师的作品。
<br/>
<br/>        只有自己不断努力地学习和创新，才会慢慢地让自己在事业上成长起来。
<br/>
<br/>        至于那天在公交车上的移动电视里看到的什么绑劫爆炸案的事情，她因为工作上太忙了，就再也没去关注过了。
<br/>
<br/>        诺筱颖知道自己的学历低，公司里的同事看不上她，所以，她更想凭自己的本事证明自己的能力并不比他们差。
<br/>
<br/>        下班后，诺筱颖回到家中给哥哥诺司峻和嫂嫂唐倩兮做了晚餐后，自己只是夹带了一个饭团，便跑去公司里继续加班。
<br/>
<br/>        结合自己查阅的资料，然后再对照那位新任总裁的条条苛刻的要求，诺筱颖花了近半个月的时间，才将初稿设计了出来。
<br/>
<br/>        可是，令诺筱颖感到发愁的是，因为她进入了创作瓶颈期，她已经发现不了自己设计的缺点在哪儿了。
<br/>
<br/>        对于设计师来说，进入到这种状态中，确实很糟糕！
<br/>
<br/>        诺筱颖正为这事犯愁的时候，脑海里忽然想起了一个人。
<br/>
<br/>        那就是——傅昼景！
<br/>
<br/>        他虽然是建筑设计师，但一定也涉及过室内设计方便的知识，不如去请教他，或许能给自己指点一二。
<br/>
<br/>        诺筱颖想到这里，就在这天下班后，回家做好了晚饭，给哥哥嫂嫂留了纸条，立即抱着自己的设计图去了顶楼傅昼景的家。
<br/>
<br/>        她在傅昼景的家门口按了半晌的门铃，也未见有人给她开门，只好抱着手里的设计图，蹲在门边等候。
<br/>
<br/>        楼道里的吸顶灯亮了后又自动灭掉了，诺筱颖轻轻地咳了声，吸顶灯又亮了。
<br/>
<br/>        这样反反复复不知道过了多久，诺筱颖在门外一边徘徊，一边继续思考自己设计图里的内容。
<br/>
<br/>        直到电梯那边发出“叮”地一声，诺筱颖满脸欣喜地迎了过去。
<br/>
<br/>        电梯门刚一开，她的出现，将正从电梯里走出来的傅昼景给吓了一跳。
<br/>
<br/>        “是你这丫头啊！”傅昼景笑着说。
<br/>
<br/>        诺筱颖微微一笑，两只眼睛，就像夜空里的白月牙一样漂亮。
<br/>
<br/>        她温柔地问候道：“晚上好！”
<br/>
<br/>        “晚上好！”傅昼景走出了电梯，站在了诺筱颖的跟前，俯视着这个身高只到他胸口处的丫头，见她怀里抱着图纸，便关心地问道，“筱颖，你找我有事吗？”
<br/>
<br/>        诺筱颖如小鸡啄米般猛点头。
<br/>
<br/>        傅昼景一边从裤袋里掏钥匙，一边温和地对诺筱颖说道：“那进屋谈。”
<br/>
<br/>        “好！”诺筱颖有些迫不及待。
<br/>
<br/>        她跟在傅昼景身后，还未进屋，就开始将自己的设计图纸。
<br/>
<br/>        傅昼景非常耐心地听她讲解。
<br/>
<br/>        好学有上进的女孩，他没理由拒绝。
<br/>
<br/>        进屋后，傅昼景带诺筱颖进了书房，诺筱颖便将自己的设计图摊在了书桌上，让他看看她的设计思路和理念。
<br/>
<br/>        傅昼景看了一下图标，才发现，诺筱颖正在设计的是他大哥傅夜沉的办公室。
<br/>
<br/>        爷爷有意退休，让大哥要接手傅氏集团的事情，他早有耳闻。
<br/>
<br/>        傅昼景又看了看大哥傅夜沉对自己的办公室提出的那整整十页纸的要求，有些忍俊不禁。
<br/>
<br/>        他一看，就知道他大哥是故意的。
<br/>
<br/>        而且，这样的装修要求，有一半是他爷爷的意思。
<br/>
<br/>        “我觉得，落地窗，是用来开阔视野的，没有必要在落地窗旁设计品茶区。而且，落地窗外是天台。倒是可以在天台上，设计一处观赏区。到了冬日的午后，还可以在那儿晒晒太阳，倘若晚上要加班，加班加累了，还可以在这儿欣赏夜空。”傅昼景微笑着说道。
<br/>
<br/>        诺筱颖顿时觉得他的这个主意不错，这有钱人嘛！还真的会享受生活！
<br/>
<br/>        “还有，就是书架的设计，这个地方不太合理。你可以在书架的这边，设计一个自动暗梯，要用的时候抽出来，就可以踩上去拿上面的书籍，不要用的时候，就这样按一下，自动收好。刚好符合业主要求的便捷，又不碍事。”傅昼景接着说道。
<br/>
<br/>        果然，贫穷限制了她的想象力。
<br/>
<br/>        诺筱颖都没想到，可以在书架上设置自动暗梯，于是又惊又喜地问：“那暗梯的踏板上，除了做防滑条，我是不是还可以做成凹凸不平的图案，然后踩在上面取书的时候，还可以做足底按摩。”
<br/>
<br/>        “你这个主意倒是很不错！”傅昼景认可地点了点头。
<br/>
<br/>        “那我按针灸来做图案好了，这样，可以按摩足底穴位。”诺筱颖微笑着说着自己的想法。
<br/>
<br/>        傅昼景不禁惊讶道：“你还懂针灸？”
<br/>
<br/>        诺筱颖赧然地挠了挠后脑勺，回答道：“也不是很懂啦！只是小时候，看过我外婆的人体穴位图而已。”
<br/>
<br/>        “我发现，你真是个鬼灵精！”傅昼景笑着说。
<br/>
<br/>        诺筱颖俏皮地眯眼一笑。
<br/>
<br/>        接下来，傅昼景一边耐心地讲解，诺筱颖一边用心地做笔录。
<br/>
<br/>        “咦？这是什么图案？我从来没见过，很漂亮。”傅昼景拿起其中一张设计图，发现柜子的四角，分别有四组独特的装饰边。
<br/>
<br/>        诺筱颖看了一眼后，不禁自豪地介绍道：“这是我们苗族的图腾，寓意着：吉祥如意，财源广进！”
<br/>
<br/>        “你是苗族姑娘啊！”傅昼景感叹，“怪不得你会唱歌！”
<br/>
<br/>        “哈哈！我还会跳舞了！”诺筱颖抬起手来，可爱地比了两个花型给傅昼景看。
<br/>
<br/>        傅昼景笑着说道：“改天有机会，你穿上你那儿的苗族服饰，跳给我看好了！”
<br/>
<br/>        “好啊！”诺筱颖爽朗地应了声。
<br/>
<br/>        言归正传，傅昼景又拿起设计图，非常赞赏地点了点头，接着说道：“嗯，这样的图腾很不错，虽然和超现代风格不太相符。但是，用作柜子上的装饰边条，还是挺好看的！”
