　　密室大门前，如今还是一片混乱，其中的无数武者，纷纷出手，轰击着大门前的禁制。
<br/>　　但是这落长恨四人的实力，显然还是要更高一筹。
<br/>　　那些寻常武者，根本都无法对付他们，就连宇文极，宇文军二人一时间都无法打破这其中禁制封锁。
<br/>　　“看来要帮他们一下了，这局面混乱，我才可以混入其中去。”
<br/>　　楚风眠心中暗自想到，他的身形顿时隐藏起来，无声无息之间接近过去。
<br/>　　“轰！”
<br/>　　在宇文极，宇文军二人联手轰击力量的一刻，悄然之间，楚风眠的野兽凝聚灵力，顿时轰击了过去。
<br/>　　“轰隆！”
<br/>　　在这巨大的力量之下，顿时无数爆炸响彻而起，由这落长恨五人布置下的禁制，在这巨大的爆炸声中，顿时破碎。
<br/>　　“好！打破他们的禁制了！”
<br/>　　“我们快快进去，去收取宝物！”
<br/>　　许许多多的武者，都是大喜惊叫一声，纷纷利用无数遁光，直接冲入到了密室之中去。
<br/>　　“可恶！”
<br/>　　看着落长恨溃败下来，被打破禁制，旁边的沧澜霸怒声喝道。
<br/>　　“落长恨，你就连宇文家的那两个家伙都挡不住吗？”
<br/>　　“沧澜霸，你不用在这里说风凉话！有人突然相助那二人，才打破了我的禁制！”
<br/>　　落长恨突然冷哼，毫不留情的开口道。
<br/>　　“快点追进去，谁敢抢夺这些宝物，就杀了谁！”
<br/>　　说着，落长恨便是遁光一动，直接便是冲入到了这密室之中去。
<br/>　　无数强者，纷纷涌入其中，场面混乱至极。
<br/>　　其中本来无数观望的武者，如今也都纷纷出手，涌入其中去，面对着如此宝藏，不管是谁都无法抵挡住其中的诱惑。
<br/>　　“我们也去，小心一点！”
<br/>　　楚风眠对着白萱说了一句，他的身形猛然一动，也是冲入到了密室之中去。
<br/>　　楚风眠进入密室，并没有收取周围的那些宝物，而是直接化为了一道遁光，向着最深处飞去。
<br/>　　这密室中的上百件宝物，楚风眠一个人想要独吞，他还没有这个本事。
<br/>　　所以楚风眠一早，便是算好，要直接冲入到其中，去夺取其中对楚风眠最为重要的宝物，那块龙血晶石。
<br/>　　这龙血晶石，便是只有真龙一族的无数精血，凝聚而成的一种晶石。
<br/>　　这龙血晶石中，蕴含庞大血气，对寻常武者来说，可以弥补缺失的血气，将身体恢复到完美的地步。
<br/>　　对许多快要突破瓶颈的武者来说，这龙血晶石便是有着极大的用处，将身体恢复到完美地步，是突破瓶颈最为重要的一步。
<br/>　　龙血晶石的价值本就极高，对楚风眠还说，这龙血晶石还有着更多的用处。
<br/>　　楚风眠修炼战龙决，他的身体已经是向着远古战龙开始转变，得到了这龙血晶石，楚风眠可以将其中的龙力都给全部抽出，融入到身躯之中，可以为楚风眠增添力量。
<br/>　　“魅影！”
<br/>　　楚风眠将魅影身法催动到极致，一瞬之间，身形便是消失，远远的脱离开了密室边缘，直接便是向着密室深处飞去。
<br/>　　一眼之间，楚风眠便是隐约看见，就在前方，足有一座小山大小的血红色晶石。
<br/>　　在这血红色晶石上面，散发出无数浓厚血气，楚风眠更是感觉得到，其中的恐怖龙力！
<br/>　　“好强大的龙力！这龙血晶石至少是以龙皇之血凝聚而成的，被我炼化的话，足以助我战龙决的力量，更进一步！”
<br/>　　楚风眠打量了一眼，脸色大喜。
<br/>　　这龙血晶石的品级，比起他想象中的还要更好，对楚风眠有着更大的好处。
<br/>　　“战龙之力！收取！”
<br/>　　楚风眠顿时催动战龙之力，右手化为了巨大无比的战龙之爪，向着那龙血晶石，顿时抓了过去，要将这龙血晶石收取。
<br/>　　“哼，果然有不知死活的人，敢贸然来到这密室深处，想要收取这龙血晶石？给我去死！”
<br/>　　忽然之间，一声暴喝从楚风眠的背后响起。
<br/>　　只见一尊血色婴儿，突然出现，这血色婴儿一掌之下，猛然对着楚风眠，直接轰击了过来。
<br/>　　这一尊血色婴儿，身上的血气浓郁到极致，有着浓厚的血腥味，这上面的血，也不是寻常精血，而是充满剧毒的毒血。
<br/>　　一掌落下，其中蕴含无数毒血，一旦击中，只怕是无数毒素钻入到武者体内，足以将生死境武者，都给直接毒死，化为一滩浓水。
<br/>　　“鬼谷圣子？”
<br/>　　楚风眠看到这突然出现的血色婴儿，顿时明白过来，到底是谁在出手。
<br/>　　正是地魔门的一尊绝世天才，鬼谷圣子，他并没有在看守大门，而是在这密室之中大肆收敛宝物。
<br/>　　如今看到楚风眠出现，想要收取这龙血晶石，他顿时出现，要直接灭杀了楚风眠。
<br/>　　“哼，一个区区神力境四重的废物，也敢来跟我们抢夺宝藏，还想浑水摸鱼得到宝藏？真是活的不耐烦了！”
<br/>　　鬼谷圣子冷哼一声，看向楚风眠，一双眼睛丝毫没有将楚风眠放在眼中。
<br/>　　神力境四重，在他看来，简直就是蝼蚁一般，挥手之间都足以灭杀的人物。
<br/>　　他只是大手一挥，无数灵力涌动起来，那血色婴儿便是一掌蕴含无数毒血，顿时陨落下去，要将楚风眠直接灭杀。
<br/>　　“想杀了我？就凭你这微末魔功？”
<br/>　　楚风眠脸上露出几分不屑，看向这一掌蕴含无数毒血轰击过来，他冷和一声，身上顿时散发出无数金光。
<br/>　　“战龙之威！”
<br/>　　“唰！”
<br/>　　这一掌打在楚风眠身边，在距离楚风眠三重的地方，突然破碎开来，其中的无数毒血，也是瞬间被净化。
<br/>　　这无数毒血，根本便是近不了楚风眠的身，被战龙之威的力量下，全部打散。

<br/>　　“这是什么血脉？居然如此克制我的力量？”
<br/>　　鬼谷圣子眼神中露出几分震惊。
<br/>　　他足以感觉到，楚风眠的力量，简直就是他的克星。
<br/>　　不过鬼谷圣子脸上的表情，很快便是调转回来，露出几分冷笑道。
<br/>　　“有着几分龙族血脉又如何？区区神力境四重的实力，我就不信你还能翻天！”
<br/>　　“毒血降世！万毒齐聚！”
