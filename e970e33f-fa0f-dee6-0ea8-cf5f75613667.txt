        诺筱颖看着傅夜沉离去的背影，有些目瞪口呆。
<br/>
<br/>        这个男人，每次来都对她说一些莫名其妙的话，然后每次都走得这么匆忙……
<br/>
<br/>        其实，她到现在，都还不知道他叫什么，却跟他做过了那些情侣之间才有的亲密举动。
<br/>
<br/>        而且，那些举动，对他来说，好像习以为常。
<br/>
<br/>        那个男人离开后，诺筱颖捡起了地上的画板，继续留在这毛坯房里搞自己的设计。
<br/>
<br/>        现在，对她来说，工作比任何事情都要重要。
<br/>
<br/>        一辆军绿色的越野车跟在了鸣笛的警车车队后，穿梭在车水马龙的柏油路上。
<br/>
<br/>        车内，傅夜沉临时换了防弹衣，怀里抱着狙击枪，一切准备就绪后，他从衣兜里掏出了诺筱颖的那绺青丝，趁着这前往现场的时间，快速编织了一条凯尔特心结手链，戴在了左手上。
<br/>
<br/>        他接到消息，郊区有个女孩被歹徒持枪绑架，需要他们特种部队协助解救人质。
<br/>
<br/>        每一次，他接到的任务，都是这种枪林弹雨、刀光剑影，歹徒穷凶极恶，有着生命危险的任务。
<br/>
<br/>        以前，他每次出任务，从来都是心无牵挂。
<br/>
<br/>        如今，他的心里有了牵挂，才会更加珍惜自己的性命。
<br/>
<br/>        一旁的战友韩剑锋见傅夜沉心事重重的样子，忍不住拍着他的肩膀打趣：“你啊！就是那种很容易在一棵树上吊死的男人！”
<br/>
<br/>        “我觉得在一棵树上吊死挺好。”傅夜沉云淡风轻地答。
<br/>
<br/>        第一次邂逅，一个迫不得已的吻，几句简单的对话，他就这样无可救药地爱上了她。
<br/>
<br/>        其实，他也没想过自己会这么快就有了心爱的女人。
<br/>
<br/>        感情这种东西，真的很不可思议，甚至微妙到无法用语言去形容。
<br/>
<br/>        “四少，你喜欢四嫂什么？”韩剑锋接着笑呵呵地问，毕竟，从来不近女色的男人，突然说自己有了心爱的女人，这能不令他好奇吗？
<br/>
<br/>        傅夜沉却剑眉紧锁，拿起手里的平板电脑，一边查看解救计划，一边严肃地转开了话题：“这种计划，对人质不利！”
<br/>
<br/>        城市的东郊，有一座荒弃的烂尾楼，灰青色的钢筋混凝土框架，触目惊心地独立在一片齐齐的有半人多高的杂草之中。
<br/>
<br/>        部分野草已经枯黄，风吹得瑟瑟飒飒地响，忽然间，一只白色的无名鸟从草丛里扑棱飞起，像幽灵般发出一声哀啸地长鸣，平添一份惊悚诡异气氛。
<br/>
<br/>        通往烂尾楼的楼梯已经被里面的歹徒封锁，警察无法上楼，只能将这里包围。
<br/>
<br/>        被绑架的人是季氏集团董事长的小女儿季雅姗，绑匪要求季董事长拿一千万出来换人。
<br/>
<br/>        因为周遭没有其他楼房，偷偷派去的无人机，也被歹徒枪击坠毁，所以，他们无法了解里面是什么情况，不过，通过远程红外检测，可以判定里面有五个歹徒。
<br/>
<br/>        傅夜沉听完身边女警员的分析后，决定用徒手攀爬的方式，进入这栋烂尾楼里。
<br/>
<br/>        在场所有警员，行动沉稳、谨慎、迅速，全部准备就绪后，特种部队一切听从傅夜沉的指挥，顺着烂尾楼的混凝土柱，一个个带着攀爬绳索，悄悄地爬到了人质被绑架的楼层。
<br/>
<br/>        室内只有三个戴着黑色丝袜头套，手里持枪的男人站在边缘查看楼下警察的动态，另外两个戴着黑色丝袜头套，手里持枪的男人则守在了楼梯口。
<br/>
<br/>        傅夜沉悄无声息，且非常安全地抵达到被绑在一根柱子上的女孩季雅姗的身边，并对季雅姗及时做了一个噤声的手势。
<br/>
<br/>        女孩只有十六七岁的样子，身上还穿着学校里的校服，有点儿婴儿肥的脸蛋上沾上了灰尘，两只大大眼睛，水灵剔透，炯炯有神，却满是畏惧与无助。
<br/>
<br/>        当傅夜沉解开女孩身上的绳索，带着她站起身来时，才发现她背后还绑着一颗定时炸弹。
<br/>
<br/>        因为刚刚女孩的起身动作，拉动了定时炸弹上的读秒器。
<br/>
<br/>        傅夜沉看到这里，立即从口袋里掏出一把削铁如泥的匕首，隔断了绑在女孩腰间的铁链。他的动作再怎么蹑手蹑脚，但终究还是惊动了歹徒。
<br/>
<br/>        一同上来的战友在韩剑锋的带队下，给傅夜沉做了掩护，和五个歹徒展开了枪战。
<br/>
<br/>        傅夜沉切下炸弹包后，以最快的速度，跑到楼板边缘，将炸弹包向空中用力抛去。
<br/>
<br/>        不过几十秒的时间，炸弹在半空中爆炸，“轰！”一声惊天动地的巨响，一阵强劲的冲击波袭来，将楼层里所有人震飞，伴随着地动楼摇，滚滚浓烟在漫天中散开。
<br/>
<br/>        傅夜沉趴倒在地的时候，只觉两耳隆隆作响，耳鸣到头晕目眩，其它声音什么都听不到。但是，他却敏感地察觉到，紧贴在他身下的楼板在发出共振的鸣响。
<br/>
<br/>        不好！
<br/>
<br/>        傅夜沉第一个意识到情况不对，大声命令道：“火速撤离！”
<br/>
<br/>        然而，就在这顷刻间，大楼晃得更加厉害，细碎的混凝土簌簌落下，扬起满屋的灰尘。
<br/>
<br/>        傅夜沉离那女孩最近，他潜意识地迅速从地上爬起来，跑到那女孩身边，一把将女孩扛在了肩头，直奔楼梯口，并往楼上跑去。
<br/>
<br/>        韩剑锋带队，拽住那几个穷凶恶极的歹徒，和战友们一起火速奔跑在了傅夜沉的身后。
<br/>
<br/>        “轰隆——”的响声再次掩盖一切声音，灰尘滚滚，将他们的身影瞬间吞噬。
<br/>
<br/>        他们是生是死，外边的警察只能眼睁睁地看着烂尾楼，在顷刻间坍塌成一片废墟。
<br/>
<br/>        生死时报，满大街的液晶屏幕里，实时新闻都在报道东郊烂尾楼里发生的绑劫爆炸案。
<br/>
<br/>        诺筱颖离开傅氏集团大厦后，上了一辆公交车。而公交车里的移动电视，也正报道这件事情。
<br/>
<br/>        现场的画面一片混乱，荧幕上有烂尾楼坍塌的画面，灰尘犹如浓烟般在空中扬出了一个骷髅头的恐怖形状，弄得人心惶惶。
<br/>
<br/>        大家都在为救助人质的那支特种部队祈祷。
<br/>
<br/>        公交车上的大妈大婶，开始纷纷祷告：“好人一生平安！”
<br/>
<br/>        诺筱颖看着移动电视里的画面，脑海里莫名其妙地联想到那个男人时，突然一阵心悸，痛得她胃里像是有什么东西在翻滚。
