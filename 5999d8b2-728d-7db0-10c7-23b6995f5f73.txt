　　房间内，几个人看着SHADOW摆放在摄像头前，写在纸上的坐标面面相觑。
<br/>　　半晌后，慕容飞才是开口道：“这特么的怎么看都像是一个圈套啊，这种低级到连小学生都能看出来的弱智圈套，SHADOW也好意思拿出来显摆？老子打心眼里看不起他！”
<br/>　　子午拿起桌子上的便利贴本子和铅笔，唰唰唰，把纸上的坐标给抄了下来。
<br/>　　“子午教官，你真打算去跟SHADOW对决？”
<br/>　　罗昊看到子午将坐标抄写下来的动作，感到意外。正如慕容飞所说，这根本就是一个低级到连小学生都能看出来的弱智圈套！周围说不定已经埋伏好了狙击手，或者布设了地雷，就等着子午往里面跳呢。
<br/>　　“你们不了解SHADOW。”子午将抄写下来的坐标重新校对了一遍，确认坐标无误后，收起纸条，然后把后面那张便利贴也跟着撕了下来，撕碎扔进垃圾桶，说道：“这家伙既然提出了要跟我对决，就一定会想方设法逼我现身，他一定还会有后手的。况且，我本来就是为了要找他才会来这里，他这么诚心的邀请我跟他对决，我没有任何理由拒绝。”
<br/>　　子午所说的SHADOW所准备的后手，很快就在网络上曝光了。
<br/>　　这是一个SHADOW所录制的视频，视频背景跟刚才SHADOW和子午视频对话时所在房间背景一样。
<br/>　　房间内没有灯光，窗帘紧闭，连帽卫衣的SHADOW坐在一张桌子前，双手手指交错相叠支着下巴，宽大的卫衣帽子扣在鸭舌帽上，将他本身就被鸭舌帽挡住了大半部分的脸彻底罩在了阴影之下。
<br/>　　视频中，SHADOW将自己的声音做了处理，声音变得沙哑而厚重。阴影中，SHADOW用沙哑而厚重的声音说道：“五天后，我会与GHOST进行我们两个人之间的终极对决，这场对决，我们两个人只有赢得人能活下来。”
<br/>　　SHADOW战书一出，顿时引发一阵哗然。SHADOW和GHOST都是成名已久，属于传说级别的人物，即便是现在，网络上都有争论SHADOW和GHOST谁更强的言论，但是自从那一战后，SHADOW和GHOST全都是消失了，所以这两个人谁更强到现在都没有一个明确的定论。
<br/>　　现在这两个传说级别的人物终于是要一战了吗？而且还是生死大战！
<br/>　　“GHOST，你可以选择像一个缩头乌龟一样，龟缩着藏起来，毕竟你要藏起来，我也没有任何办法。”视频中，SHADOW继续说道：“不过，如果你不应战，我会在五天后释放一批电脑病毒，比勒索病毒更加恐怖的电脑病毒，这批电脑病毒会让全世界的网络陷入瘫痪……”
<br/>　　视频播放到这里就结束了，视频最后SHADOW看似给了子午一个选择，但是随便是谁都能看出来，这根本就不是选择！
<br/>　　“SHADOW这混蛋，这招够毒的啊。”
<br/>　　慕容飞撇嘴骂咧道：“这是把子午教官架在火堆上烤啊！”
<br/>　　子午对此倒是没有太大的反应，SHADOW会有后招逼自己应战，是早有预料的。
<br/>　　李然沉吟一声，说道：“SHADOW这个视频发布出来，是切断了子午教官的全部退路。原本大家或许只是对SHADOW和GHOST之间的这场终极对决感到期待，但是现在情况发生了改变，子午教官不得不跟SHADOW正面交手，如果子午教官选择退避的话……”
<br/>　　“SHADOW一定会把子午教官现在的身份曝光出去，如此一来，全世界会把矛头指向我们国家。”罗昊倒吸一口冷气，说道：“这才是SHADOW这次所下战书中，最毒辣的一环！”
<br/>　　注意到房间里的人把目光全都看向自己这边，子午只是微微一笑，说道：“那就来吧，我们之间总是要有一战的。”
<br/>　　第二天一早，来接应洛菲克的人到了，在数名全副武装雇佣兵的护送下，洛菲克上了一架停在停车场上的直升机，直升机从小镇上空飞过的时候，洛菲克还十分嚣张的对着站在小镇广场上的罗昊、李然等人竖起一个中指。
<br/>　　“趁还有时间就多嚣张一会儿吧。”罗昊不屑的冷笑，说道：“再过一会儿，你就嚣张不出来了。”
<br/>　　洛菲克离开后没多久，前来接应罗昊、李然他们的人也到了，负责人姓汤，三十来岁，戴着一副黑框眼镜，看上去很斯文的一个人。他没多介绍自己，只说称呼他“汤姆熊”就可以了，平时在工作的时候，手下人也是这么称呼他的，久而久之也就习惯了。
<br/>　　“我们需要六张机票，其中两张回国，其余四张去K国。”罗昊对正在开车的汤姆熊说道。
<br/>　　巫师和周柏都中了毒，虽然不危及生命安全，但罗昊还是决定不让他们两人参与到这次行动中。
<br/>　　“没问题，我会在最短时间内办妥你们所需要的东西。”汤姆熊驾驶着汽车，回答道。
<br/>　　汤姆熊话不多，在去往临时安全屋的途中，除了这样一句简短的对答外，就再没有更多的交流；不过他的办事效率很高，下午的时候，就把机票和护照送到了罗昊手中。
<br/>　　护照上，除了照片是真的以外，其他信息全都是伪造的，但是护照本身却是真的。
<br/>　　罗昊将巫师和周柏的护照和机票交给两个人，说道：“今天晚上的飞机，汤姆熊会派人送你们去机场，一路小心。”
<br/>　　“真的不需要我们去帮忙吗？”周柏对罗昊问道。
<br/>　　“不用。”罗昊轻轻摇头回答道：“真有什么情况，我们几个人可以应付，宿鸟已经带着野牛、蜂鸟和墓碑提前前往K国，我们会在K国汇合。”
<br/>　　周柏闻言，收起桌子上的护照和机票。
<br/>　　当天晚上，周柏和巫师就搭乘飞机回国；罗昊、李然、慕容飞、李毅峰和子午是第二天中午的航班，他们可以睡一个好觉，养足精神以后再出发。
<br/>　　在万米高空飞行了十几个小时后，庞大的A380客机降落在K国机场，五个人拎着简单的行李刚从抵达出口走出，就看到了站在候机楼大厅中，戴着一副墨镜，骚包到不行的林晓阳，而在林晓阳身边还站着一个靓丽的身影却是完全出乎了罗昊的意料。
