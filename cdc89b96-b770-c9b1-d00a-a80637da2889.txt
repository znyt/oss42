　　全面图一出就吸引了在场所有人的目光，苏行工程队的能力早在狼牙墓碑建造而出的时候，就已经让整个狼牙都为之震惊，而当下众位狼牙军官又再一次被苏行工程队折服。
<br/>　　所有人都很清楚一场战役前期的准备有多重要，狼牙战团的不断扩军和训练，就是为了一次性的大爆发，而众基地的支持和资源共享，以及狼牙那些部门****夜夜的研究和赶制所需品，更是大战胜利的奠基石。
<br/>　　苏行的工程队，则是最重要的一环，负责了战地优势这一极具重要性的任务！
<br/>　　“相信你们看到地图的时候，就差不多明白该怎么打了。”何峰开口说道。
<br/>　　陈少爷点着头，指着那些坐落于基地各处的高台，道：“这是射击队的方位，一共五百个高台？”
<br/>　　“不，四百个。”何峰纠正道，而后没有多解释就再次开口，并且看向了狄升：“四面城区的高墙内已经放置了新武器，交由防守队使用。”
<br/>　　听到这样的安排众人都无异议，虽然当下谁都想尝试一下新武器，见识一番这批武器的威力，但在这场战役之中防守队不适合与他们站在同一战线，战斗模式独树一帜的防守队更无法配合其他队伍的行动，让他们退进高墙内部掌管那些新武器，最为合适。
<br/>　　紧接着何峰又点着四面高墙道：“防守队注意，这四面墙内有暗道，只能供一人行走，所以进去之前先分配好每个定点，新武器的攻击方向只有一处，那就是面向基地外部的方向，威能极强，在大尸潮攻来的时候就可以开火了。”
<br/>　　“能量用光为止？”狄升有些紧张的开口发问。
<br/>　　何峰面上浮起了一个意味不明的笑容：“恐怕等你们手中新武器用到毫无威力的时候，大尸潮的冲击还没结束呢！”
<br/>　　狄升心底一惊，这才恍然察觉到这次大尸潮的数量极为可怕，必须毫无保留的攻击才行！
<br/>　　“明白了。”狄升慎重点头。
<br/>　　随后何峰又道：“你们仔细看下这四面外墙，高墙内虽有新武器，但同样每隔一段距离就有扇门，那是引丧尸进来的通道，到时候会有无数丧尸头破血流的往内墙挤，外面的高墙大程度的挡住尸潮，防守队就朝着丧尸最多的地方开火，不需要射击技术，把外面地面砸开了都没事。”
<br/>　　“其余的丧尸会沿着通道进入两面墙的中心地带，但它们面前还有一座内墙，内墙虽然也有数道门，但想要进入基地没那么容易，还得先面对狼牙战团从北面带来的那三万新兵。”
<br/>　　“三万人就坚守这两座墙之间，他们每个人都是新人类，分成两次轮班，以小队形式合作抵挡丧尸，由于已经将丧尸分流从外墙的通道分批进入，所以收到的冲击已经减缓许多，这三万人有精英队的老兵带着他们只要好好配合就行。”
<br/>　　“但他们无法抵挡全部的丧尸群，所以外墙和内墙的这些通道都建造成了依次岔开的阵型。”何峰一一解释着，不放过任何一个细节，同时看向了负责三万人大队的秦渊中将：“三万人分两批轮班，这次大战虽然需要全力以赴但不是拼命，所以如有不敌立即再次分波，由老兵引流将部分丧尸放进基地，也就说从内墙的通道走，你们只需要在能力范围内杀丧尸即可。”
<br/>　　秦渊面色一震，严肃点头：“明白！”
<br/>　　紧接着何峰又指了指外墙：“这外墙建造的又高又宽，墙顶其实已经全部砌成了平地，边缘地带有高出一小节的防护栏。”
<br/>　　听到这里的徐锋眼神一变，犀利道：“这是三战队呆的地方？”
<br/>　　“不错。”何峰慎重的看着三名战队队长：“高的这面外墙只能抵挡低阶丧尸，但是五阶以上的丧尸完全可以爬上来，七阶八阶那样的丧尸更不用说，数量是多少谁也不知道，但依照楚涵的判断，这次大尸潮很有可能会有丧尸王，尸潮也不一定只有一波，说不定会来好几次，这些分批出现的尸潮也必然会有超级丧尸统领，也就是说高阶丧尸哪怕出现也不像以前的尸潮那般一股脑的拥挤在最后，而是夹杂在低阶丧尸之中。”
<br/>　　江凌若明白了过来，道：“也就是说我们虽然任务艰巨，但并不是一次混战，高阶丧尸不会一次性冒出来，而是一只接着一只，一批接着一批，我们这次需要注意体能保留，需要持久战？”
<br/>　　“对！”何峰欣慰的看着江凌若，道：“三战队分配到的银净水数量最多，也就是这个原因，你们可没有轮班。”
<br/>　　路冰泽当下也一改嬉皮笑脸，严肃道：“明白了，这回耍不了小心机，要正面应战。”
<br/>　　何峰点着头，接下来就看向了杨天：“五万新兵不及那些老兵带回来的三万人战力，他们没办法待在两墙之间，唯有通过基地内部这些房屋的地理优势来作战。”
<br/>　　“游击？”杨天瞬间了然。
<br/>　　何峰面上的笑意越来越浓：“就是游击，基地外圈的两墙只占了一小部分的面积，最大的战区还是在内部，五万人的站位如何分配不用我多说吧，城市作战你比我清楚。”
<br/>　　杨天嘴角勾起：“别的不说，基地城市包在我身上，更何况这城市的迷宫图都已经在面前，那些丧尸可不认得路。”
<br/>　　“接下来就是侦查队。”何峰望向了年级最轻的中将蒋天庆：“你们负责的是游走，清点混入基地内部的高阶丧尸，无论是从三战队防线出落单的，还是从下方通道战区的内墙进来的，而后以最快的速度传递给射击队。”
<br/>　　说到这里何峰看向了陈少爷：“射击队的任务也不用我提点，这种合作你们打过很多次，落单的高价丧尸，以及射击范围内临时出现的变故，都交由射击队援助，任务看起来只是辅助但不可或缺。”
<br/>　　陈少爷还是那副憨厚的表情，问道：“那丧尸引诱剂？”
<br/>　　其余人也看向何峰，这一次大尸潮战役中，怎么没有提到丧尸引诱剂？
<br/>　　这可是每次对抗丧尸的必要用品啊！
