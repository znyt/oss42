　　“这夜家的人，居然就这样走了？”
<br/>　　“夜众星，居然是带来了奖励，这龙丹，简直是无价的宝物，就这样赐予楚风眠了？”
<br/>　　“看来学院的高层，也不想对付楚风眠，毕竟这样一尊天才，在我们北邙学院崛起，对我们北邙学院也有着极大的好处。”
<br/>　　一尊尊种子弟子，看着这一切的发生，心中都是有着各自的考虑。
<br/>　　其中的一些种子弟子的心中，都有着一种想要投靠楚风眠的心思。
<br/>　　夜天君一死，现在学院的表示，在他们看来，这绝对就是代表着楚风眠即将要成为北邙学院新的第一人的象征。
<br/>　　不少种子弟子的心中都在考虑，如何来投靠楚风眠。
<br/>　　虚空深处。
<br/>　　两道遁光的速度，快到极致，这二人，正是那夜众星跟夜牧老祖二人。
<br/>　　如今夜牧老祖脸上的怒火，也还是久久不消，他看向夜众星的目光，都极为不满。
<br/>　　“难道家主就这样看着这小子耀武扬威？杀了我们夜家的人，还加以培养？”
<br/>　　夜牧老祖冰冷的开口道。
<br/>　　“学院的那些老古董，难道不想活了，敢跟我们夜家作对？”
<br/>　　“夜牧，冷静一点，这个安排，就是家主亲自定下的。”
<br/>　　夜众星听到夜牧老祖的话，平静的开口道。
<br/>　　“家主的心中，早有算计。”
<br/>　　“这楚风眠，身上有着大气运，我曾经试探了他，此人的剑道，绝非是他这般年纪可以修炼而成的，很有可能他还有着我们所不知道的许多奇遇，这次归墟天之行，以他的气运，足以是得到无数好处。”
<br/>　　“家主便是计划，打算利用他的气运，从归墟天中得到无数宝物，再来对付他，倒时候他的一切奇遇，都将属于我们夜家。”
<br/>　　夜众星也是流露出了几分杀意。
<br/>　　“杀了我们夜家的人，怎么可能就这样善罢甘休！”
<br/>　　两人一边交谈这，一边身形便是就在虚空之中，逐渐消失。
<br/>　　而另一边，悟剑峰上。
<br/>　　一座石桌，楚风眠，寒月漓，玄北圣者三人，都坐在这石桌前的石凳上，交谈着。
<br/>　　“这一次武斗大会，本来为师以为你只能进入前十，夺得一个名额，倒是没想到你居然夺得了榜首的位置。”
<br/>　　玄北圣者看着楚风眠，他也感觉到，楚风眠身上的气势，似乎是跟离开之前，又有了变化。
<br/>　　楚风眠的境界，没有任何的变化，但是在这武斗大会上，他见识了无数天才的剑术，也令他的九域剑术，得到领悟，逐渐升华。
<br/>　　境界未变，但是楚风眠现在的实力，的确是跟上一次离开北邙学院的时候，截然不同了。
<br/>　　这种实力的变化，也体现在了楚风眠身上的气势上。
<br/>　　不过玄北圣者，却并没有出言去问。
<br/>　　每一尊武者，都有着自己的秘密，有着自己的底蕴，楚风眠身上秘密众多，他也不打算去问。
<br/>　　“你这次杀了夜天君，倒是惹来了一个不小的麻烦，我之前倒是忘记提醒了你，这夜天君背后的夜家，可不好对付。”
<br/>　　玄北圣者开口道，提到夜家的时候，他的脸色，都是变得极为凝重。
<br/>　　以他的实力，在北邙学院中，都是无数老祖中的强者，但是对夜家，他也是极为忌惮。
<br/>　　玄北圣者提醒道。
<br/>　　“这次你杀了夜天君，夜家肯定不会善罢甘休，这次你是在武斗大会上，杀了夜天君，更是夺得了榜首的位置，算是对学院有着大功，所以现在夜家的人，不会杀你，但是肯定不会放弃杀了你的计划，你还要小心才行。”
<br/>　　“嗯。”
<br/>　　楚风眠同样点了点头。
<br/>　　玄北圣者这说的，楚风眠自然清楚。
<br/>　　这一次，虽然看起来是学院没有怪罪楚风眠，更是赐下了赏赐。
<br/>　　但是楚风眠清楚，这也是做给其他人看的。
<br/>　　楚风眠夺得武斗大会榜首的位置，便是为北邙学院杨威，有着大功，现在北邙学院杀楚风眠，便是在杀功臣。
<br/>　　一旦是现在杀了楚风眠，绝对是会引起极大的反应，所以夜家的人，不敢杀楚风眠。
<br/>　　正是算准了这一点，楚风眠才敢这样大摇大摆的回到北邙学院之中去。
<br/>　　就算是刚刚玄北圣者不出手，那夜众星，也会出手，阻止夜牧老祖的。
<br/>　　不过夜家的人，也绝不可能会放过楚风眠的。
<br/>　　这一次放了楚风眠，但是下一次，夜家可就是要来真的了。
<br/>　　“你在留在北邙学院中，有着极大的危险。”
<br/>　　玄北圣者想了一下，开口道。
<br/>　　“以你现在的资质，不管是前往任何一方势力，就算是七大宗门，七大宗门都会庇护与你，你不如先离开北邙学院，躲避夜家的追杀。”
<br/>　　“不，我不会离开。”
<br/>　　楚风眠摇了摇头开口道。
<br/>　　“现在学院刚刚赐下奖励，这个时候我若是离开，便是正好给夜家一个理由，倒时候他们杀我，正好是有了理由。”
<br/>　　“夜家又如何？这一次我前去归墟天，正是要去度过生死大劫，只要是我度过了生死大劫，那夜家的人又如何？夜牧老祖？夜众星？我都可以斩杀，倒时候夜家的人，在敢来对付我，也要考虑一二。”
<br/>　　楚风眠自信的开口道。
<br/>　　度过生死大劫，便是就要斩杀超越生死的老古董。
<br/>　　这种话，要是其他武者说出来的，玄北圣者只会以为那个武者脑子坏了。
<br/>　　就算是在狂妄的天才，都不敢说出这种话。
<br/>　　但是现在，从楚风眠的口中，说出这番话，玄北圣者的心中，却是相信。
<br/>　　当初楚风眠前往武王城，参加武斗大会的时候，在他看来楚风眠最多只能进入前十，夺得一个归墟天的名额。
<br/>　　可现在，楚风眠就得到了榜首的位置，更是杀了夜天君。
<br/>　　当初在武斗大会擂台上，楚风眠可以抗下夜牧老祖的一道刀锋。
<br/>　　只要是楚风眠度过生死大劫的话，也许楚风眠的实力，真的会达到跟这些老祖抗衡的程度。
<br/>　　听到楚风眠如此自信，玄北圣者也不准备说什么。
<br/>　　若是想要成为一代强者，遇到危险，自然不能逃，这种人挡杀人佛挡杀佛的性格，才能成为真正的强者，挡在面前，诸天神佛，一样斩杀。
