　　见到猎物失去了反抗能力，尸兄的八颗脑袋皆是露出有些兴奋的表情，兴冲冲的向小胖撞去。
<br/>　　这一下如果撞到小胖身上，几乎不会有任何存活下去的可能了。
<br/>　　就在这千钧一发之际，刘哔身影一动，瞬间出现在尸兄身后，轻飘飘的伸出一只手掌，握住了八头尸兄的一根手指。
<br/>　　“诶嘿嘿哈哈哈，你是在找死么……”
<br/>　　扩音器传来嘲讽的声音，但仅过去几秒钟的时间，又戛然而止。
<br/>　　“轰！”
<br/>　　墙壁倒塌的轰鸣声响起，刘哔单手握着八头尸兄的手指，狠狠将其砸在了一边病房的墙壁上。
<br/>　　随后，不慌不忙的抬起右脚，狠狠一脚向尸兄胸口踏去。
<br/>　　一阵肉眼可见的气浪从刘哔脚下传开，将尸兄庞大的尸体震成了肉沫，一时间，走廊像是下起了血雨一般，将小半条走廊染得血红。
<br/>　　置身在这修罗场一般的场景中，刘哔双眼通红，抬头看了一眼摄像头的方向，眼中，露出像野兽一般的凶芒。
<br/>　　“乖乖等好，我会去找你。”
<br/>　　唇形微动，但却没有发出一点声音，顺手吞噬掉八头尸兄的残骸，刘哔拖着小胖，掀开了走廊内紧闭的铁珊栏。
<br/>　　扬声器内久久没有声音，或许，博士并没有再看他们，又或许，博士受到惊吓，早就已经离开了。
<br/>　　刘哔加快了速度，挨个房间搜寻过去，最终停在了一间病房前。
<br/>　　门上挂着一个标牌，上面是写着几个金色的阿拉伯数字，507，是这间病房的名字。
<br/>　　嘴角露出一抹笑容，他感应到里面有几股生人的气息，看样子，这个病房内，不仅仅只有博士一个人。
<br/>　　“诶嘿嘿嘿嘿，晓佳大夫，我这个人，是从来不喜欢强求的，可是如果你不答应我，这个小家伙，就会立马变成一个小尸狗了哦。”
<br/>　　门内传来博士有些病态的笑声，刘哔眉头一挑，顿住了想要打开房门的动作。
<br/>　　“我这枚针，可是最新研制的，只要你心甘情愿的，做我的女人，我就……不会伤害它~”
<br/>　　没察觉到房门外的异样，刚才刘哔和八头尸兄战斗的时候，博士并没有多过于关注，在他眼中，或许自己早就成了那头尸兄的腹中之餐了吧。
<br/>　　“答应的话，就给老子点点头！”
<br/>　　博士的笑声继续在里面传来，刘哔失去了耐性，伸手敲了敲屋门。
<br/>　　门内的声音戛然而止，片刻，博士的声音从扬声器中传出：“还真是小看你了，就连大队长都解决不了你们。”
<br/>　　和先前不同，博士的声音，开始认真了起来。
<br/>　　“不好意思，打扰您的雅兴了，请问您能麻烦一下，将门打开吗，如果您不打开，那我可就自己进去了。”
<br/>　　手中拖着昏迷的小胖，刘哔一脸恭谦的站在门口，等待着博士的回应。
<br/>　　片刻，屋内传来了一阵脚步声，一个身穿白色医服的瘦高男子开门，然后戒备的向后退去。
<br/>　　“谢谢，这样是再好不过了。”
<br/>　　脸上带着礼貌的微笑，刘哔从门口进来，打量着病房内的场景。
<br/>　　这是一个被改造成监控室模样的房间，一些监控仪器胡乱摆放着，有几张简单的家具，两个被绳子绑住的女人，正躺在地上不断挣扎着。
<br/>　　看到刘哔进来，女人眼中露出一种叫做希望的光芒，之前博士虽然没有注意，但刘哔和八头尸兄的战斗她们可都看到了。
<br/>　　如此强大，肯定能就她们出去，况且，看监控器上的唇形，也不是让她们乖乖等好，说会来找她们的吗。
<br/>　　肯定是来救她们的，她们，得救了！
<br/>　　“咦，还挺好看的，怪不得让你这样的人都这样执着。”刘哔瞥了一眼被束缚住的两名女子，饶有兴致的点评了一句。
<br/>　　事实上，这两个人的颜值都很不错，毕竟在危机爆发之前，也是属于院花那种级别的美女。
<br/>　　只是像现在这样的环境，长的这么漂亮，自然也会引起一些幸存者的不怀好意。
<br/>　　虽然博士对那什么晓佳大夫的心思也不单纯，但不可否认的是，如果失去了博士的庇护，这个晓佳大夫，现在还不知道是什么场景。
<br/>　　就像他原本的世界中，这种级别的女子，如果没有太过于强大的后台或者实力支撑，大都会沦为一些强者的玩物。
<br/>　　在末世初期的话，甚至会被不知道多少人玩弄，这是极为可悲，又无法避免的事情。
<br/>　　“呜呜~”
<br/>　　一阵微弱的狗叫声响起，刘哔目光一转，看到了躺在地板上的一只白色茶杯犬，一根空空如意的针管，被随意扔到茶杯犬旁边。
<br/>　　是哪个变异狗么？
<br/>　　刘哔的注意力很快被茶杯犬转移了过去，对于这种变异，他心中略微有些好奇。
<br/>　　博士所制作的强化药剂原理，他大概有一些了解，无非是用一些其他的物质来中和病毒，借此强行催化食用者身体上的潜力。
<br/>　　但这样的催化十分危险，如果中和不好，就会被病毒彻底入侵，而变成为没有理智的怪物。
<br/>　　这也是博士变态的地方，像这样的实验，在刘哔原本的世界中，早在几年前就有人提出了，甚至还组建了许多顶尖科学家进行研究，但一直没有取得什么进展。
<br/>　　而这一整个科研团队都无法做到的事情，却被博士一个人，在这样艰苦的条件下完成了，甚至还研究出了如何控制尸兄，以及让动物变异后保持理智和记忆的办法。
<br/>　　这也是刘哔对此感兴趣的原因。
<br/>　　是个人才。
<br/>　　“你是真的有些本事，竟然能找到这里，但是……差不多，也就适可而止吧！”
<br/>　　博士的声音有些沉闷，完全没有了先前的跳脱，刘哔目光一凝，注意到了其不知何时注射完毕的四枚针管。
<br/>　　一阵压抑的喘息声响起，博士头部微垂，像是在忍受着非人的痛苦，让人心感不妙的是，他的嘴角，却带着一股变态版的笑容。
<br/>　　“这可是，我最得意的作品，神之基因……知道为什么取这个名字么，是因为，它能给你带来的力量，足以匹敌神灵啊！”
<br/>　　博士脸上的笑容越来越疯狂，而他身上的衣服，也被肌肉撑得寸寸裂开，露出了下面夸张的古铜色皮肤。
