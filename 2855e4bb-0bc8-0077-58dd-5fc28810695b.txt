<br/>        陈明不仅要将这株竹子炼制成宝物，还要炼制成一件特殊的宝物，毕竟，像是这样适合陈明的天材地宝，可谓是万中无一，陈明想要将这株竹子活着炼成宝物。

<br/>        若是直接用火炼化，提取精华，那么便是死物，陈明要的，是把这株竹子炼制成活物，有自己意识的神兵，有成长性的神兵。

<br/>        看了不少收集到的炼器经文，并没有陈明想要的活炼之法，都是将材料炼制成死物。

<br/>        像是活炼之法，恐怕只有仙门才会有收藏，但是要去找群星仙门要活炼之法，怕不是要被打死，这样的秘密，会告诉陈明一个外人。

<br/>        那么怎么办呢？
<br/>    
<br/>        从星冥的口中，陈明也是知道，玉虚宫曾经可是一处仙门，而且是距今为止最后一位仙人的出处，不过经历鬼仙的事，迅速跌落云端。

<br/>        玉虚宫的宗主长叹一声，然后道：“对于这件事，我也不是很清楚，只是知道祖师在成仙之后，似乎是发生了一件大事，将整个宗门的资源全部挥霍一空了，经历过那一件大事之后，我们玉虚宫便沦落至此，而祖师，也不知了去向。”

<br/>        看来玉虚宫对于对抗鬼仙这件事，并不怎么知情，这位丹鼎仙人也是惨，刚刚飞升成仙，就遇到了比他更强的鬼仙，若不是靠着对这一块大陆的了解，占着主场优势，恐怕连封印鬼仙都做不到。

<br/>        陈明继续问道：“那你们祖师，就没给你们留下什么有用的东西？”
<br/>    
<br/>        玉虚宫宗门道：“倒是留下了一些莫名其妙的经文，但是失去了祖师，根本没人看得懂啊！”
<br/>    
<br/>        陈明道：“可以带我去看看吗？”
<br/>    
<br/>        玉虚宫宗门倒是不怎么介意，数百年前，也有不少宗门打这些经文的主意，不过看了之后都很失望，根本没有得到什么有用的东西。
<br/>    
<br/>        带着陈明来到一处快要腐朽的阁楼，玉虚宫宗主伸手指向阁楼，然后道：“这里便是祖师当年居住过的地方，其中收藏着祖师时不时翻开的经文。不过原本都已经腐朽了，这些都是拓本。”

<br/>        随手拿起一本书，陈明翻开一看，嗯，这真的是字？不是鬼画符？

<br/>        怪不得没人看的懂，原来连字都认不出来了！

<br/>        估计丹鼎仙人自己，都想不到自己留下的这些珍贵的经文，最后居然是因为字迹太潦草失传了。
