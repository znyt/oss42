　　四品炼药炉，离火金炉拿出。
<br/>　　这时，易老却忽然抛来一个乾坤袋。
<br/>　　萧逸一愣，疑惑道，“易老，这是？”
<br/>　　易老淡淡道，“我早知你能完成那些任务，并存够五千剑点，兑换凝丹五行录。”
<br/>　　“也早知以你的领悟力，必可悟透凝丹五行录，自然要提前给你准备好炼丹材料。”
<br/>　　萧逸打开乾坤袋一看，顿时一惊。
<br/>　　这是一个高级乾坤袋，空间极大。
<br/>　　天材地宝无数，最低都是三品药材。
<br/>　　妖兽内丹数以千计，最差都是三级。
<br/>　　“这....”萧逸一惊，这乾坤袋里的东西，价值惊人，甚至比上次抢夺盛宝商行的那批商物还要高数倍以上。
<br/>　　萧逸很清楚，易老是个性格古怪，又墨守成规的人，他虽然守着外门宝库，却绝不会动用半分。
<br/>　　证明，这些东西，都是易老自己找来的。
<br/>　　而且，易老不可能知道自己短短半月就能完全悟透，所以，他可能是提前很久就已经准备好了的。
<br/>　　恰好一月前，自己刚回来时，易老竟破天荒地不在外门宝库....
<br/>　　萧逸已经能猜到，肯定是之前自己出去历练时，前脚刚走，易老后脚就去给自己准备这些材料了。
<br/>　　数量如此庞大的天材地宝和妖兽内丹，易老肯定是煞费了不少心机。
<br/>　　“易老....”这一刻，萧逸心中一阵暖流缓缓躺过。
<br/>　　来到这个世界后，除了依依和萧家长辈，也就只有面前这个老人对自己这么好了。
<br/>　　“不用废话。”易老板起了脸，道，“开始炼丹，让我看看你的水平。”
<br/>　　“是。”萧逸认真地应答一声，开始炼药。
<br/>　　他准备炼制洞玄丹，洞玄丹的丹方，早在之前的炼药指导前，易老就给了他，他也会炼。
<br/>　　一份份材料投入炼药炉。
<br/>　　待得洞玄丹即将凝丹时，萧逸快速投入数颗妖兽内丹。
<br/>　　一旁的易老满意地点点头，“嗯，时机把握得很准。”
<br/>　　半晌，当一颗完美的，无杂质的洞玄丹被炼制而出。萧逸也是心中一阵喜悦。
<br/>　　他已经确定，自己可以快速炼制这些无杂质丹药了。
<br/>　　易老拿过丹药，审视了数秒，道，“不错，竟然还是完美级别的丹药。”
<br/>　　“在炼药这一方面，你应该不输紫炎了，听说，他炼制过大量的完美丹药。”
<br/>　　易老沉声说道，“紫炎此子，走的是火之一道，能操控那恐怖的紫炎，想必他的武魂极为厉害。”
<br/>　　“如今，你能这么快悟透凝丹五行录。证明你的领悟力不逊色于他，甚至可能更强。”
<br/>　　“炼药水平，也应该与他差不多。”
<br/>　　“唯独输给他的，也就只有武魂这一方面了。”
<br/>　　萧逸翻了个白眼，道，“易老，你不觉得总在我面前提别人多么多么厉害，很打击我吗？”
<br/>　　“打击？”易老冷笑道，“这半个月来，你已经很打击我了。”
<br/>　　“再说了，我要是不好好敲打你一番，你还不骄傲得上天？”
<br/>　　“剑派内，无论是叶铭、顾长空，还是其他各堂的首席，都曾在成为首席时，得意过，骄傲过，导致修为停滞。”
<br/>　　“我既然指导你修炼，自然不会让你走这些弯路。”
<br/>　　“哼。”易老冷声道，“那些个内堂首席弟子，长老们如宝贝般护着。走了弯路，也会好好开解。”
<br/>　　“但我嘛，可不会这么麻烦。”易老沉声道，“若你骄傲自满，我会直接将你扔到东荒十八城那些凶险的妖兽森林中，独自活上十天半月。”
<br/>　　“让你看看什么叫天外有天，也让你见识见识洞玄境巅峰妖兽的厉害。”
<br/>　　“我...”萧逸无语了。
<br/>　　......
<br/>　　“好了，继续开始炼药吧。”易老说道。
<br/>　　“炼药，只是辅助手段。一个武者，修为才是重中之重。”
<br/>　　“别忘了我指导你的目的，便是让你突破到洞玄境。”
<br/>　　萧逸点点头，认真道，“是。”
<br/>　　萧逸再次开始炼药。
<br/>　　顺带着，易老也在一旁帮忙炼药。
<br/>　　萧逸的炼药本事，已经过关。他自然无需萧逸再浪费时间，自己一个人慢慢炼。
<br/>　　只见他拿出一奇怪火炉，而后凝聚火焰。
<br/>　　再之后，一份份材料投入，整个过程，行云流水，速度比萧逸更快。
<br/>　　不多时，当丹药出炉时，萧逸看了一眼，竟是完美级别的丹药。
<br/>　　这证明，易老本身的炼药师阶品，必然很高。
<br/>　　一直到入夜，两人才停下炼药。
<br/>　　无杂质的洞玄丹，而且还是完美级别，两人加起来，足足炼制了三百颗。
<br/>　　80颗由萧逸炼制。
<br/>　　220颗，由易老炼制。
<br/>　　萧逸毕竟是第一次炼制无杂质丹药，谨慎了许多，故慢了些。
<br/>　　当然，易老的炼药速度，确实很快。
<br/>　　而且，很多时候，他还停下来看萧逸几眼，给出一些评价，否则，他的速度会更快。
<br/>　　“三百颗洞玄丹，应该够你用了。”易老说道。
<br/>　　“以你的控火兽武魂品阶，凝聚的气泉，应该是长宽高各三十丈的程度。”
<br/>　　“三百颗，足够你凝聚气泉，而后再提升两重修为。”
<br/>　　“今夜，你回去好好休息。破丹化脉凝气泉，过层很是重要，你务必养足精神。”
<br/>　　易老吩咐着。
<br/>　　“嗯。”萧逸点点头。
<br/>　　“明日不必来寻我。”易老忽然道，“你直接去内门的闭关室，我会给你准备好。”
<br/>　　闭关室，专门为需要突破修为的弟子或者执事准备。
<br/>　　内有静气凝神的蒲团，以助武者快速进入状态。
<br/>　　而且突破时，武者最忌打扰。
<br/>　　所以，闭关室是独立的，且带有禁制。
<br/>　　弟子进入后，除非自己出来，或者剑派长老强行打破禁制，否则他人无法进入打扰。
<br/>　　不过，闭关室，只有内门弟子和执事才能进入。
<br/>　　外门弟子和外门执事，要突破，只能在自己的居所。
<br/>　　“谢易老。”萧逸感激地道谢一声。
<br/>　　“去吧。”易老摆摆手。
<br/>　　......
<br/>　　翌日，闭关室内。
<br/>　　萧逸坐于蒲团上，彻底开始突破。
<br/>　　无杂质的洞玄丹入腹，化为精纯药力。
<br/>　　一颗丹药要完全发挥功效，大概需要半个时辰左右。
<br/>　　萧逸可不想浪费这些时间。
<br/>　　当即利用冰鸾剑武魂，快速吸收起来。
<br/>　　一颗颗丹药如糖豆般扔入口中，顷刻间发挥功效。
<br/>　　按照之前易老教导的办法，准备将体内冰火气丹破开，以及将九条真脉融化。
<br/>　　气丹为基，真脉为材，方成小世界。
<br/>　　萧逸确信，自己的气泉，绝不止长宽高各三十丈。
<br/>　　他想看看，自己的气泉，是哪种级别的。
<br/>　　怀着期待，一百颗丹药在不多时已经完全吞服。
<br/>　　可是....体内冰火气丹，却完全没有要破开的迹象，仅仅是破了一丁点儿。
<br/>　　九条真脉也一样，只融化了不足百分之一。
<br/>　　“这....”萧逸顿时一惊，不信邪地将剩下二百颗丹药全部吞服。
<br/>　　这一次，起效果了。
<br/>　　只不过，冰火气丹仍旧没能破开，只是破了一个小角。
<br/>　　九条真脉，也仅仅融化了半条。
<br/>　　“怎么回事？”萧逸皱起了眉头，“以洞玄丹蕴含的药力，且足足三百颗，就算是长宽高数百丈的小世界也该凝聚成了。”
<br/>　　“我却连破丹化脉这个过程都没有完成，甚至只有一丁点儿效果？”
<br/>　　半晌，萧逸忽然惊醒。
<br/>　　他的冰火气丹，本身就比寻常气丹强许多；他的真脉，本身就被金脉丹增幅过。
<br/>　　之后的九颗五行丹，更是将这两者增幅到一个可怕，更兼完美的地步。
<br/>　　破丹化脉凝气泉，需要多少力量，凝聚出的小世界又有多大，这两者占了很大的因素。
<br/>　　另外，据易老所说。小世界的大小，很大程度上，也取决于武魂的强弱。
<br/>　　气丹、真脉、武魂，便是决定小世界大小的因素。
<br/>　　换个说法，破开小世界的难易程度，也取决于这三者。
<br/>　　“完了。”这是萧逸如今的唯一念头。
