　　越是前进，黑暗便越是浓郁，到了后来，两人干脆放弃抵御这种黑暗物质了，只是紧守心台，护住神魂即可。神魂乃是人体最神秘的地方，一旦那里出错，那便完了。
<br/>　　还好，元始早已形成自己的神魂界域，更有混沌钟护住元神，暂时不会出事，只是时间长了可就不一定了。
<br/>　　步入殿宇之中，不知走了多远，在最深处大殿之前，两人停住了脚步，他们看到前方有一层乌光，将整个大殿覆盖。
<br/>　　“有一个生灵！”
<br/>　　两人震惊了，透过乌光，隐隐能够看到那大殿之上的云纹宝玉大床之上，躺着一个人形生物，一动不动，却又伟岸无比。
<br/>　　元始双眼放光，运转道眼，穿透层层迷雾，仔细观看里面之人。
<br/>　　终于，他看清了看得真切，在残破的大殿之中，破败的云床之上，黑雾翻滚，躺着一个人。
<br/>　　他浑身漆黑无比，身体破败不堪，那浓郁的黑雾，正是从他身体之内散发出来，他正是所有黑暗物质的真正起源之地。
<br/>　　所有的黑暗，都来自那一具尸体。
<br/>　　他很残破，像是被轰炸过，从头颅到下肢，都非常破败。
<br/>　　上半截身体缺少了一大半，头颅也只有半颗，双脚只剩下了骨骼，而且就连骨骼也是漆黑一片。他的胸腔也破烂无比，一颗比拳头略大点的心脏静静躺在那里，漆黑无比。
<br/>　　他全身上下散发着一种无比邪恶的气息，但同时也有一股浩瀚无边的气息，更有一种远超混元强者才有的威压与气机。
<br/>　　哪怕死去了，都能如此，可见他的修为有多么恐怖。
<br/>　　黑暗物质完全是由他那腐烂的身体散发出来的，从而侵染着这片宫殿，进而传到外面，传出整个神秘秘境之中。
<br/>　　仔细看，他身上有金色的神链，将他捆绑在床上，同时那层乌光也应该是镇压在此地的，只是被黑暗污染了，成为黑色。
<br/>　　元始仔细看了看，并没有发现其他东西，但这里的黑暗物质却让人很是难受。因为黑色物质是在是太浓郁了，近乎化成液态，要将他们同化。
<br/>　　“因果之术，溯本还原，时光逆转，昔日再现！”
<br/>　　元始大喝一声，催动体内无上道则，根据因果关系，开始推演此地的过往。
<br/>　　轰隆隆！
<br/>　　像是开天辟地一般，这个地方开始演化诸多场景。
<br/>　　“开！”
<br/>　　元始遭遇到了莫大的阻难，因为此地有混元强者，这个等级的强者，自身便自然蒙蔽天机，若不是他以因果之术推演，以他的修为，还无法推算道什么。就算如此，他都要遭受巨大的反噬。
<br/>　　终于，他看到了一些模糊却又真实无比的场景。
<br/>　　他看到了那具尸体，竟然曾经睁开眼睛，吐出大量的黑色物质，他看到无数精气被这人吸收，用来抵抗身体之内的黑暗物质，而那些精气，竟然……竟然是那些至尊或者真仙身上的精气。
<br/>　　显然这些至尊或者真仙便是之前他们遭遇到的，而他们却又是被斩仙台索来的，那么，斩仙台便是此人所布置的了。
<br/>　　而且，元始还看到此人吸走别人的道则与精气后，身上的黑暗物质便会被排出一些。
<br/>　　元始一阵头大，他明白了，所有的祸乱根源都在此处，斩仙台，亦是此人所布置。
<br/>　　随后，他又看到了，此人竟然是从混沌之中的一个窟窿之中坠落下来，一来便与此界刚刚开辟天地的盘古大战，大战之中，他们打破宇宙，破灭道则，此人竟硬生生盘古斩杀，并吸走盘古与此界精气，之后他已经无力再战，只好将自己封印在这时空乱流之中，每过百万年时间，便吸取一些人间的至尊或者真仙来补充精气。
<br/>　　到此，元始已经全部知道了，此人定然是从另一个更高级的界面中来，因为大战，所以遭受毁灭，意外降落到此地的。同时他也知道了，此界并不是小世界，它其实并不比洪荒差多少，只是因为当初大战，那人竟然将此界的精气吸收大半，才导致世界倒退，退化成如今的样子。
<br/>　　画面自此而断，再上面，比如此人是从何而来，便推算不出来。
<br/>　　“嘶！”
<br/>　　元始倒吸一口气，眼睛瞪得大大的，看着殿宇之内的那具尸体，或者不能算是尸体，毕竟他还可能活着。此人到底有多恐怖啊，吸收如此多精气，竟然仅仅只能压制黑暗，苟延残喘的活着。那么与他大战之人，又有多恐怖？
<br/>　　倒是旁边的道祖，他修为不够，并不能看清那些画面，所以他也并没有像元始那般惊骇。但一看元始的表情，他就大致猜出现什么，表情也很是沉重。
<br/>　　元始看着大殿，沉思了片刻，便在旁边找了个地方坐下，他要开始突破，此事已经不是他能够管的了，就算此人身体残破，修为大损，也不是他这个没有突破混元之人能够对付的。
<br/>　　他现在已经管不了那么多了，这事只有本尊来才可以解决。他取出太玄残匾开始领悟上面的大道，他需要尽快突破，这时已经不是担心黑暗物质的时候了，反正他只是本尊的执尸，他早晚会被收回，他只需将大道领悟即可，管他是不是被黑暗侵蚀，黑暗同样是一种可怕的大道，他的元始之道并不惧怕，相反，元始之道来而不拒，这样的东西，多多善益。
<br/>　　旁边的道祖见此，便问道：“道友可曾了解此事原由？”
<br/>　　元始淡淡看了他一样，道：“不错，此事便是他所为，不过，此事已经超出你我能力太多，吾劝道友还是别管。”
<br/>　　道祖闻言，也沉默了，他不是那种不知轻重之人，从元始的话中，已经足够他了解许多了，元始都不敢管此事，他就更不能了。他看着元始参悟残匾，便再次问道：
<br/>　　“那道友可曾寻到解决吾等身上黑暗之法？”
<br/>　　“不曾！”元始淡淡回答道，这次连眼皮都不曾抬一下。
<br/>　　“嗯。”
<br/>　　道祖见此，也不在言语，亦坐下参悟那匾额之上的大道，他已经看到了下一个境界，想必用不了多久便可以突破，若是突破，想必能够寻得方法。
<br/>　　就这样，两人找了个地方，便开始参悟太玄大道，谁也不知道殿宇之中的人物何时醒来，他们需要时间来突破。
<br/>　　“嗯？”
<br/>　　不知过了多久，元始醒来，发现那些黑暗物质竟然不能入侵自己的神魂深处的灵魂，在他的灵魂之中，一小朵白色的火焰在静静燃烧着，谁也不能靠近，黑色物质刚接近，便被那白色火焰燃烧。
<br/>　　灵魂之火！
<br/>　　元始一阵意外，想不到灵魂之火还有这样的作用。但仔细一想，他也明白了，灵魂之火是一种意志之火，希望之火，黑色物质入侵他的灵魂，就相当于夺舍他，这灵魂之火如何能忍，自然要反抗了。
<br/>　　“灵魂之火能够燃烧黑暗物质，那么……”元始心思微动，有意将黑色物质往灵魂深处引去。果然不管是什么，只要入侵灵魂，都被灵魂之火燃烧。
<br/>　　“咦？竟然还有剩下的，那是……黑暗本源？”元始一阵意外，没想到黑暗物质被燃烧了，却剩下些纯净的黑暗本源。
<br/>　　黑暗本源又为黑暗道源，乃道之本源，既能增加修为，又能增强对大道法则的感悟，，混元之境的修炼，便是修炼大道本源，明悟自身大道，将之领悟到极致，便可以凝聚出大道本源，又称道元，混元、浑源。
<br/>　　大道不是法则，但却又与法则有关，大道包含万千，并无高低，法则却有强弱。
<br/>　　黑暗本源那是一种道源，自然也是一种大道，元始大喜，将之炼化。之后，在元始的有意引导下，黑暗物质不断被灵魂之火燃烧，炼化，留下本源给他修炼。
<br/>　　就这样，五百年过去了。
<br/>　　轰！
<br/>　　这天，忽然一声剧烈震动，惊醒了沉浸在领悟之中的元始。
