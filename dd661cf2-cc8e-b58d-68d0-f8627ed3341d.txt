　　先天神体，便是传说中继承天地本源力量的身躯。
<br/>　　传闻之中，足有数百种，夜天君混乱神体，寒月漓的玄寒之体，曲无音的化玄神体，都是先天神体之一。
<br/>　　但是先天神体，也有着各自的排名，其中的混乱神体，便是在无数先天神体之中，排名第十位。
<br/>　　足以在无数先天神体之中，都位列前十。
<br/>　　而玄寒之体，化玄神体，都不过百位。
<br/>　　混乱，本就是天地之中最早诞生而出的力量之一。
<br/>　　这夜天君所继承的混乱之力，便是代表继承了天地之间最为古老的力量之一。
<br/>　　这种力量，对世间的大部分力量，都足以压制住。
<br/>　　就算是楚风眠身上的战龙之力，都没有混乱之力古老，都要被压制住。
<br/>　　在这威压之下，楚风眠真正足以动用的力量，不足他全力的十分之一。
<br/>　　夜天君的这一招，便是以混乱之力，这最为古老的力量，压制一切。
<br/>　　就算是实力比其他强大，也会被压制住，至少失去十倍的力量。
<br/>　　在战斗之中，失去十倍的力量，在想要战胜夜天君，几乎是不可能的一件事。
<br/>　　正因如此，就算是让一些超越生死的老古董，来对付夜天君，也要被他的混乱之力压制住。
<br/>　　这一招，简直是天命之术，跟他对抗，便是逆天而行。
<br/>　　“混乱神域。”
<br/>　　楚风眠口中，冷冷的开口道。
<br/>　　这正是夜天君，以混乱之力，开辟而出的领域，只要是笼罩在其中，便是每时每刻都会受到混乱之力的威压。
<br/>　　看来这夜天君，已经是将混乱神体的力量，给掌控的七七八八了，才足以凝聚领域来。
<br/>　　“看来你知道的还不少。”
<br/>　　夜天君听到楚风眠的话，笑着开口道。
<br/>　　“既然你知道这混乱神域，想必也明白吧，你，根本没有跟我对抗的资本，从天生上，我就要比你强大，我动用全力，想要杀了你，轻而易举。”
<br/>　　“你能够逼我动用全力，已经是值得自傲了，这混乱神域，自从凝聚出来，还是第一次出手。”
<br/>　　夜天君的话语中，代表着无边的傲气。
<br/>　　他是先天神体，天生便是高人一等。
<br/>　　动用这混乱神域，别说是同境界武者，就算是比起高上他一大重境界的强者，想要击败他，都不可能。
<br/>　　失去十倍的力量，就算是人让超脱生死的人物来，想要击败他，都极为困难。
<br/>　　这混乱神域一出，他不相信，他还会败。
<br/>　　“你的剑术再强又如何？在我的混乱神域面前，你的一切反抗，都将没有任何意义，你的无数奇遇，无数手段，也没用。”
<br/>　　夜天君看着楚风眠，大笑起来。
<br/>　　“从命中注定开始，你就不可能胜过我，现在，我看你如何抵挡！”
<br/>　　夜天君一边猛然开口，一边便是动了。
<br/>　　他的大手一抓，周围的无数灵力，便是化为了一根长棍，一棍向着楚风眠砸了过去。
<br/>　　在这领域之中，夜天君便是至高无上的皇帝，其中的一切，都由他所掌控。
<br/>　　这一棍轰击，轰杀一切，比起刚刚威力还要强大数倍。
<br/>　　而楚风眠，却是受到威压影响，实力减弱十倍。
<br/>　　想要抵挡这一棍，简直是难上加难。
<br/>　　“轰！”
<br/>　　一声巨响，这一棍直接砸在了楚风眠的身躯之上。
<br/>　　连绵不断的轰鸣声，不断的响起。
<br/>　　楚风眠的身躯，就在这烟雾之中，逐渐消失。
<br/>　　擂台之外，全场寂静。
<br/>　　本来众人看到楚风眠逆转局势，都以为楚风眠未必不是没有跟夜天君一战的资本。
<br/>　　谁能想到，在这一瞬间，局势再一次逆转，而这一次，楚风眠都被逼入到了绝境之中。
<br/>　　一双双眼睛，注视着楚风眠所在的地方。
<br/>　　这一棍，楚风眠到底能不能抵挡住，在场武者心中，都没有底。
<br/>　　“吼！”
<br/>　　在这烟雾之中，突然只见，一身龙吟声，猛然响彻而起。
<br/>　　这一声龙吟，震散了周围的烟雾，令楚风眠的身躯，猛然从中窜了出去。
<br/>　　只见楚风眠的身上，一道道龙鳞，围绕在他的身边，组成了一道阵法，正是远古龙王身的力量。
<br/>　　在楚风眠的身边，还围绕着无数剑光，一眼看去，足有十万，组成了一道剑阵。
<br/>　　这黑玄剑阵，跟远古龙王身的力量，从刚刚突然之间，一起爆发了出来，才将这夜天君一棍的力量，给抵挡住了。
<br/>　　但是就在这样，楚风眠身上的衣衫，还是破烂了不少，这一棍的力量，的确强大。
<br/>　　要不是楚风眠身躯远超常人的坚韧，就算是有着剑阵守护，楚风眠也会被活活震死。
<br/>　　现在，楚风眠以黑玄剑阵，在加上他的远古龙王身，居然是在这混乱神域之中，都足以抗衡夜天君的力量。
<br/>　　“这一棍，楚风眠居然抵挡得住了。”
<br/>　　无数众人，看着楚风眠居然没有受伤，都已经震惊无比了。
<br/>　　虽然看现在楚风眠的样子有些狼狈。
<br/>　　但是他们就算是在擂台外，都足以清楚的感觉的出，夜天君的这一棍，到底是有着多么强大的力量。

<br/>　　而楚风眠，就在这擂台之中，承受着夜天君这一棍最强的力量，现在居然都足以抵挡的住，已经是极为惊世骇俗了。

<br/>　　楚风眠，迟早要败。
<br/>　　有了混乱神域在，夜天君，便是永远不败的象征。
<br/>　　而楚风眠，迟早会撑不住。
<br/>　　肉体凡胎，跟先天神体相比。
