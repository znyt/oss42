　　在约定的地方见面之后，那三个乘客看到翠云，也都愣了一下，叶少阳跟他们说明情况，三人表示理解，同时也对翠云的化妆手段赞不绝口。
<br/>　　按照之前的约定，那个年纪稍大的汉子租了一匹马过来，收了大家的份子钱之后，把大家的大件行礼都放在马背上，然后牵着马，一起出发。
<br/>　　一共五个人，除去叶少阳和翠云，另外三人中，那个领头的老者叫老万，还有一个中年汉子，叫陈毛子，另外一个就是跟翠云同村的那个小伙，叫二蛋。
<br/>　　出发之前，老万跟他们约定好，万一遇到土匪，大家争取一起凑钱买路，谁也不许耍赖，一般土匪为了长远打算，不会收走全部的钱，不过如果土匪真的要没收全部钱财，大家一定要配合，绝不能反抗，不然惹恼了土匪，大家都会有危险。
<br/>　　大伙自然一口答应下来。
<br/>　　五人从县城的西门溜出去，然后从一片田野里经过。起初还能听见从县城南门外传来的炮火声，走了一两个小时之后，炮火声也不大听得见了。五人历经几个村庄，终于走进了山区，开始翻山越岭……
<br/>　　一直走到天黑，这荒郊野外，客栈是没有的，五人在一座山的半山腰找到了一个荒废的村子，找了几间但相对较完整的房子，住了下来。
<br/>　　虽然老万清楚这一带没有土匪，但是谨慎起见，五人没有生火，各自拿出干粮，随便解决了晚饭。叶少阳跟他们坐在一起的时候，想到自己一个百年后的人，居然坐在这里，跟一群过去的人一起在山上赶路，真是有一种说不出来的奇妙感受。
<br/>　　这一晚平安度过，第二天继续赶路。翠云毕竟是个姑娘，平时在家虽然也下地干活，但是在爬山赶路这件事上毕竟比不过男人，渐渐的有点走不动了。叶少阳看在眼里，从马上拿下一些姓李自己背着，让翠云上马，别人自然也没什么意见，翠云推辞了一番也就答应了。
<br/>　　按照老万的计划，他们需要在野外走三四天左右，绕过两省交界的地面，进入江西省之后，再就近进入江西的一个县城，然后再想办法租马车，去各自要去的地方。
<br/>　　走到第三天的时候，这天下午，天气突然转阴，五人感觉不妙，打算找地方躲雨，可是没等找到地方，雨就下了起来，虽然不是特别大的雨，但是在野外淋了半天，几个人还是几乎浑身湿透。
<br/>　　幸好叶少阳用来包那些朱砂一类的法药用的是不透水的牛皮袋，不然这些法药真就泡汤了。
<br/>　　五人只好在泥泞中冒雨前进，寻找躲雨的地方，最终在山上发现了一座破庙，于是进去躲雨。
<br/>　　这是一座山神庙，年久失修，几间屋子都塌了，只有大殿还算完整，好歹能躲雨，五个人找地方坐下，身上全都湿透了，看上去十分狼狈。
<br/>　　大伙想找东西生火取暖，结果屋子里没有木柴之类，外面又在下雨，更不可能找到。二蛋和陈毛子把目光落在了山神像上，这山神像是木头做的，有一些地方已经腐烂，二蛋也不管了，上去把神像弄下来。
<br/>　　老万吓了一跳，得知他的意图之后，劝他使不得。
<br/>　　二蛋是个愣头青，不听劝，想找东西把神像砸开，被叶少阳拦住，说道：“我们借用人家的地方休息，已经是打扰，再砸了人家的神像当柴烧，太不合适。”
<br/>　　二蛋哈哈大笑，看着大伙说道：“我说各位，这都是民国了，还这么迷信干嘛，哪来的什么神，都是骗人的鬼话！”
<br/>　　大伙当然不让他砸，免得山神降罪，连累自己，正在争执，从外面突然传来一阵脚步声，五人大惊，抬头看去，是一堆男人走了进来。
<br/>　　一个个都穿着制服，扛着枪，都是当兵的。他们也不知道山神庙有人，也是过来躲雨的，一眼看到叶少阳等人，都愣了一下，一个个相继走进来，一个留着山羊胡、看上去像是带头的士兵说道：“老乡们，哪儿来啊。”
<br/>　　“回老总的话，我们是赶路的。”老万急忙上前赔笑，想给他们掏烟，拿出来一看，包裹烟叶子的纸包也透水了，只好解释了一番。
<br/>　　这些当兵的也没在意，自己找地方坐下来，循着他们五个打量过去，觉得就是普通的赶路人，也没怎么在意，有几个目光朝翠云看去，翠云把脑袋埋在双腿中间，装作睡觉，几个人看见她粗壮的腰身，也就没什兴趣了。
<br/>　　这十几个士兵找了一块干地方坐下来，立刻把身上的衣服脱去拧水，然后也跟叶少阳他们一样，到处找柴火一眼就看到了倒在地上的山神像，这些士兵都是上过战场的，没什么顾忌，用随身带的长刀劈开了神像，正好神像中间是填充的稻草，一点就着，十几个人也不管有妇女在场，直接脱光衣服在火上烤。
<br/>　　当兵的自然是惹不起的，老万等人互相张望着，根本不敢开口。
<br/>　　叶少阳坐在翠云前面当着她，默默望着这些野蛮的士兵，心里头有一种不好的预感。
<br/>　　这些士兵不光烧了山神像，还把窗户也拆下来烧，折腾了老半天，总算把身上衣服烤了个大半干，穿上之后，坐在一起讨论事情。
<br/>　　“吴连长，你说我们这咋办，临阵脱逃，如果被督军抓到，肯定就是毙了……”一个士兵对那个山羊胡子的长官说道。
<br/>　　叶少阳听见这句话，心里头咯噔了一下，这一群，居然是逃兵……
<br/>　　吴连长想了想道：“而今之际，回是回不去了，兄弟们得谋个出路，你们有什么好的去处没有？”
<br/>　　十几个人讨论起来，一个人压低声音说道：“不如去黑云寨……”
<br/>　　大伙顿时都不出声了。
<br/>　　吴连长阴沉着脸，对那人说道：“小三，你让我去当土匪？”
<br/>　　小三抱拳说道：“吴连长，我们都是一起滚过来的兄弟，我问大家一声，你们想活命吗？”
