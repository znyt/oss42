　　第145章
<br/>　　乔野是个耿直的汉子。
<br/>　　他没有熬过方伯山守财奴一般的眼光，含羞带愧道：“是有贼人往家里送了一些银子，我还了还账，如今还剩下一些！”
<br/>　　咦！
<br/>　　方伯山睁大眼睛，没想到啊没想到，我一直以为只有我这种贼眉鼠眼的人才会干这种事，没想到你乔野浓眉大眼的也背叛了我。
<br/>　　“还不去给我取来！你们各家一文也不能少！”
<br/>　　如今在方伯山眼中，程大雷一是幽州王看重的红人，二是齐德强的密友，这两者显然是都不能得罪的。
<br/>　　更何况，我一文钱没捞着，你们却个个吃得满嘴流油，这种事……休想！
<br/>　　方城主眼里不揉沙子，关于钱的事他一分一厘都错不了。方城主姓方，孔方兄的方！
<br/>　　最后的结果令方伯山大吃一惊，小到牢头厨子，上到乔野这样的武将，就连自己身边的管家，都拿了蛤蟆寨的银子。可以说，以程大雷为中心，只要但凡和他能扯上关系的，都被蛤蟆寨买通了。怪不得程大雷在牢里能享受那样的待遇……
<br/>　　这是买程大雷的安全，要是买自己的脑袋，估计也……
<br/>　　这样一想，方伯山忽然觉得自己的人生好好没有安全感。
<br/>　　城主府，后宅。
<br/>　　程大雷在木桶里泡了一会儿，由丫鬟伺候着沐浴，更衣，辫发……从里到外焕然一新。若是以前，程大雷还不适应被十五六岁的小姑娘盯着洗澡，但近些日子因为和柳芷的相处，程大雷已经习惯很多。
<br/>　　那句鸡汤怎么说来着：人改变不了世界，只能改变自己去适应世界。
<br/>　　程大雷由丫鬟领着来到大厅，齐德强和方伯山都等在那里。等程大雷缓步进来，二人同时起身。
<br/>　　“程将军好俊的相貌！”方伯山立刻赞道：“这套衣服就像为程将军量身定做的一般！”
<br/>　　“是么，我以前买衣服时，服务员也经常这么说：先生你好帅喔，这件衣服真适合你。”
<br/>　　“……”齐德强。
<br/>　　“……”方伯山。
<br/>　　有丫鬟捧着铜镜到程大雷面前，程大雷端详自己在镜中的相貌。当真是人靠衣服马靠鞍的缘故，这帝国的官服委实精致，黑底云章素雅得紧，一点也不是后世大红大紫的妖艳贱货。
<br/>　　两鬓垂苏，头发被编得一丝不苟，程大雷微微点头：可能方伯山并没有吹捧自己，只是实话实说而已。
<br/>　　咦，自己以前买衣服付账时也常有这种错觉。
<br/>　　“程将军，主公的其他赏赐都在这里了。”
<br/>　　齐德强伸手指着旁边几物：盔甲一副，白银百两，佩刀一柄，还有一枚官印。
<br/>　　要说杨龙停的赏赐已经很诚恳了，单是这幅精钢鱼鳞甲没有几十两银子都下不来。不过程大雷平素没有穿盔甲的习惯，他信手摸起那枚军印。
<br/>　　入手颇重，黑铁打造，底部刻着四个字：左将军程。
<br/>　　只不过……为何官印上座着的是一只蛤蟆？
<br/>　　“程将，喜欢么，这是主公专门为你打制的？”齐德强。
<br/>　　程大雷深吸一口气，义正辞严道：“愿为主公牵马执蹬，死而后已！”
<br/>　　方伯山一愣：这人对幽州王的感情这么深啊！
<br/>　　他挥挥手，让手下抬着箱子过来。
<br/>　　“程将，哈哈，这几日让程将受委屈了，也是大水冲了龙王庙一家人不认一家人。手下们有眼无珠，这些银子还是要还给程将军。”
<br/>　　那几口箱子里装着的银两，粗略该有三四千两，自然蛤蟆上下打点关系送出去的更多。不过被这些人挥霍一通，如今也就剩下这些了。
<br/>　　乔野等人守在门口一副认罪等待受罚的表情，眼睛还是忍不住看一眼箱子里的银子，把吞进去的银子吐出来，就像割他们的肉一样，他们怎么能不心疼。
<br/>　　程大雷笑笑，道：“诸位近日辛苦，本……将军今日就职，这些银子就当我谢诸位兄弟的，让弟兄们拿下去分了吧。”
<br/>　　第一次用这种称呼，自己还有些不习惯呢。
<br/>　　诸人用诧异的眼神看着程大雷，眼中闪烁着苍蝇见血般的红光，然后几乎异口同声道：“多谢程将军！”
<br/>　　就连齐德强也有几分钦佩程大雷的气魄，这是实实在在的一掷千金。同时心中也有些警惕：一个不贪财的山贼，那他贪图的是什么。
<br/>　　等过了这件事，必定要除掉他，他这个人……留不得。
<br/>　　“方大人，牢里还有我几个兄弟，麻烦方大人一块儿放出来。”
<br/>　　方伯山：“程……”
<br/>　　“程将军放心，我立刻就去把弟兄放了，程将军的弟兄岂不就是咱们的弟兄。”牢头迫不及待道。
<br/>　　方伯山狠狠瞥了他一眼，随之哈哈笑道：“手下人都会处理好的。”
<br/>　　是会处理好，怕比自己吩咐处理得更好。
<br/>　　“另外，我还有些私事想和方城主单独谈谈，齐将可否……”
<br/>　　“我懂，我懂。我正好想去外面转转。”
<br/>　　齐德强迈步走出大厅，同时心里琢磨不明白，一个山贼要和方伯山谈什么。
<br/>　　等程大雷和方伯山出来时，他发现方伯山得态度格外亲近了几分，走了几步，终究忍不住问了出来：“程将，你和方城主商量了些什么，好像他很高兴？”
<br/>　　“哈，也没什么，我只是把独家代理权给他了。”
<br/>　　“……”齐德强。
<br/>　　都说官匪不同炉，但有时候，官就是匪，匪就是官，官匪勾结的事情也不是发生过一次两次。
<br/>　　和方伯山打好关系，可以说好处多多，但靠杨龙停赐下的这张虎皮撑不了多久，没有永远的朋友，也没有永远的敌人，只有永远的利益。所以程大雷就把辣椒的独家代理权给了方伯山。
<br/>　　城主也是做生意的。
<br/>　　方伯山一路将程大雷送出城主府，道：“程将，回去的路途遥远，不如我差几个人送程将回去。”
<br/>　　“不必了，我暂时还不想回去，而且……”程大雷跨出大门：“我的弟兄差不多也该到了。”
<br/>　　跨出门口的一瞬，本来安静的城主府门前，突然从四面八方窜过来几个人，然后人越聚越多，最后竟有百八十众，城主府门前黑压压一片。
<br/>　　众人垂肩，抱拳，齐声道：“大当家！”
<br/>　　
<br/>(本章完)
