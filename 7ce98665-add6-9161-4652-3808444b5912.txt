　　棕色透明的机油糊在汽车前挡风玻璃上，模糊了驾驶员的视线，汽车引擎被子弹打穿，冒出阵阵白烟，失去动力的汽车瞬间失去控制，在马路上走着蛇形步。
<br/>　　跟在这辆轿车后面的二手SUV在来不及制动的情况下，一头撞在失控的轿车上。
<br/>　　哐！
<br/>　　轿车车门被撞得凹陷下去，原本走着蛇形步的轿车，被撞得在马路上打起转来，车轮跟水泥马路摩擦发出尖锐刺耳的摩擦声，路面上留下黑色的橡胶摩擦印痕。
<br/>　　哐！
<br/>　　轿车一头撞在路边消防栓上，水柱从被撞歪的消防栓中喷出，眨眼间地面上已经满是水渍。
<br/>　　轿车中的两个人脑袋垂在方向盘上，生死不明；SUV的车头也是被撞得瘪了下去，车门被推开，SUV的驾驶员解开保险带，爬下汽车，可是他刚爬下汽车，一个点射便是打入他的后背，血花溅起。
<br/>　　点射来自罗昊的方向。罗昊跟智天使和吴云都打过交道，所以SUV驾驶员从汽车中出来的一瞬间，罗昊就认定这辆车里的人不是智天使和吴云，直接就一个点射招呼上去。
<br/>　　慕容飞盯着狙击镜，锁定了一辆从他面前驶过的红色马自达轿车，扣动扳机。
<br/>　　砰！
<br/>　　安装了补偿器的枪口闪过一朵橘红色的膛口焰，.408口径的钨芯穿甲弹从枪膛中飞旋而出，以五倍音速朝着红色马自达驾驶座上的人影飞去。
<br/>　　铿！
<br/>　　子弹击穿车门，命中了驾驶座上的司机；弹头强大的侵彻力并没有让它在击中司机身体后，就停留在司机身体中，弹头贯穿了司机的身体，改变弹道轨迹后，击中了副驾驶上那名替身的大腿。子弹穿过替身大腿，打断了他的股动脉，钻进副驾驶座椅中。
<br/>　　子弹击中司机的刹那，司机被子弹拉着往他右手边位移，而在子弹传出他身体的瞬间，司机身体又被安全带反拉回来，鲜血喷射而出，很快汽车的前左右玻璃上便是被一片嫣红遮盖。
<br/>　　马路上，枪声四起，从写字楼中冲出来的汽车有好几辆被拦截了下来，但是毫无意外，里面并没有智天使和吴云。
<br/>　　而且因为从地下车库冲出来的汽车实在太多，想要全部拦截下来根本不可能，不得不说，智天使制定的这套撤离方案成功了！
<br/>　　指挥中心内，赵天宇很快从子午牺牲的影响中调整过来，他是这次行动的最高指挥官，他很清楚作为一个指挥官，在什么样的时间应该做什么样的事情。
<br/>　　赵天宇很快就跟K国快速反应部队的司令官取得联系，要求他在各个路段设卡拦截车辆。智天使这个人实在太危险了，说他一个人能顶得上一个集团军都不为过！他做事不计后果，为了达成目的可以不择手段，任何人任何事都可以拿来牺牲！
<br/>　　赵天宇让人把晕倒的云熙送去休息，这丫头不到二十岁的年纪被特招进龙隐部队，成为子午的老师，她跟子午之间的感情早就超过了普通的师生情，甚至可以说，在暗影回归之前，云熙就是从子午身上找到那种被父亲关爱的温暖。
<br/>　　“傻小子，小熙晕倒了，现在被送去休息了。”
<br/>　　赵天宇想了想拿起对讲机对罗昊说道：“你是龙刺，你应该清楚自己在这个时候应该做什么！”
<br/>　　罗昊在听到云熙晕倒在指挥中心的消息后，拳头猛地握紧，但是随后，他又缓缓松开，深吸一口气，令自己冷静下来。没错，他现在要做的是怎么抓住智天使，而不是担心云熙的问题。
<br/>　　赵天宇之所以在这个时候把云熙晕倒的消息告诉罗昊，是有意而为之，他必须要让罗昊明白，作为一个战场指挥官必须要懂得取舍，即便有些时候这种舍弃会很难以接受，也必须要这么去做。
<br/>　　……
<br/>　　混在车流中冲出写字楼地下停车库的吴云驾驶着汽车，在其他车辆的掩护中很快就冲破了写字楼外的火力封锁，沿着马路疾驰，在穿过两个街区后，吴云猛打方向，汽车往右急转，沿着小路又行驶了一段距离，吴云把车停在路边，跟智天使两个人离开汽车。
<br/>　　吴云拿出一颗进攻手雷，拉开保险销拆掉保险夹，从车窗扔进汽车中，随后跟智天使两个人快速走进路边的弄堂中。
<br/>　　轰！
<br/>　　进攻手雷发生爆炸，把这辆车身上留着五六个弹孔，车窗玻璃也被子弹打穿的破二手车炸得剩下一个架子，火焰包裹着汽车残骸还在燃烧着，黑烟升腾而起……
<br/>　　吴云和智天使穿过弄堂后，吴云用钥匙打开一间出租屋的门，里面停着两辆大功率摩托车，摩托车上挂着安全帽。
<br/>　　两个人跨上摩托车，戴上安全帽，拧动插在摩托车上的钥匙发动摩托车，随后冲出安全屋，沿着只允许非机动车辆通过的小路疾驰。
<br/>　　智天使利用固定在车载架上的手机拨打了一个电话，电话只响了一声就被接通了，“影天使完蛋了，我跟暗天使正在按计划撤离。但是我这里得到一消息，GHOST有学生，而且目前就在K国！”
<br/>　　“GHOST的学生？”小丑冷笑一声，说道：“好的，我知道怎么做了，你们尽快从K国撤离出来！”
<br/>　　智天使挂断电话后，把手机从车载架上拿起来，甩手扔到路边。手机砸在墙壁上，瞬间屏幕开裂，后盖飞离；砸在墙上反弹回来落在路面上的手机，被跟在智天使后面吴云的摩托车压过，彻底报废。
<br/>　　智天使的撤离计划在他进入K国的时候就已经完成，他把自己撤离计划的时间误差甚至精确到了三分钟以内！
<br/>　　“狂龙，那些汽车全都散开了，路上行人和车辆太多，不方便我们拦截！”
<br/>　　林晓阳在从指挥中心得到消息后，把情况告诉了罗昊。
<br/>　　“妈的！”
<br/>　　罗昊一拳头砸在汽车引擎盖上，把汽车引擎盖都砸得凹陷下去。他抬头看了眼还在冒出滚滚浓烟的写字楼顶楼，说道：“但愿妖瞳的判断没有错，我们去写字楼看看。”
