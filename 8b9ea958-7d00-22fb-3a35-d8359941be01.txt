<br/>    在紫金山走走逛逛，还真能发现不少好东西，凌仙在一个地摊前停了下来，伸手拿起一柄飞剑，问道：“店家，这个怎么卖？”
<br/>    
<br/>        店家看了凌仙一眼，眼冒金光，“少侠真是好眼光，这一柄飞剑，可是大有来头，乃是一位妖王手中的兵器，传闻，只要得到了这一柄兵器，便有机会称王！”
<br/>    
<br/>        陈明也是看了一眼这一柄飞剑，并没有看出什么特殊的地方，听这店家忽悠的倒是有模有样。
<br/>    
<br/>        不过凌仙有着戒指里的老爷爷护佑，陈明倒是也不怕。
<br/>    
<br/>        凌仙和店家砍了一会儿价，伸手拿起一个木制的小鼎，道：“这样，买了你这飞剑，把这个小玩意儿送给我了！”
<br/>    
<br/>        店家眼见飞剑卖出去了，也没怎么在意，直接将小木鼎送到了凌仙的手里。
<br/>    
<br/>        师徒五人走了不远，陈明伸手直接将那个小木鼎拿了过来，在手中把玩，“嗯，这小木鼎之中居然有着一缕先天大道之气！”
<br/>    
<br/>        凌仙笑道：“师父，这先天大道之气，对于我的道经修行可是有着重要的作用，道经的修行，实在是太慢了！”
<br/>    
<br/>        木鼎在陈明的手中瞬间粉碎，他可是无量道体，对于先天大道的气息再敏感不过，直接将这木鼎之中的一缕先天大道之气提取了出来，陈明感受了一番，简直就是最为纯净的大道气息，陈明打过一道灵气在先天大道之气上，灵气经过这先天大道之气的过滤，瞬间变得纯净无比。
<br/>    
<br/>        陈明伸手将凌仙拉了过来，伸手一拍，直接将这一缕先天大道之气拍入了他的体内，“你现在还驾驭不了这一缕先天大道之气，将它养在你的道宫之中，以后修行，只需要吸收灵气便好，不需要炼化。”
<br/>    
<br/>        凌仙拱手道：“多谢师父！”
<br/>    
<br/>        这先天大道之气，也算是一个宝贝，而且对于凌仙来说，更是珍贵的宝物，毕竟道经的修行，可谓是缓慢之极，现在，李素衣的修为早就已经超过了凌仙。
<br/>    
<br/>        有了这先天大道之气，凌仙的修行应该会快不少。
<br/>    
<br/>        继续在紫金山逛街，李素衣倒是没有什么兴趣，看来李素衣想要的东西，在拍卖会上。
<br/>    
<br/>        陈鱼倒是对一株看起来奇怪的杨柳枝感兴趣起来，那是一株杨柳枝，尾端一片漆黑，顶端却是有着一颗翠绿的芽。
<br/>    
<br/>        和店家砍价说价了一番，最后花了三百灵石将其买了下来。
<br/>    
<br/>        陈明也是没有看出其他的东西，只是觉得这一株杨柳枝之中有着死劫之力，还有着新生之力，还真是奇怪。“带回去，将这一株杨柳枝养起来，说不定有什么特殊的地方。”
<br/>    
<br/>        在紫金山逛了一圈，最后还是卓青瑶和陈鱼买了一大堆的东西，你不要以为都是什么宝物，不不不，都是胭脂衣服首饰什么的。
<br/>    
<br/>        女人的天赋，学不来。
<br/>    
<br/>        在金犴安排的地方住了两天，拍卖会终于是如期而至，陈明倒是想要知道，那个让李素衣心动不已的东西到底是什么。
<br/>    
<br/>        拿着自己的请帖，来到最高贵的十间房间之一，陈明在其中坐下，自有大妖送来酒水，各色灵果。
<br/>    
<br/>        陈明也是开始静静开始等待拍卖会的正式开始，在看台之上，一个穿着红裙，露着雪白大腿的妖女道：“诸位能够光临紫金山，让紫金山蓬荜生辉，废话不多说，想必诸位也是等急了，现在，拍卖即刻开始！”
<br/>    
<br/>        就在这时，房间门口有一人进来，在陈明的耳边耳语了几句，“金毛吼大人，请燕山之主过去议事，不过请隐秘行事，不要让其他人发现燕山之主已经离开，大君还有眼线在此处。”
<br/>    
<br/>        道完，将一袭黑袍递到了陈明的手中。
<br/>    
<br/>        陈明点头，将自己手指上的储物戒递给卓青瑶，“里面有四千万灵石，你们想买什么就自己叫价。”
<br/>    
<br/>        道完，陈明披上黑袍，已经跟着来人出去了。
<br/>    
<br/>        一般主角买东西，应该都是一些别人看不上的奇奇怪怪的东西，陈明跟着来人来到最底层最低等的房间之中。
<br/>    
<br/>        小小的房间之中，已经挤了三位大妖王，其中有着陈明已经见过的长河，还有着满头金发乱舞的金毛吼，另外一位，则是陈明从来没有见过的鹤长白。
<br/>    
<br/>        这鹤长白的祖上，传闻是仙人的坐骑，不过家道沦落了，便在这十万大山之中落草为王。
<br/>    
<br/>        陈明听到的时候眼皮直抽抽，在十万大山之中当一个大妖王，还是家道沦落了，仙人的坐骑有这么夸张吗？
<br/>    
<br/>        地面上铺着一张地毯，三位大妖王就这么盘膝坐在地毯上，三位倒是没有什么不满的神色，他们也知道这一次要谈的是秘密之事。
<br/>    
<br/>        陈明也学着他们的样子，盘膝坐下，四人坐了四方，陈明看向愁眉苦脸的长河，问道：“事态已经这么严重了吗，长河你这么愁眉苦脸的？”
<br/>    
<br/>        长河摇摇头，道：“不是，我在想，我家那个败家玩意儿，会在我离开之后挥霍多少灵石买那些奇奇怪怪的东西！”
<br/>    
<br/>        看来其他大妖王也是一样的，留下了人在房间中继续拍卖，让其他人以为他们还在。
<br/>    
<br/>        金毛吼咳了咳，道：“诸位贤弟，想必我这次请大家来此的原因，大家都已经猜到一二了。”
<br/>    
<br/>        鹤长白道：“其他的五位大妖王呢？”
<br/>    
<br/>        金毛吼道：“其他五位兄弟，太过靠近大君，现在根本走开不得。”
<br/>    
<br/>        陈明三人都是点点头，如今大君的野心可谓是昭然若揭，距离大君最近的五位妖王自然是脱不开身。
<br/>    
<br/>        金毛吼继续道：“现在乃是多事之秋，我听到一些如同天方夜谭的消息，传闻有一尊恶鬼蛊惑了妖族大君，向他许诺只要攻下十万大山，便可以赐大君妖族君主之位，统领这苍天之下的所有妖族。”
<br/>    
<br/>        鹤长白谨慎道：“恶鬼？大哥可知道恶鬼代表的是什么意思？”
<br/>    
<br/>        金毛吼摇摇头，道：“嗯，难道还有什么别的意思？”
<br/>    
<br/>        鹤长白道：“所谓恶鬼，便是死灵，与生灵两不相容，恶鬼乃是死灵，没有了生命的存在，绝对不可信！”
<br/>    
<br/>        金毛吼道：“可是，我们是无法说服大君的，他已经被恶鬼蛊惑了，我们现在能做的，便是想办法如何抵抗住大君的攻势。”
