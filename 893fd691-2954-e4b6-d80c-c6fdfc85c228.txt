<br/>        “放门口的桌子上吧！”唐倩兮没好气地说。

<br/>        女医生便把药膏给放在了门边的桌子上。

<br/>        她人还未转身，就只见童筱颖疾步走过去，拿起药膏，毫不领情地直接将药膏扔进了桌子底下的垃圾桶里。

<br/>        “你——”女医生看着童筱颖这举动，刚想说什么的时候，又将想要说的话给咽进了肚子里。

<br/>        童筱颖柳眉微蹙，开始收拾自己的东西准备回家。

<br/>        唐倩兮权当什么都没看见。

<br/>        女医生自知讨了个没趣，甩手离开了。

<br/>        那个女医生是十爷的人，平常医务室里都没人，所以女医生都不在“夜惑”，这会儿突然出现在这儿，一定是十爷给临时叫过来的。
<br/>
<br/>        女医生离开后，唐倩兮用手肘戳了戳童筱颖的手臂问：“怎么，你怕十爷害你啊？”

<br/>        童筱颖点了点头，随即眉开眼笑：“嫂嫂，今晚我们一起回家吧！”
<br/>
<br/>        “好！反正今晚，我手下的姑娘都下班了。可以提前回家了！”唐倩兮微笑着应了声。

<br/>        童筱颖原本以为唐倩兮知道，心想着连嫂嫂都不知道的话，季雅姗一定不想让更多的人知道她的事情，于是只能对唐倩兮撒了谎，摇头表示自己不认识。

<br/>        唐倩兮抿了抿唇，觉得也对，那个人好像只有晚霞一个人认识，而且，晚霞也不肯跟她透漏那个人是谁，和十爷是什么关系。
