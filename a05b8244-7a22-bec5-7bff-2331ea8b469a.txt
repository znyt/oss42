　　光是眼前这极为震撼的画面，就给人强烈的压迫感，这一击的强度，不用想也猜得到。
<br/>　　“我来！你们准备好补刀！”
<br/>　　四宝立刻就要祭出罗汉金身，但望着头顶俯冲下来的火鸟，他其实并没有把握，更不要说他之前受了伤，勉强打坐了一个周天，也只回到八成状态。
<br/>　　但除了他，在场三人没人能接下这一击。
<br/>　　他观想出了帝释天，一尊高大的巨神，手持实质化的加持宝杵，朝头顶正在俯冲的巨大火鸟的脑袋打了下去。
<br/>　　“轰……”
<br/>　　一道极强烈的波动，将三人都震飞出去十几米远。
<br/>　　吴嘉伟撞击在墙上又倒下去，将爬起来，一目看去，四宝趴在地上，一边吐着血，一边正努力想要爬起来。
<br/>　　这就是上古异兽拼命的力量，超级血牛四宝的罗汉金身也扛不住，不过也得亏是他，换成其余人，只怕这时候已经挂了。
<br/>　　“吼……”
<br/>　　又是一声长鸣，大伙抬头看去，火鸟在四宝头顶上空盘旋着，正在蓄力，无数流火——那是毕方的羽毛，正在不断落下，显然是刚才那一次撞击给他带来的后果，但他仍然又飞了起来，很快蓄积了足够的力量，再度朝着四宝的后背俯冲下来。
<br/>　　“和尚！”
<br/>　　吴嘉伟大叫一声，赶过去来不及了，于是作法念咒，将一身法力灌入藏锋，朝前掷去。
<br/>　　噗嗤一声，宝剑正中火鸟的脑袋，穿了过去，溅起一大波火焰——火鸟的血，火鸟发出刺耳的惨叫，却丝毫不停，不断旋转身体，扇动着流火，如同一支燃烧着火焰的巨剑，刺向四宝的后心。
<br/>　　“哔——”
<br/>　　一声清鸣，在边上响起，一只跟毕方差不多大、也是浑身燃烧，但比毕方颜色更深一点的火鸟，横着朝毕方冲过去。
<br/>　　是岳恒的真身，四大灵兽中的朱雀。
<br/>　　异兽之中，有四大灵兽，代代传承，岳恒就是朱雀的转世传人，道风唤醒了他的真身，闭关修炼到现在，但他毕竟不是终极版的朱雀，没有他的祖先那么强大的实力，但这一刻他也使出了全力，朝毕方撞过去。
<br/>　　他来不及阻止毕方对四宝的这一击，但如果毕方不放弃进攻，纵然四宝会被撞死，岳恒这一下也会撞击在他身体的侧面，等于用身体挨住这一下。
<br/>　　他不敢赌，再说跟四宝也没深仇大恨，他也不想去赌，于是他放弃了四宝，在空中转了一圈，与岳恒对撞了过去。
<br/>　　一个是觉醒的朱雀传人，一个是修了千年的上古异兽，两只都是火鸟。
<br/>　　他们身上的火焰，照亮了半个天空。
<br/>　　周围几个战场的打斗双方都被这极其震撼的一幕深深吸引，不约而同的放弃了打斗，一起转头看过来。
<br/>　　“轰……”
<br/>　　两只火鸟迎头撞上，爆发出一团巨大的火焰，不断扩散开来，将半边天空照得通红。余波产生的气浪从山上横扫而过，离的最近的几棵大树拦腰折断，一时间流火遍山，不过这种火焰点燃不了人间的东西，很快就消散了。
<br/>　　两只鸟，都趴在地上，身上的火都灭了，奄奄一息。朱雀浑身焦黑，在地上努力想要爬起来。
<br/>　　“小岳岳，小岳岳你没事吧！”孙映月冲了上去，一把将浑身烧成黑炭的朱雀抱起来，只见他浑身冰凉，眼神涣散，像一只被烤熟了的鸟。
<br/>　　“水……”
<br/>　　朱雀口中发出含糊的字眼。
<br/>　　“水！”孙映月自己身上没水，转身冲大伙喊道。
<br/>　　吴嘉伟赶紧跑过来，拿出一瓶农夫三拳给她，孙映月往朱雀嘴里灌了几口，朱雀并没喝下去，而是张口喷出一大口水蒸汽，涣散的眼神很快对焦，用力抖了一下身子，化成人形，无力地躺在孙映月的怀里，道：“小月月，我没事。”
<br/>　　吴嘉伟在边上听的一头雾水，问道：“你俩到底谁才是小yueyue？”
<br/>　　“我是小月月，他是小岳岳。”孙映月解释。
<br/>　　吴嘉伟挠着后脑勺，以他的智商，到底是没听明白。
<br/>　　见岳恒没事，大伙这才调转目光去看毕方。
<br/>　　也是熄了火焰的一只鸟，身上蒙着厚厚的一层黑灰，像是裹了一个黑色的壳在身上。
<br/>　　就在大伙想要走过去的时候，黑色的壳突然裂开了，大伙吃了一惊，急忙站住，警惕地望过去。
<br/>　　咔嚓。
<br/>　　壳裂开了，一道火光泄露出来，就好像中间燃烧的煤球被人扒开了一条缝，露出了藏在里面的火焰。
<br/>　　“还没死吗，我去解决了他！”
<br/>　　吴嘉伟说着提剑过去，就在这时，壳更加裂开来，然后一道火焰破壳而出，冲天飞起，却不再是之前的火鸟，拖着长长的尾巴，浑身燃烧着橙红色的火光，转头看了一眼众人，扬起头，发出一声尖啸的清吟，朝远处飞去了。
<br/>　　“那是凤凰吗？”孙映月呆呆地望着天上那个神奇的生灵，喃喃说道。
<br/>　　“他是不死鸟，死亡帮助他渡劫重生了。”岳恒也远远望着飞走的凤凰说道。
<br/>　　“那怎么办！”孙映月听到重生两个字就叫起来。
<br/>　　“放心，他虽然重生，但是虚弱的很，怕是需要休养许多年，再说他堪破了生死，眼中不会再有我们了。”
<br/>　　本来他也是星月奴请来的帮手，双方并没有什么深仇大恨。
<br/>　　大伙放下心来，一起望着飞走的凤凰——不管他是不是真的凤凰，但这幅画面还是美极了，深深印刻在他们脑海中。
<br/>　　让他们无法想到的是，二十年后，他们中间那些仍然还活着的人，有幸又见到了这只凤凰，时光流转，腥风再起，不过那又是另一个故事了。
<br/>　　望着凤凰于飞的画面，岳恒心中突发触动，仿佛瞬间领悟了飞行的真谛，一种前所未有的奇妙感觉充斥了他的神识，那种强烈的冲动无法克制，在这种本能的驱使下，他大叫了一声，现出真身，一飞冲天，在天空翱翔起来。
<br/>　　飞行的真谛。
<br/>　　在与毕方经历了这一战之后，本就隐约显现，又看到凤凰于飞，一下子就顿悟了，之前闭关清修了那么久，仿佛一棵被深埋的种子，经过了漫长的积累和储备，在一次雨水的洗礼之后，彻底破土而生！
