　　第524章
<br/>　　这天夜里，程大雷匹马单枪离开蛤蟆城，一路向东，要跋涉千山万水，去东海之地寻找华氏一族。
<br/>　　这自然不是件轻松的事情，不提西凉和东海相隔万里之遥，关键是，程大雷就连华氏一族在不在东海，他都不知道。
<br/>　　中间隔了这么多年，又赶上兵荒马乱，说不得他们早就集体搬家了。偌大一个帝国，要找一个名不见经传的小村庄谈何容易。
<br/>　　程大雷的计划是从凉州出发，一路穿过京州、青州，再由兖州进入徐州。抵达徐州后，再具体打听华氏一族。程大雷心里琢磨着，华氏一族既然以医术传世，多少该在周边有些名气，或多或少得跑出来几个徒弟，自己到时候潜下心打听，无论如何能打听出些消息。
<br/>　　一路晓行夜宿，程大雷也不敢靠近城池，免得让人当通缉犯抓起来。他大半是走在山林中，睡在山林中。
<br/>　　如此走了十几天路，程大雷离了凉州，又小心翼翼穿过京州。到这一日，算着到了青州和兖州的交界处。
<br/>　　眼前被一条大河拦住了去路，河水由北向南，这片地界连日下了几天暴雨，河水开阔有五六十米。
<br/>　　以黑牛的本事，是能托着程大雷过河的，而程大雷的水性也不算弱。但因为连日暴雨的缘故，河水十分湍急，程大雷觉得没必要冒险，心里还是想着找座桥过河。
<br/>　　他骑在牛背上，沿着河流的方向往南走，路上行人渐渐多了。程大雷下意识将头低下去，以程大雷如今在帝国的名声，一旦被人发现了，还不知有多少人要杀他。以程大雷的本事自然不必怕，但毕竟做贼心虚。
<br/>　　程大雷出门前，做了一番乔装，故意打扮成文生公子的模样。以如今帝国的信息传播渠道，天下也没几个人知道程大雷的真实相貌。不过，程大雷座下那头黑牛实在有些显眼，程大雷穿着文生公子衫，委实有些奇怪。
<br/>　　但走着走着，程大雷便没了这方面的担忧。因为路上奇怪的人太多了，有披头散发者，有赤膊纹身者，有身后背着两柄车轮大斧者。当然，更多的还是面黄肌瘦，瘦骨嶙峋的平民。
<br/>　　如今帝国诸王分立，各路诸侯都发了招贤令，要招募天下高手。所以，隐匿在世间的妖魔鬼怪，旁门左道都闻风而动，从地底下冒了出来，想要投靠到各路诸侯手上奔一个前程。
<br/>　　程大雷混在他们中间，丝毫不起眼，甚至显得有些平平无奇。
<br/>　　他混在行人中，听他们说话，下游三十里，有一座石桥。跨过这座石桥，就算是兖州地界。到了兖州，便距离程大雷的目的地近了。
<br/>　　程大雷一路盘算着路程，大概有二十天，自己就能抵达徐州。总的说来，程大雷赶路的速度是很快的，主要原因，还是程大雷座下这头水陆双栖的黑牛。
<br/>　　他垂头琢磨着，不知不觉已能看见石桥，抬起头，见桥头聚了三五百人。人都聚在一起，却没人过桥。程大雷心里有些奇怪，下来牵着牛靠近桥头。
<br/>　　“我们黑河九鬼，靠山此山，靠水吃水。现在这座桥，就是我们兄弟的。但凡想要从此过，都要留下买路财。”
<br/>　　程大雷靠得近了，才发现不是大家不想过桥，而是桥头被人堵住了。五六十精壮汉子守在桥上，各持利刃，为首一人方口阔目，额头一颗大痦子。
<br/>　　此人的外号是独角鬼，结拜兄弟九个，在这一带打家劫舍。程大雷距离他们二三十步远，见他们脚下躺着几具尸体，应该是想过桥不能，被当场杀死的。
<br/>　　遇见这种事，程大雷一点也不觉得奇怪。天下大乱，这种拦路劫匪层出不穷，程大雷一路过来，也遇见过几波，避不过的，程大雷也就亮出匹夫剑，超脱了他们性命。
<br/>　　程大雷扫了他们一眼，没觉得他们之间有什么高手。五六十人，也没办法对自己构成威胁。他之所以没有立刻动手，是顾忌现场太多人，怕暴露了行踪。
<br/>　　“诸位好汉，在下乃是兖州黄土镇的人。这里有一份薄礼，送给诸位好汉买酒喝。”一个商人模样的年轻公子靠前，捧上一口装满银子的小箱子。
<br/>　　“呔，当我们黑河九鬼是要饭的么。我们不是向你借一分，不是向你借两分，我们是全要。”独角鬼大声道，一脚将这年轻商人踹倒在地，碎银子散落一地。
<br/>　　被堵在桥头的人叫苦不迭，有些是辛苦拼搏一年，才挣了些糊口的钱粮。若被这水贼劫走，全家老小都得喝东北风去。
<br/>　　程大雷搜寻着脑海中的信息，喜字队收集情报，对于江湖上有本事有势力的人物，也有一份名册。然而关于这黑河九鬼……程大雷从来没有听说过。
<br/>　　大概是小得不能再小的小蟊贼，喜字队觉得不值得被程大雷看在眼里。
<br/>　　当然，程大雷如今已经今非昔比，在帝国皇帝的书桌上都挂着号呢。论当贼，他已是贼这一行祖师爷级别的人物，自然不必在意这几个蟊贼。
<br/>　　“哈哈哈！”
<br/>　　万不得已，程大雷已经打算出手，正这时候，一阵洪亮的声音响起。
<br/>　　“几个小蟊贼，便敢劫本当家的路，啧啧，本大家免不了今天要路见不平，替天行道。”
<br/>　　独角鬼斜睨起眼睛，道：“是那条路上的兄弟，报个名字吧？在这条河上，我们兄弟还真没怕过谁。”
<br/>　　“呃……哈哈哈！提起本当家的名字，你还要洗耳听真，这天下那个不知，谁人不晓，我怕说出来，吓破你的狗胆！”
<br/>　　声音在耳边嗡嗡作响，大家都回过头去看。程大雷心中也有些好奇，这是谁呐，口气这么大。如果真的是江湖上有名有姓的，自己不会没听说过。
<br/>　　回过头，只见一条大汉，骑着一条狰狞黑牛，身披大氅，手中攥着一柄宣花大斧，距离程大雷是越来越近，越来越近……
<br/>　　程大雷张大了嘴巴，很久没办法合拢……
<br/>　　“本当家……姓程。”
<br/>(本章完)
