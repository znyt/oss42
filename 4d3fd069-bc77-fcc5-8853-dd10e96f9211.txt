        “你真的跟连星儿不熟？”邢乐游狐疑地睨着明司寒。
<br/>
<br/>        就算他明司寒的名声大噪，也不至于让一个刚进他们校门的新生立马就认出来，而且还一副自然熟的样子过来主动打招呼吧？
<br/>
<br/>        明司寒答非所问，云淡风轻地说道：“你们传令下去，从今以后，不许任何一个男的议论连星儿，也不许任何一个男的追求连星儿，还有不许任何人跟连星儿走得近。”
<br/>
<br/>        顿时，邢乐游、张子晋和仲正浩不禁面面相觑。
<br/>
<br/>        “莫非，连星儿是你前女友？”张子晋难以置信地问。
<br/>
<br/>        仲正浩却不以为然：“司寒连女朋友都未曾有过，怎么可能有前女友！”
<br/>
<br/>        “噢！我知道了，咱们寒哥是想着自己养肥了再开吃！”邢乐游咧嘴道。
<br/>
<br/>        “吃个头噢！寒哥这话，不是摆明了要全校孤立她吗？”张子晋忍不住白了邢乐游一眼。
<br/>
<br/>        仲正浩下意识地问道：“司寒，你跟连星儿有仇啊？”
<br/>
<br/>        明司寒冷不丁地打断了这三人的调侃：“对，我跟她有仇。”
<br/>
<br/>        “？！”邢乐游、张子晋和仲正浩三人同时一脸茫然。
<br/>
<br/>        尽管如此，明司寒还是在人海茫茫之中，一眼就看到了排队打饭的队伍中的连星儿。
<br/>
<br/>        连星儿也没想到，自己的室友，竟然是那个和自己在云城一起探洛羽的班同住一个酒店房间的顾千凝。
<br/>
<br/>        因为一进门彼此就认识，所以连星儿和顾千凝很默契地成了对铺，而且很快就成立了两人组的小团体，报完到后就一起来食堂打饭。
<br/>
<br/>        “食堂的糖醋排骨是招牌菜，只不过分量太少了。”顾千凝感慨道。
<br/>
<br/>        轮到连星儿打饭的时候，她特意打了两份糖醋排骨的量，然后和顾千凝一起分享。
<br/>
<br/>        顾千凝也没料到，她随口一句话，就让连星儿上了心。
<br/>
<br/>        打完饭菜，坐在白色长桌前，还未开吃前，连星儿就将自己盘子里的糖醋排骨，夹了一大半给顾千凝。
<br/>
<br/>        “星儿，你真好！”顾千凝感动道。
<br/>
<br/>        连星儿咧嘴一笑：“我们都是‘羽翼粉’，自家姐妹，应该的。”
<br/>
<br/>        “羽翼粉”是洛羽的粉丝代称。
<br/>
<br/>        一提起洛羽，顾千凝就有说不完的话题。
<br/>
<br/>        或许是因为有共同的兴趣爱好，连星儿觉得自己和顾千凝特别投缘。
<br/>
<br/>        而食堂里，盯着连星儿的人，不仅仅只是明司寒，还有那边一个不起眼的角落里的朱小溪和张蓉蓉。
<br/>
<br/>        “连星儿不是去美国念书了吗？怎么也在我们临大？”朱小溪愤愤不满地说道，“可恶，我怎么到哪儿都甩不到连星儿的影子啊？她是不是故意跟踪我噢！”
<br/>
<br/>        张蓉蓉不禁安慰道：“还好她不跟我们一个系。”
<br/>
<br/>        “看到她，我就觉得自己晦气。”朱小溪一边搅动着盘子里的饭，一边恼火地说道。
<br/>
<br/>        张蓉蓉此刻却八卦起来：“我倒是听说到一个关于连星儿的八卦新闻！你要不要听听？”
<br/>
<br/>        “什么八卦新闻？”
<br/>
<br/>        “今天早上我们这届新生报到的时候，连星儿主动去搭讪我们校的大校草，结果，大校草说跟她不熟，她当场被‘打脸’，而且，大校草还传令到学校里的每个男生，说谁都不许追求连星儿。”
<br/>
<br/>        “大校草？谁是大校草？”朱小溪一脸懵。
<br/>
<br/>        张蓉蓉立即掏出手机，打开校内网里的风云人物，将明司寒的照片指给了朱小溪看。
<br/>
<br/>        朱小溪一眼就认出了照片里的这个男生。
<br/>
<br/>        “他、他不就是那次在网咖里，推了一下连星儿，害得我被泼了一身冰激凌奶茶的男生吗？”朱小溪惊诧不已地说。
<br/>
<br/>        若不是她这么一提，张蓉蓉还真就把这事儿给忘了。
<br/>
<br/>        朱小溪又接着说道：“暑假的时候，连星儿说她出门旅游，在高铁上还遇见了他，并且还问他要了五百块钱赔偿我的衣服。连星儿把钱都转给我了！”
<br/>
<br/>        “那他们岂不是早就认识了？”张蓉蓉惊愕道。
<br/>
<br/>        朱小溪连连点头。
<br/>
<br/>        张蓉蓉撇了撇嘴，忽然间想到了什么，接着说道：“好像，大校草还命令说，要全校孤立连星儿。”
<br/>
<br/>        “孤立连星儿？”朱小溪没明白。
<br/>
<br/>        张蓉蓉解释道：“就是不许任何人跟连星儿走得近。”
<br/>
<br/>        “那连星儿岂不是要被‘冷暴力’对待了？”朱小溪顿时有些幸灾乐祸。
<br/>
<br/>        而白天还跟自己好好的顾千凝，到了晚上，连星儿便发现顾千凝对待她，就像突然变了一个人。
<br/>
<br/>        晚自习的时间，是新生到班级教室的集合点。
<br/>
<br/>        连星儿跟顾千凝约好要一起去教室集合的，但是当她洗完澡出来，却发现寝室里只剩下她一个人。
