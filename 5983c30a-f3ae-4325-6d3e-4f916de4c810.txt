　　赵天宇是什么人？
<br/>　　龙刺！龙隐部队现任最高指挥官！
<br/>　　他这样的人什么把戏没见过，什么阴谋诡计没经历过，他这样的人嗅觉何等灵敏？在救护车被拦下，黑西装男出示调查令的时候，赵天宇就立刻想明白了其中的关键。
<br/>　　自己这些人被盯上了！
<br/>　　不管是因为什么原因，反正自己这些人有麻烦了！
<br/>　　拦在救护车前的这些疑似K国安全部门的人，总共有十二个人，赵天宇不怀疑他们身上携带着武器，但是这对于赵天宇和罗昊来说，并不具备任何威胁。如果他们愿意完全可以在对面这十二个人来不及拔出武器的情况下，把他们全都放倒。
<br/>　　“调查令？”赵天宇扫了眼对方手里的A4纸，问道：“你隶属于哪个部门？因为什么事情需要我们配合调查？”
<br/>　　“K国安全局第九处。”黑色西装男回答道。
<br/>　　“第九处？”赵天宇挑了挑眉毛，说道：“第九处是对外间谍处，我不明白对外间谍处会找到我，并且需要我配合调查？”
<br/>　　黑色西装男一时语塞不知道应该要如何回答赵天宇的问题，只能回避道：“对不起将军，具体情况我也不清楚，我只是接到这样的命令。”
<br/>　　“我需要跟我们国家大使馆联系。”赵天宇拿出手机，说道：“在我国大使馆人员抵达之前，我不会跟随你们去任何地方，你们有需要可以跟我方大使馆进行沟通。”
<br/>　　“将军，请你配合我们的工作！”西装男语气比起之前强硬了一些。
<br/>　　“除非你们能拿出确切的证据，证明我们触犯了贵国的法律，否则的话，我们不会跟你去任何地方！”赵天宇盯着对方说道。
<br/>　　“将军，请你配合我们的工作！”西装男撩开外套，做出要拔出腰间手枪的动作。
<br/>　　“如果我是你，不会做出这样愚蠢的决定。”赵天宇指了指西装男腰间枪套中的手枪，说道：“这个东西一旦出现在你的手中，事情的性质就发生了改变，你需要明白，你将要面对的是什么！”
<br/>　　西装男略一思考，最后还是撩开的西装放下了，没有把枪给拔出来。
<br/>　　“你没有做蠢事。”
<br/>　　赵天宇看着西装男，问道：“你第一次杀人是什么时候？”
<br/>　　“什么！？”西装男被赵天宇这个问题问得有些发懵，但最后还是回答道：“我没杀过人。”
<br/>　　“记住，开枪打人和开枪打靶是完全不同的。你可以在靶场上打出100环，但是当你面对一个人的时候，在最后时刻，你未必有胆量扣动扳机。”赵天宇说道：“我第一次杀人是在我十五岁的时候，法庭上，法官问我，我是想要去监狱还是进军队。我选择了去军队，所以论起杀人我比你们所有人都要专业，如果我想要动手，你们现在已经是死人了。”
<br/>　　西装男两侧鬓角流淌下汗水，让身边几个人看住赵天宇和罗昊后，拿出手机跟他的上司取得联系。
<br/>　　“必须要把他们带回来！这是总统阁下亲自下达的命令！”
<br/>　　“明白！”
<br/>　　西装男脸色有些难看的往赵天宇和罗昊的方向看了一眼，收起电话。
<br/>　　在西装男跟他上次进行联系的时候，罗昊轻声对赵天宇问道：“师父，你这段‘经历’我怎么不知道，而且听着这么耳熟？”
<br/>　　“正好电影里看到，就拿来用的。”赵天宇说道：“我们国家又不是美国，犯了罪法官怎么可能给你选择？”
<br/>　　返回过来的西装男对身边的K国安全局第九处探员，命令道：“立刻把他们带走！”
<br/>　　哗啦！哗啦！哗啦！
<br/>　　眨眼间，十几个K国安全局第九处探员就从腰间快枪套中拔出了手枪，瞄准了赵天宇和罗昊两个人。
<br/>　　“我们要动手吗？”罗昊对赵天宇问道。
<br/>　　“大使馆的人正在赶来的路上，应该很快就能到，尽量拖延时间。”赵天宇对罗昊吩咐道：“如果被他们带走，没有罪也变成有罪了。随便套上一个罪名，我们就成国际间谍了。”
<br/>　　“他们这个样子对我们可是虎视眈眈，完全不打算让我们拖延时间。”罗昊看着一步步逼近过来的K国安全局第九处探员，对赵天宇说道。
<br/>　　“必要的时候可以防御，但是不要下死手。”赵天宇命令道：“杀了人，我们浑身是嘴都说不清楚了。”
<br/>　　“将军，你们必须要跟我走一趟了。”西装男盯着赵天宇，说道：“只是协助调查而已，如果确认你们没有涉及危害我们国家安全，很快就会恢复你们的自由。”
<br/>　　“你的这些谎言可真是太低级了。”罗昊轻轻摇头，对着西装男竖起中指，说道：“我读小学时候撒的谎都要比你现在说的高级。”
<br/>　　“那就对不起了中校。”西装男皱眉，一挥手说道：“把他们全都抬走！”
<br/>　　两名K国安全局第九处探员双手握着手枪，身体微弓朝着罗昊靠近，罗昊眼睛就像X光一样从那两个朝自己靠近的第九处探员身上扫过，很快就制定出了进攻方案。
<br/>　　两名第九处探员走到罗昊面前，其中一名探员继续用枪指着罗昊，另外一名探员伸手摘下挂在后腰的手铐。
<br/>　　就在那名探员拿出手铐的瞬间，罗昊动了，他上前一步，左手一拳挥击在用枪指着自己的第九处探员脸上，把他打得脚下一个踉跄，身体往一边歪倒，右手探出，牵制住另外一名探员的右手腕一拧，把他的右手反剪到身后，把他手里的手铐抢夺到自己手里，将手铐一端拷在他的右手腕上，抬脚踹在他的膝盖上，抄起他的左手拷在手铐中，拔出他腰间快枪套中的手枪，指着那个被自己一拳打倒在地还没爬起来的第九处探员。
<br/>　　“别乱动，如果把我吓到了，一不小心扣动扳机就不好了。”罗昊对着那个躺在地上的第九处探员威胁道，紧接着抬脚把他踢晕过去。
<br/>　　“该死！”
<br/>　　带队的西装男咒骂一声，拔出手枪，打开保险，对着周围剩下的第九处探员招呼道：“抓住他们！”
