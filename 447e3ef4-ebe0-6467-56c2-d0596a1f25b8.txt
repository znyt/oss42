　　沾满暗红色血迹的纸上只是潦草的写着几个字：知名不具。
<br/>　　知名不具？！什么意思？‘二哥’的眉头紧锁，将字条递给了身边的其他人，慢慢的坐回到椅子上面。
<br/>　　站在他身边的老三同样看到了上面的字迹，脸色亦是难看的要命，喘着粗气说道：“二哥，这……”
<br/>　　‘二哥’并没有让老三把话说完，突然一抬手止住了他，同时沉声说道：“敌暗我明，照他的要求办吧！”
<br/>　　“什么？二哥，你……”
<br/>　　“闭嘴，如果不想自己的族人哪天也死于非命，那就去办事。”‘二哥’狠狠瞪了身边的老三一眼，低声吼道。
<br/>　　这下，老三不吱声了，跺了一下脚大步离开了房间。
<br/>　　‘二哥’说的没错，如果不想再死人，那就只能按照那份情报上的要求去做，否则的话下一次死的人会更多。
<br/>　　毕竟只有千日做赋没有千日防贼的道理，王家虽然是千年世家，但其在武力方面强不强大，若是在十年前或许家里还有一些家丁，家将什么的，但是后来被李承乾一顿整治之后，家丁、家将全都没了，武装力量不及当初百分之一。
<br/>　　这样的情况下，如果隐王的势力想要针对他们，别说杀他们几个人，就是把整个王家老宅都有屠了，估计也不是什么难事儿。
<br/>　　至于说什么会说是隐王的人干出这种事，原因也很简单，因为李承乾如果想要对付王家根本不必用这么麻烦的手段，直接派兵过来就可以完全彻底的将整个王家彻底从这个世界上抹去。
<br/>　　房间中的其他人在看了字条上的内容之后，一个两个也都沉默了，看了一眼‘二哥’全都默默的离开了房间。
<br/>　　事情已经无法改变，王家无法跟一个没有底限的组织博弈，而且隐王的势力基本上都在暗处，就算是想要与其拼命都找不到对手。
<br/>　　所以迫于压力，王家不得不妥协，按照从咸阳发来的情报中的内容开始收集一些材料，然后启用自家的几个小的钢铁厂开始试验。
<br/>　　但他们不知道的是，不管是钨矿还是锰矿，这些东西都已经在情报科的暗中监视之下，王家虽然收集的不多，但依旧被情报科抓住了把柄。
<br/>　　李承乾既然已经开始布局，自然每一步都有自己的用意，之所以独孤玉凤会丢出这样一份资料，不过是为了吸引更多人上钩而已，否则在独孤玉凤出现的那一天，只要他派兵将那片树林一围，有多少势力也别想跑掉。
<br/>　　所以情报科在掌握了王家的动作之后并没有打草惊蛇，反而是继续躲起来观察，反正王家家大业大，就算是想跑也跑不掉。
<br/>　　而王家却因为过于相信隐王的势力，并没有察觉到这是一个圈套，依旧在不断的试验着资料上的配方，终于在第十天的时候，成品钢材出炉了。
<br/>　　对于这种新式的钢材，王家自然不会错过，马上采取了一些手段进行试验，而试验的结果表明，新式材料要远远好于以前的百炼钢。
<br/>　　那些由百炼钢打制的刀剑在新材料面前完全就是渣渣，同样一把新材料打造的横刀竟然可以连断五把百炼钢打制的横刀，除了刀口稍微些卷刃之外，在使用方面没有任何的其他影响。
<br/>　　这下子，王家也惊了，这对于家族来说完全就是一个意外的惊喜，只要掌握了这样的工艺和材料，日后发达指日可待。
<br/>　　于是乎王家一边派人将新制出来的横刀样品送往咸阳，一边又派人带着这份材料的配方赶往辽东。毕竟那里山高皇帝远，王家在那里经营了几年，已经有了一些势力，就算是搞一些小动作也不会有什么影响。
<br/>　　……
<br/>　　李毅恒这十多天完全就是在纠结中度过的，王家那边一直没有消息，就算同面对死亡的威胁也是一样。
<br/>　　但是他相信王家一定会站在自己这边，因为人只要不是傻子，就不会试图与一个歇斯底里的人讲道理，除了按照那个人的指示去做，没有任何回旋的余地。
<br/>　　而且李毅恒也想信王家不会去举报自己，因为不说王家已经与他是一条绳上的蚱蜢，如果他被举报了，那么他有足够的把柄可以把王家拉下水，相信一个千年世家应该不会因为一时冲动，就跟他这个疯子同归于尽！
<br/>　　但尽管如此，一天没有收到消息，他就一天不能彻底放下心来，总是想着会不会出现什么意外。
<br/>　　终于，这一下午的时候，黄浦在出去一趟之后，拿回了他们需要的东西：“皇子殿下，王家回信了，已经确定了资料是真的。”
<br/>　　“真的么？东西在哪里？”李毅恒听到资料是真的喜出望外。
<br/>　　“东西还没有到，只是通过飞鸽传书确定了资料的真伪。”黄浦同样显的十分兴奋，在确定了资料是真的以后，他否定了以前独孤玉凤是钓他们出来的鱼饵的判断，开始相信这一切都是真的，也就是说，他们的机会来了。
<br/>　　“真的能够确定么？”李毅恒高兴了一会儿之后，又有些怀疑，有些不相相信。
<br/>　　“千真万确，资料是真的，新式材料完全强于百炼钢。”黄浦重重一点头。
<br/>　　李毅恒再次得到确认之后，终于是放下心来，不过另外一件事情又让他皱起眉头：“那我们接下来要怎么办？真的要按照独孤玉凤那个女人说的，去皇城劫人？”
<br/>　　“只怕不去是不行了，那个女人神秘的很，情报科的出身让她对一切都十分小心，我们的人根本就无法跟踪她。而且我觉得她应该一直在观察我们这边的动静，我们如果没有行动的话，估计她是不会出现的。”黄浦想了想之后，说出了自己的判断。
<br/>　　李毅恒事实上也在担心这个问题，现在被黄浦说了，他也是深有同感的点了点头，然后有些郁闷的说道：“可那是皇城啊，以我们现在的力量就算再增加十倍的人手，也不足以攻进去。”
<br/>　　“我觉得没有必要那么麻烦。”黄浦摇了摇头说道：“从皇城里带一个人出来，除了杀进去之外，还有另外一个办法。”
