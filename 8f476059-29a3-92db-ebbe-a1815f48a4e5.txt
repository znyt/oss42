　　第311章
<br/>　　在帝国，戎族以残暴著称，这搞得人人谈戎色变。事实上，戎族不仅对帝国残暴，对自己人也比较残暴。
<br/>　　帝国西北的草原上，生活着数百个部落，他们虽然都是放牧为生，各部落间却有不同的信仰。互相之间打来打去，彼此把对方男人都杀了，女人抢回去的事情根本算不上新闻。
<br/>　　所以说，戎族崇拜强者，谁实力强，谁就是草原上的王。那些什么北蛮王、西羌王都是打出来的，凭的可不是道德文章。
<br/>　　程大雷已经从樊梨花口中得知这件事，如此行事手段就干脆果断些，大手一挥，就是杀敌斩将，显了雷霆手段。
<br/>　　果然，此举收到了极好的效果，围上来的诸部落都停在周围不敢近前。随着程大雷的声音落地，周围窃窃私语声响起。
<br/>　　“程大雷……”
<br/>　　“该不会就是那个杀掉草原王的程大雷！”
<br/>　　“我听说西颜部的赤檑也是被他所杀！”
<br/>　　程大雷骑在马上，本来嚣张无比，周围窃窃私语声传入耳中，程大雷忽然一激灵，心情变得忐忑起来。自己刺杀草原王的事情也已经传到这里来，这些人该不会同仇敌忾，一起动手杀了自己吧。
<br/>　　周围有数百人，真若一伙人同时杀过来，自己也只有逃命这一个选择了。
<br/>　　程大雷却发现，周围人都下马放下了兵器，脸上的战意似乎已消失不见。
<br/>　　似乎杀掉草原王后，他们并没有愤怒，自己反而得到了他们的尊重。
<br/>　　“请问阁下能否放了我兄弟，为此我西戎部愿意支付赎金。”一个戎族汉子走到程大雷面前。
<br/>　　“你兄弟？”
<br/>　　程大雷怔了怔，方才明白对方说得是谁，正是自己一开始擒到，现在还被手下押着的那戎人。
<br/>　　西戎部……据樊梨花所说，是这片地盘最大的一个部落，倒是不能得罪。
<br/>　　“我们只是来到此地做生意，如果你们不向我们动手，我们自然也不会动手。”程大雷的言下之意是说真如果动手，却也不害怕你们。
<br/>　　“既然你这当哥哥的已经来了，我就把他放了吧。”
<br/>　　名叫黑岩的汉子摇摇头：“我兄弟被你擒了，是阁下的本事大，按草原上的规矩，我们该为此支付赎金。阁下想要什么，尽管开口？”
<br/>　　程大雷眨眨眼睛：“你们有什么？”
<br/>　　黑岩愣了愣，道：“我愿意用五十张兽皮，二十匹骏马，一名女奴换回我弟弟。”
<br/>　　“唉，女奴就算了，带姑娘来的，再要女人不方便。”程大雷挥挥手：“赶紧回去取吧，我在这里等你。”
<br/>　　“大当家，他们会不会是回部落码人了？”秦蛮。
<br/>　　“应该不会吧，据说戎族还是蛮讲信用的。”程大雷挥挥手：“更何况咱们手里还有人质么，来来，咱们继续卖东西。”
<br/>　　程大雷用干净利落的手段解决一场战斗，倒是没有激起众人的愤怒，反而帮他赢得了尊重。再加上有自己的名声助力，接下来的交易进行的很顺利。
<br/>　　众人纷纷围上来，程大雷手上的东西很快被交换完了。
<br/>　　这时候，黑岩也驱马回来，程大雷索要的兽皮，骏马都已带回来。
<br/>　　程大雷也遵照约定，把那戎人放了。这人名叫律鸿，是西戎王的儿子，身份却也不低。
<br/>　　律鸿摆脱程大雷后，恨恨的看了程大雷一眼，揉着发酸的肩膀，眼神里却是佩服多过愤怒。
<br/>　　“律鸿，这位就是当初刺杀草原王那位，草原上数万人要杀他都没有得手，凭你也敢向其出手。”
<br/>　　经过黑岩如此一渲染，周围人看程大雷的目光都异样起来，恐惧中夹杂着佩服。
<br/>　　“好说，好说啦。”程大雷随意挥挥手。
<br/>　　“这个女奴，是我送给阁下的。”说着，黑岩推过来一人。
<br/>　　“唉，说了不收女人的。”程大雷摆摆手：“你这个人还真是，还真是……太客气啦。”
<br/>　　当视线触及这奴隶时，程大雷的视线怔住。见这女孩不过十四五岁左右，又黑又小，黑色头发披散下来，诡异的是，一双眼睛是瘆人的银色。
<br/>　　看着程大雷的目光像一头饿极了的幼狼，程大雷下意识伸过去手，想将她的脸蛋扳正，她忽然张开牙齿，在程大雷虎口留下一个伤口。
<br/>　　“这是送给阁下的，我主要是想求阁下一件事。”黑岩。
<br/>　　这女孩明显是夷人之女，程大雷心中一动，若有这样个漂亮姑娘站在自己身后服侍，岂不也是很有面子。
<br/>　　“什么事？”程大雷拍拍银眸女孩的脑袋，目光没有看黑岩。
<br/>　　“下个月是我成亲的日子，我需要一百匹帝国的花布，我们没办法出现在帝国，可否帮我带一批布过来，价格好商量。”黑岩。
<br/>　　这就有回头客了，程大雷点头答应，忽想起一件事。
<br/>　　“黑兄，花布不是问题，只不过来往的路上不方便，一次我们无法运这么多货过来。你知不道附近有别的路？”
<br/>　　“这样呐……”黑岩沉思片刻，道：“我倒是知道一条路，只不过，只不过……”
<br/>　　“只不过什么？”程大雷问。
<br/>　　“只不过……唉，我让人带阁下看看，阁下就明白了。”
<br/>　　程大雷收拾换来的货物，让大队人马和黑岩的人一起出发。
<br/>　　临走时，程大雷没忘了那女孩，他勾了勾女孩的鼻尖，问：“你叫什么名字？”
<br/>　　女孩没有吭声，用诡异的双眸盯着程大雷。
<br/>　　黑岩道：“她是个哑巴，没有名字，别看她长得瘦小，力气很大的，你要给她饭，她就替你做事。”
<br/>　　“没有名字么？”程大雷道：“那我就给你一个名字，以后你就叫银眸，跟着我混，我管你饭吃。”
<br/>　　姓名：银眸
<br/>　　年龄：14
<br/>　　技能：无
<br/>　　隐藏属性：铁力
<br/>　　咔嚓，程大雷挥动斧头，斩断了她身上的铁链。
<br/>　　小女孩大概是听懂了程大雷的话，轻轻点了点头，眼中瘆人的光芒柔和了几分，但仍是和先前一样的冰冷。
<br/>　　由黑岩和律鸿引路，离开草原，进入山区，又拐过曲折的道路，前面出现一道峡谷。
<br/>　　黑岩指着峡谷道：“原先这里是有一座桥的，只是后来被毁了，路也就废了。”
