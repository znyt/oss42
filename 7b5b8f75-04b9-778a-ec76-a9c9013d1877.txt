        “我说过，我不想听。麻烦你，放我走！”童筱颖偏过头去，扭动着双腕试图从傅昼景的手中挣脱出来。
<br/>
<br/>        傅昼景索性直接捧住童筱颖的脑袋，扬起她那张不施粉黛也天生丽质的面庞，强迫她与自己四目相对。
<br/>
<br/>        她仍旧试图挣扎，想要推开他的束缚，而这个时候，他却将手下的力度加重，将她牢牢地禁锢在自己与墙壁之间。
<br/>
<br/>        两人的唇瓣几乎快要吻上，而从傅昼景那温热的呼吸里，透着一丝危险的气息，悉数全洒在了童筱颖的脸上。
<br/>
<br/>        此时此刻的傅昼景，让童筱颖觉得好陌生。
<br/>
<br/>        她觉得他像是变了一个人，已经不再是她所认识的那个昼景了。
<br/>
<br/>        待她冷静下来时，他才缓缓开口：“筱颍，大哥的心思，没有你想象中的那么单纯。他不会是真正爱你的。你不要陷进去了。”
<br/>
<br/>        “我的事，你能别多嘴吗？”童筱颖一脸不悦地反驳，再次试图挣扎。
<br/>
<br/>        傅昼景忽然疾言厉色道：“筱颍，大哥当年读高一的时候，因为强奸未遂，进过少年监！”
<br/>
<br/>        语毕，童筱颖惊愕得定格了身子。
<br/>
<br/>        “大哥晚上在围墙角强暴了同年级的女生，当场被学校里巡逻的保安抓住时，大哥连裤子都脱了，而女生衣衫不整。那个女生是学校里的校花，女生的父母还将大哥给告上了法庭！若不是爷爷当年有钱有权有势，将大哥从少年监里捞出来送去军校，并出了补偿费和封口费，只怕这个污点会跟随大哥一辈子。”傅昼景接着叙述道。
<br/>
<br/>        他神情淡定，说得有板有眼，完全不像是在编造是非。
<br/>
<br/>        童筱颖难以置信地睁大了眼睛，瞪着傅昼景，忽然间一句话也说不上来。
<br/>
<br/>        傅昼景见童筱颖似乎是信了自己的话，这才慢慢地放开了她的脑袋，大手顺势捧着她的脸颊，指腹爱怜地在她眼角的肌肤上摩挲着。
<br/>
<br/>        “筱颍，我说的都是真的。我知道，你委身于大哥，一定不是你自愿的。是他强迫你的，对不对？筱颍，你真的愿意跟有个不良污点的大哥在一起吗？”傅昼景深沉地质问。
<br/>
<br/>        “大哥的那件事情，你是亲眼所见大哥强暴了那个校花吗？”童筱颖面无表情地反问。
<br/>
<br/>        傅昼景微微摇了摇头。
<br/>
<br/>        童筱颖倏然后退了一步，挥开了傅昼景的手，不以为然道：“既然，你不是亲眼所见。那么你所说的，我一句也不信。”
<br/>
<br/>        眸光微沉，她毅然转身离去。
<br/>
<br/>        傅昼景看着童筱颖黯然失色的背影，嘴角得意地咧开一抹弧度。
<br/>
<br/>        她嘴里说着不信，但她心里已经信了他的话，更何况，他说的是事实。
<br/>
<br/>        童筱颖离开医院后，并未去傅氏集团大厦里上班，而是直接回到了公寓里，整个人郁郁寡欢地躺在了客厅的沙发上。
<br/>
<br/>        大哥到底是一个什么样的男人，又有着什么样的过去，她突然间好奇又心痛了起来。
<br/>
<br/>        还有傅昼景。
<br/>
<br/>        他又为什么会变成现在这个令她完全陌生的样子？
<br/>
<br/>        或许是因为大中午被季思妍折腾到医院跑上跑下，又被傅昼景给灌输了太多令她难以接受的信息量，童筱颖躺在沙发上看着天花板发呆时，不知不觉中就睡了过去。
<br/>
<br/>        她搬到这公寓里来住，傅夜沉自然是“娶妻随妻”地跟着一起住了进来。
<br/>
<br/>        下班时，他提前去了设计部，但并未看到她人，孙莉莉告知他，她一下午都未来办公室，他便想着可能是她哪里不舒服在家里休息。
<br/>
<br/>        这不，傅夜沉一进门，看到童筱颖躺在客厅的沙发上，潜意识里真以为她是哪里不舒服，急得连忙踱步过去，伸手便将童筱颖从沙发上揽入了自己的怀中。
<br/>
<br/>        “筱颖，你这是怎么了？”一道焦虑的关问声飘忽入耳。
<br/>
<br/>        童筱颖睡醒后，睁开双眼，看到傅夜沉剑眉紧锁，于是抬起手来，轻轻地抚平了他的眉心：“大哥，你回来了。”
<br/>
<br/>        “是不是最近加班太累呢？我让孙部长给你派个贴身助手。”傅夜沉接着说道。
<br/>
<br/>        童筱颖会心一笑：“不累！只是我今天偷懒了！不想去公司上班。”
<br/>
<br/>        “真有你的！”傅夜沉松了口气，指腹轻轻地点了点这小女人的鼻尖，“还把我吓了一跳！”
<br/>
<br/>        “大哥，你就这么紧张我？”童筱颖随之抬起双手，勾住傅夜沉的脖子，眯了眯眼睛，俏皮地问道。
<br/>
<br/>        “当然，你可是我的宝贝老婆大人。”傅夜沉嘴角微扬。
<br/>
<br/>        看着傅夜沉，童筱颖忽然间想起了傅昼景所说的那些话，心里犹豫了一下，故作不经意间，试探性地问道：“大哥，你有没有，这一辈子都令你难以忘怀的一晚？”
<br/>
<br/>        “有！”傅夜沉眼里有光，嘴角含笑，不假思索地回答，“还不止‘一’晚！”
